softwareupdate

A tool for updating macOS App Store apps via the command-line. More information: https://ss64.com/osx/softwareupdate.html.

  • List all available updates:

softwareupdate --list

  • Download and install all updates:

softwareupdate --install --all

  • Download and install all recommended updates:

softwareupdate --install --req

  • Download and install a specific app:

softwareupdate --install update_name
