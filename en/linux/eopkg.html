eopkg

Package manager for Solus. More information: https://getsol.us/articles/package-management/basics/en/.

  • Install a specific package:

sudo eopkg install package_name

  • Update all packages:

sudo eopkg upgrade

  • Search for packages:

sudo eopkg search search_term
