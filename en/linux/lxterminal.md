# lxterminal

Terminal emulator for LXDE.
More information: <https://wiki.lxde.org/en/LXTerminal>.

- Open an LXTerminal window:

`lxterminal`

- Open an LXTerminal window, run a command, and then exit:

`lxterminal -e "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>`"`

- Open an LXTerminal window with multiple tabs:

`lxterminal --tabs=`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">tab_name1,tab_name2,...</span>

- Open an LXTerminal window with a specific title:

`lxterminal --title=`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">title_name</span>

- Open an LXTerminal window with a specific working directory:

`lxterminal --working-directory=`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
