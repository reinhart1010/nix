aa-enforce

Set an AppArmor profile to enforce mode. See also: aa-complain, aa-disable, aa-status. More information: https://gitlab.com/apparmor/apparmor/-/wikis/manpage_aa-enforce.8.

sudo aa-enforce --dir path/to/profile
