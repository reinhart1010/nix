# wifi-menu

Interactively connect to a wireless network.
More information: <https://manned.org/wifi-menu>.

- Set up a wireless connection interactively:

`wifi-menu`

- Interactively set up a connection to a network and obscure the password:

`wifi-menu --obscure`

- Display help:

`wifi-menu --help`
