pinout

View the current Raspberry Pi's GPIO pin-out information on the terminal with an ASCII diagram. More information: https://www.raspberrypi.com/documentation/computers/raspberry-pi.html#view-a-gpio-pinout-for-your-raspberry-pi.

  • View the pinout information and GPIO header diagram for the current Raspberry Pi:

pinout

  • Open https://pinout.xyz/ in the default browser:

pinout -x
