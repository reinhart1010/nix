toolbox

Manage containerized command-line environments on Linux. Some subcommands such as create have their own usage documentation. More information: https://manned.org/toolbox.1.

  • Run a toolbox subcommand:

toolbox subcommand

  • Display help for a specific subcommand (such as create, enter, rm, rmi, etc.):

toolbox help subcommand

  • Display help:

toolbox --help

  • Display version:

toolbox --version
