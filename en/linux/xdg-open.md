# xdg-open

Opens a file or URL in the user's preferred application.
More information: <https://man.archlinux.org/man/xdg-open.1>.

- Open the current directory in the default file explorer:

`xdg-open .`

- Open a URL in the default browser:

`xdg-open `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com</span>

- Open an image in the default image viewer:

`xdg-open `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/image</span>

- Open a PDF in the default PDF viewer:

`xdg-open `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/pdf</span>

- Display help:

`xdg-open --help`
