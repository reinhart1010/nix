# a2enconf

Enable an Apache configuration file on Debian-based OSes.
More information: <https://manpages.debian.org/latest/apache2/a2enconf.8.en.html>.

- Enable a configuration file:

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">configuration_file</span>

- Don't show informative messages:

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">configuration_file</span>
