urpmi.update

Update the list of packages from a package repository in Mageia. Note: Mageia documentation uses medium and repository as synonymous. See also: urpmi, urpme, urpmi.addmedia, urpmi.removemedia, urpmf, urpmq. More information: https://wiki.mageia.org/en/URPMI#urpmi.update.

  • Update all enabled media:

urpmi.update -a

  • Update specific media (including disabled media):

urpmi.update medium1 medium2 â€¦

  • Update all media that contain a specific keyword:

urpmi.update keyword

  • Update all configured media:

urpmi.update e
