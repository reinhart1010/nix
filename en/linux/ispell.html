ispell

Interactive spell checking. More information: https://www.cs.hmc.edu/~geoff/ispell.html.

  • Start an interactive session:

ispell

  • Check for typos in the specified file and interactively apply suggestions:

ispell path/to/file

  • Display the version:

ispell -v
