pipewire

Start the PipeWire daemon. More information: https://docs.pipewire.org/page_man_pipewire_1.html.

  • Start the PipeWire daemon:

pipewire

  • Use a different configuration file:

pipewire --config path/to/file.conf

  • Display help:
