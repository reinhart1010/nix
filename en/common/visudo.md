# visudo

Safely edit the sudoers file.
More information: <https://www.sudo.ws/man/1.8.13/visudo.man.html>.

- Edit the sudoers file:

`sudo visudo`

- Check the sudoers file for errors:

`sudo visudo -c`

- Display version information:

`visudo --version`
