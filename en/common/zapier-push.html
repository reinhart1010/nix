zapier push

Build and upload a Zapier integration. More information: https://github.com/zapier/zapier-platform/blob/main/packages/cli/docs/cli.md#push.

  • Push an integration to Zapier:

zapier push

  • Disable smart file inclusion (will only include files required by index.js):

zapier push --disable-dependency-detection

  • Show extra debugging output:

zapier push -d|–debug
