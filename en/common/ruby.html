ruby

Ruby programming language interpreter. More information: https://www.ruby-lang.org.

  • Start a REPL (interactive shell):

irb

  • Execute a Ruby script:

ruby script.rb

  • Execute a single Ruby command in the command-line:

ruby -e command

  • Check for syntax errors on a given Ruby script:

ruby -c script.rb

  • Show the version of Ruby you are using:

ruby -v
