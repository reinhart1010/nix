# psysh

A runtime developer console, interactive debugger and REPL for PHP.
More information: <https://psysh.org>.

- Open a shell in the current directory:

`psysh`

- Open a shell in a specific directory:

`psysh --cwd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Use a specific configuration file:

`psysh --config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>
