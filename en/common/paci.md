# paci

A package manager for Bash scripts.
More information: <https://github.com/tradebyte/paci>.

- Update the list of available packages and versions (it's recommended to run this before other `paci` commands):

`paci refresh`

- Configure its behaviour:

`paci configure`

- Search for a given package:

`paci search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Install a package:

`paci install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Update a package:

`paci update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>
