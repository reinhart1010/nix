<hr># docsify

Initialize and serve markdown documentation.
More information: <https://cli.docsifyjs.org>.

- Initialize a new documentation in the current directory:

`docsify init`

- Initialize a new documentation in the specified directory:

`docsify init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Serve local documentation on `localhost:3000` with live reload:

`docsify serve `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Serve local documentation on `localhost` at the specified port:

`docsify serve --port `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">80</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Generate a sidebar markdown file in the specified directory:

`docsify generate `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
