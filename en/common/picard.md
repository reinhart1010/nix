# picard

Next generation MusicBrainz tagging application.
More information: <https://picard.musicbrainz.org/>.

- Start Picard:

`picard`

- Open a set of files:

`picard `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file1.mp3</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file2.mp3</span>

- Display the version of Picard installed:

`picard --long-version`
