# jcal

Display calendar information in the Jalali format, with the current day highlighted.
More information: <http://www.nongnu.org/jcal/>.

- Display a calendar for the current month:

`jcal`

- Display the previous, current, and next months:

`jcal -3`

- Display a calendar for a specific year (4 digits):

`jcal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">year</span>

- Display a calendar for a specific month and year:

`jcal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">year</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">month</span>
