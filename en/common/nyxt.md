<hr># nyxt

Nyxt is a keyboard-driven web browser for power users.
More information: <https://nyxt.atlas.engineer>.

- List all profiles:

`nyxt --list-data-profiles`

- Set the `init.lisp` file path:

`nyxt --init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Change the path to the auto-config file:

`nyxt --auto-config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Print system information:

`nyxt --system-information`
