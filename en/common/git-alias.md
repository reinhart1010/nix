# git alias

Create shortcuts for Git commands.
Part of `git-extras`.
More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>.

- List all aliases:

`git alias`

- Create a new alias:

`git alias "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">name</span>`" "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>`"`

- Search for an existing alias:

`git alias ^`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">name</span>
