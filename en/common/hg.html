hg

A command-line interface for Mercurial, a distributed source control management system. Some subcommands such as hg commit have their own usage documentation. More information: https://www.mercurial-scm.org.

  • Execute Mercurial command:

hg command

  • Call general help:

hg help

  • Call help on a command:

hg help command

  • Check the Mercurial version:

hg --version
