wget

In PowerShell, this command may be an alias of Invoke-WebRequest when the original wget program (https://www.gnu.org/software/wget) is not properly installed. Note: if version command returns an error, PowerShell may have substituted this command with Invoke-WebRequest. More information: https://learn.microsoft.com/powershell/module/microsoft.powershell.utility/invoke-webrequest.

  • View documentation for the original wget command:

tldr wget -p common

  • View documentation for PowerShell's Invoke-WebRequest command:

tldr invoke-webrequest

  • Display version:

wget --version
