Get-Content

Get the content of the item at the specified location. Note: This command can only be used through PowerShell. More information: https://learn.microsoft.com/powershell/module/microsoft.powershell.management/get-content.

  • Display the content of a file:

Get-Content -Path path\to\file

  • Display the first few lines of a file:

Get-Content -Path path\to\file -TotalCount 10

  • Display the content of the file and keep reading from it until Ctrl + C is pressed:

Get-Content -Path path\to\file -Wait
