pkg

Utilitaire de gestion de paquets pour Termux. Plus d'informations : https://wiki.termux.com/wiki/Package_Management.

  • Mets à jour tous les paquets déjà installés :

pkg upgrade

  • Installe un paquet :

pkg install paquet

  • Désinstalle un paquet :

pkg uninstall paquet

  • Réinstalle un paquet :

pkg reinstall paquet

  • Cherche un paquet :

pkg search paquet
