git fsck

Vérifier la validité et la connectivité des nœuds dans un dépôt Git. N'applique aucune modification. Voir git gc pour nettoyer. Plus d'informations : https://git-scm.com/docs/git-fsck.

  • Vérifier le registre courant :

git fsck

  • Lister tous les tags trouvés :

git fsck --tags

  • Lister tout les nœuds racine trouvés :

git fsck --root
