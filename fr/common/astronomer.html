astronomer

Outil qui détecte les étoiles non-légitimes d'un robot sur les projets GitHub. Plus d'informations : https://github.com/Ullaakut/astronomer.

  • Analyse un dépôt :

astronomer tldr-pages/tldr-node-client

  • Analyse le nombre maximum d'étoiles sur un dépôt :

astronomer tldr-pages/tldr-node-client --stars 50

  • Analyse un dépôt en affichant les rapports comparatifs :

astronomer tldr-pages/tldr-node-client --verbose
