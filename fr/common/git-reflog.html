git reflog

Affiche un log des changements locaux comme HEAD, tags et branches. Plus d'informations : https://git-scm.com/docs/git-reflog.

  • Afficher le reflog de HEAD :

git reflog

  • Affiche le reflog d'une branche spécifique :

git reflog nom_de_branche

  • Affiche les 5 dernières entrées dans le reflog :

git reflog -n|–max-count 5
