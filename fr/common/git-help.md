# git help

Afficher le manuel de Git.
Plus d'informations : <https://git-scm.com/docs/git-help>.

- Afficher le manuel d'une sous-commande :

`git help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Même chose dans un navigateur :

`git help --web `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Afficher la liste des sous-commandes disponibles :

`git help --all`

- Lister les manuels disponibles :

`git help --guide`

- Lister toutes les variables de configuration disponibles :

`git help --config`
