# which

Localise un programme dans le chemin de l'utilisateur.

- Fouille la variable d'environnement « PATH » et affiche l'emplacement des programmes exécutables correspondants à la requête :

`which `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">exécutable</span>

- Affiche tous les exécutables correspondants à la requête, s'il y en a plus qu'un :

`which -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">exécutable</span>
