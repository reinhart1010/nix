az

La CLI officielle de Microsoft Azure. Certaines commandes comme az login ont leur propre documentation. Plus d'informations : https://learn.microsoft.com/cli/azure/reference-index.

  • Connexion à Azure :

az login

  • Gère les informations de souscription Azure :

az account

  • Affiche tous les disques administrés Azure :

az disk list

  • Liste toutes les machines virtuelles Azure :

az vm list

  • Gère les services Kubernetes Azure :

az aks

  • Gère les ressources réseaux Azure :

az network

  • Démarre en mode interactif :

az interactive

  • Affiche l'aide :

az --help
