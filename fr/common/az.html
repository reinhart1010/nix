az

La CLI officielle de Microsoft Azure. Certaines commandes comme az login ont leur propre documentation. Plus d'informations : https://learn.microsoft.com/cli/azure.

  • Connexion à Azure :

az login

  • Gère les informations de souscription azure :

az account

  • Affiche tous les disques managés Azure :

az disk list

  • Liste toutes les machines virtuelles Azure :

az vm list

  • Gère Azure Kubernetes Services :

az aks

  • Gère les ressources réseaux Azure :

az network
