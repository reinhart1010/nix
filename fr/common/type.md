<hr># type

Affiche le type de la commande que le terminal exécuterait.
Plus d'information: <https://manned.org/type>.

- Affiche le type d'une commande :

`type `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">commande</span>

- Affiche tous les emplacements contenants l'exécutable spécifié :

`type -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">commande</span>

- Affiche le nom du fichier qui serait exécuté :

`type -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">commande</span>
