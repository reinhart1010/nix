choco

சாக்லேட்டி தொகுப்பு மேலாளருக்கான கட்டளை வரி இடைமுகம். install போன்ற சிலச் சார்கட்டளைகளுக்குத் தனிப் பக்கம் உள்ளது. மேலும் விவரத்திற்கு: https://chocolatey.org.

  • சாக்லேட்டி கட்டளையை இயக்கவும்:

choco கட்டளை

  • பொது உதவியை அழைக்கவும்:

choco -?

  • ஒரு குறிப்பிட்ட கட்டளையில் உதவியை அழைக்கவும்:

choco கட்டளை -?

  • சாக்லேட்டி பதிப்பைச் சரிபார்க்கவும்:

choco --version
