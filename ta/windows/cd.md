# cd

தற்போதைய பணி அடைவின் பெயரைக் காட்டுகிறது அல்லது மாற்றுகிறது.
மேலும் விவரத்திற்கு: <https://docs.microsoft.com/windows-server/administration/windows-commands/cd>.

- அதே இயக்ககத்தில் ஒரு கோப்பகத்திற்குச் செல்லவும்:

`cd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">கோப்பகத்திற்கான/பாதை</span>

- தற்போதைய கோப்பகத்தின் பெயரைக் காண்பி:

`cd`

- தற்போதைய கோப்பகத்தின் பெற்றோர் வரை செல்லுங்கள்:

`cd ..`

- வேறு இயக்ககத்தில் ஒரு கோப்பகத்திற்குச் செல்லவும்:

`cd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">கோப்பகத்திற்கான/பாதை</span>` /d`
