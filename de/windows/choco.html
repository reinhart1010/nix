choco

Ein Kommandozeilenwerkzeug für die Chocolatey Paketverwaltung. Manche Unterbefehle wie install sind separat dokumentiert. Weitere Informationen: https://chocolatey.org.

  • Führe einen Chocolatey-Befehl aus:

choco befehl

  • Zeige die allgemeine Hilfe an:

choco -?

  • Zeige die Hilfe für einen bestimmten Befehl an:

choco befehl -?

  • Überprüfe die Version von Chocolatey:

choco --version
