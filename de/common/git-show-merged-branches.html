git show-merged-branches

Gibt alle Branches aus, welche bereits in die aktuelle HEAD-Datei gemerget wurden. Weitere Informationen: https://github.com/tj/git-extras/blob/master/Commands.md#git-show-merged-branches.

git show-merged-branches
