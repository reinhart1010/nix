# tldr

Zeigt kurze Zusammenfassungen (tldr-Seiten) von Kommandozeilen-Befehlen an.
Weitere Informationen: <https://tldr.sh>.

- Zeige die tldr-Seite für einen Befehl an (Hinweis: So bist du hierher gekommen!):

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">befehl</span>

- Zeige die tldr-Seite für `cd` an und überschreibe die Standardplattform:

`tldr -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">android|linux|osx|sunos|windows</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">cd</span>

- Zeige die tldr-Seite für einen Unterbefehl:

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">git-checkout</span>

- Aktualisiere die lokalen Seiten (wenn er Client Caching unterstützt):

`tldr -u`
