git var

Bir Git mantıksal değişkeninin değerini yazdırır. Ayrıca bu komuttan daha çok tercih edilen git config'e bakılması önerilir. Daha fazla bilgi için: https://git-scm.com/docs/git-var.

  • Yerel bir Git mantıksal değişkeninin değerini yazdır:

git var GIT_AUTHOR_IDENT|GIT_COMMITTER_IDENT|GIT_EDITOR|GIT_PAGER

  • Tüm Git mantıksal değerlerini sırala:

git var -l
