# git repack

Bir Git deposundaki paketlenmemiş nesneleri paketle.
Daha fazla bilgi: <https://git-scm.com/docs/git-repack>.

- Mevcut dizindeki paketlenmemiş nesneleri paketle:

`git repack`

- Paketlemeden sonra gereksiz nesneleri sil:

`git repack -d`
