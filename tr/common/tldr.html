This entry is very new in the tldr-pages project, hence translation data is currently unavailable for a while.

tldr

Komut satırı araçları için tldr-pages projesinden basit yardım sayfaları görüntüler. Daha fazla bilgi için: https://github.com/tldr-pages/tldr/blob/main/CLIENT-SPECIFICATION.md#command-line-interface.

  • Bir komutun tipik kullanımını göster (ipucu: burayı görüntülemek için kullandığınız komutun aynısı!):

tldr komut

  • Linux için tar tldr sayfasını göster:

tldr -p linux tar

  • Bir Git alt komutu için yardım al:

tldr git-checkout

  • (Eğer alıcı önbellek oluşumunu destekliyorsa) Yerel paketleri güncelle:

tldr -u
