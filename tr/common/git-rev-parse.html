git rev-parse

Belirtilen sürümler için metaveri görüntüle. Daha fazla bilgi için: https://git-scm.com/docs/git-rev-parse.

  • Bir dalın commit verisini göster:

git rev-parse dal_ismi

  • Mevcut dal ismini göster:

git rev-parse --abbrev-ref HEAD

  • Kök dizinin mutlak konumunu göster:

git rev-parse --show-toplevel
