pkg

ابزار مدیر بسته ها برای ترماکس. اطلاعات بیشتر: https://wiki.termux.com/wiki/Package_Management.

  • ارتقا تمامی بسته های نصب شده :

pkg upgrade

  • نصب یک بسته :

pkg install package

  • حذف یک بسته :

pkg uninstall package

  • نصب دوباره یک بسته :

pkg reinstall package

  • جستجوی یک بسته :

pkg search package
