img2txt

이미지를 색상 ASCII 문자로 변환하고 텍스트 기반 색상 파일로 출력합니다. 더 많은 정보: https://manned.org/img2txt.

  • 출력 열 수를 특정 값으로 설정:

img2txt --width=10

  • 출력 행 수를 특정 값으로 설정:

img2txt --height=5

  • 출력 글꼴 너비를 특정 값으로 설정:

img2txt --font-width=12

  • 출력 글꼴 높이를 특정 값으로 설정:

img2txt --font-height=14

  • 이미지 밝기를 특정 값으로 설정:

img2txt --brightness=2
