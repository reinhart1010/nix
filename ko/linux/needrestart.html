needrestart

라이브러리 업그레이드 후 다시 시작해야 하는 데몬 확인. 더 많은 정보: https://github.com/liske/needrestart.

  • 오래된 프로세스 나열:

needrestart

  • 상호작용 모드로 서비스 다시 시작:

sudo needrestart

  • [v]자세히 또는 [q]조용히 모드에서 오래된 프로세스 나열:

needrestart -v|q

  • [k]커널이 오래되었는지 확인:

needrestart -k

  • CPU 마이크로코드가 오래되었는지 확인:

needrestart -w

  • [b]배치 모드에서 오래된 프로세스 나열:

needrestart -b

  • 특정 [c]구성 파일을 사용하여 오래된 프로세스 나열:

needrestart -c 경로/대상/설정

  • 도움말 표시:

needrestart --help
