smbpasswd

Samba 사용자 추가/제거 또는 비밀번호 변경. Samba 사용자는 기존의 로컬 유닉스 계정이 있어야 합니다. 더 많은 정보: https://manned.org/smbpasswd.8.

  • 현재 사용자의 SMB 비밀번호 변경:

smbpasswd

  • 지정된 사용자를 Samba에 추가하고 비밀번호 설정 (사용자는 시스템에 이미 존재해야 함):

sudo smbpasswd -a 사용자명

  • 기존 Samba 사용자의 비밀번호 수정:

sudo smbpasswd 사용자명

  • Samba 사용자 삭제 (유닉스 계정이 삭제된 경우에는 pdbedit 사용):

sudo smbpasswd -x 사용자명
