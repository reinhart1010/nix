cockpit-ws

브라우저 애플리케이션과 cockpit-bridge와 같은 다양한 구성 도구 및 서비스 간 통신. 더 많은 정보: https://cockpit-project.org/guide/latest/cockpit-ws.8.html.

  • 127.0.0.1에서 포트 22로 SSH 인증을 통해 시작:

cockpit-ws --local-ssh

  • 특정 포트에서 HTTP 서버 시작:

cockpit-ws --port 포트

  • 특정 IP 주소에 바인딩하여 시작 (기본값은 0.0.0.0):

cockpit-ws --address ip_주소

  • TLS 없이 시작:

cockpit-ws --no-tls

  • 도움말 표시:

cockpit-ws --help
