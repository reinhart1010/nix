script

터미널 출력을 파일로 기록합니다. 더 많은 정보: https://manned.org/script.

  • 현재 디렉토리의 typescript라는 이름의 파일에 새 세션 기록:

script

  • 사용자 지정 파일 경로에 새 세션 기록:

script 경로/대상/세션.out

  • 기존 파일에 추가하여 새 세션 기록:

script -a 경로/대상/세션.out

  • 타이밍 정보 기록 (stderr에 출력됩니다):

script -t 2> 경로/대상/타이밍_파일

  • 데이터가 발생하는 즉시 출력:

script -f 경로/대상/파일
