sed

스크립트 방식으로 텍스트 편집. 같이 보기: awk, ed. 더 많은 정보: https://keith.github.io/xcode-man-pages/sed.1.html.

  • 모든 입력 라인에서 apple(기본 정규식)을 mango(기본 정규식)로 모두 교체하고 결과를 stdout에 출력:

명령어 | sed 's/apple/mango/g'

  • 특정 스크립트 [f]파일을 실행하고 결과를 stdout에 출력:

명령어 | sed -f 경로/대상/스크립트_파일.sed

  • 모든 입력 라인에서 apple(확장 정규식)을 APPLE(확장 정규식)로 모두 교체하고 결과를 stdout에 출력:

명령어 | sed -E 's/(apple)/\U\1/g'

  • 첫 번째 라인만 stdout에 출력:

명령어 | sed -n '1p'

  • file 내 모든 apple(기본 정규식)을 mango(기본 정규식)로 교체하고 원본을 file.bak으로 백업 저장:

sed -i bak 's/apple/mango/g' 경로/대상/파일
