# cordova

HTML, CSS 및 JS가 포함된 모바일 앱.
더 많은 정보: <https://cordova.apache.org/docs/en/latest/guide/cli/>.

- cordova 프로젝트 생성:

`cordova create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지_명</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_명</span>

- 현재 작업 공간 상태 표시:

`cordova info`

- cordova 플랫폼 추가:

`cordova platform add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플랫폼</span>

- cordova 플랫폼 제거:

`cordova platform remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플랫폼</span>

- cordova 플러그인 추가:

`cordova plugin add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플러그인id</span>

- cordova 플러그인 제거:

`cordova plugin remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플러그인id</span>
