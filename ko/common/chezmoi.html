Chezmoi

Go로 작성된, 다중 머신 도트파일 관리자. 참고: stow, tuckr, vcsh, homeshick. 더 많은 정보: https://chezmoi.io.

  • chezmoi를 설정하고, ~/.local/share/chezmoi에 Git 저장소를 만듬:

chezmoi init

  • Git 저장소의 기존 도트 파일에서 chezmoi를 설정:

chezmoi init 레포지토리_주소

  • 하나 이상의 도트 파일 추적을 시작:

chezmoi add 경로/대상/도트파일1 경로/대상/도트파일2 …

  • 로컬 변경 사항으로 저장소 업데이트:

chezmoi re-add 경로/대상/도트파일1 경로/대상/도트파일2 …

  • 추적된 도트파일의 소스 상태를 편집:

chezmoi edit 경로/대상/도트파일_또는_심볼릭링크

  • 대기 중인 변경사항 보기:

chezmoi diff

  • 변경 사항을 적용:

chezmoi -v apply

  • 원격 저장소에서 변경 사항을 가져와 적용:

chezmoi update
