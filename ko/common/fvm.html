fvm

Flutter 버전 관리. 더 많은 정보: https://fvm.app/documentation/guides/basic-commands.

  • Flutter SDK 버전을 설치. 프로젝트 설정에 version 없이 사용:

fvm install 버전

  • 프로젝트에서 특정 버전의 Flutter SDK를 설치:

fvm use 버전 옵션

  • Flutter SDK의 글로벌 버전을 설정:

fvm global 버전

  • FVM 캐시 삭제:

fvm destroy

  • Flutter SDK의 특정 버전을 제거:

fvm remove 버전

  • Flutter SDK의 설치된 모든 버전을 나열:

fvm list

  • Flutter SDK의 모든 릴리스를 나열:

fvm releases
