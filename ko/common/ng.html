ng

Angular 애플리케이션 생성 및 관리 도구. 더 많은 정보: https://angular.io/cli.

  • 디렉토리 내에 새로운 Angular 애플리케이션 생성:

ng new 프로젝트_이름

  • 애플리케이션에 새 컴포넌트 추가:

ng generate component 컴포넌트_이름

  • 애플리케이션에 새 클래스 추가:

ng generate class 클래스_이름

  • 애플리케이션에 새 디렉티브 추가:

ng generate directive 디렉티브_이름

  • 루트 디렉토리에서 애플리케이션 실행:

ng serve

  • 애플리케이션 빌드:

ng build

  • 유닛 테스트 실행:

ng test

  • 현재 Angular 설치 버전 표시:

ng version
