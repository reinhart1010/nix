powershell

이 명령은 powershell 대신 pwsh를 사용하는 크로스 플랫폼 버전의 PowerShell(이전에는 PowerShell Core로 알려짐)과 혼동될 수 있습니다. Windows의 원본 powershell 명령은 레거시 Windows 버전의 PowerShell(버전 5.1 이하)을 사용하는 데 여전히 사용할 수 있습니다. 더 많은 정보: https://learn.microsoft.com/powershell/module/microsoft.powershell.core/about/about_pwsh.

  • 최신 크로스 플랫폼 버전의 PowerShell(버전 6 이상)의 명령에 대한 문서 보기:

tldr pwsh

  • 레거시 Windows PowerShell(버전 5.1 이하)의 명령에 대한 문서 보기:

tldr powershell -p windows
