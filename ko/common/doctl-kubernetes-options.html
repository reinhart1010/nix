doctl kubernetes options

doctl의 Kubernetes 명령에 사용할 수 있는 값을 가져옴. 더 많은 정보: https://docs.digitalocean.com/reference/doctl/reference/kubernetes/options/.

  • Kubernetes 클러스터를 지원하는 지역 목록 나열:

doctl kubernetes options regions

  • Kubernetes 클러스터에서 사용할 수 있는 머신 크기를 나열:

doctl kubernetes options sizes

  • DigitalOcean 클러스터와 함께 사용할 수 있는 Kubernetes 버전을 나열:

doctl kubernetes options versions
