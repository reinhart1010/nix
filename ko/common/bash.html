bash

Bourne-Again SHell. sh-호환 명령 행 인터프리터. 더 많은 정보: https://gnu.org/software/bash/.

  • 대화식 쉘 시작:

bash

  • 명령 실행:

bash -c "command"

  • 파일에서 명령 실행:

bash file.sh

  • 파일에서 명령 실행하고, 터미널에서 실행 된 모든 명령 기록:

bash -x file.sh

  • 파일에서 명령 실행하고, 첫 번째 에러에서 중지:

bash -e file.sh

  • stdin에서 명령 실행:

bash -s

  • bash의 버전 정보 출력 (echo $BASH_VERSION을 사용하여 버전 문자열만 표시):

bash --version
