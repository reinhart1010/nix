hugo server

Hugo의 내장 웹서버로 사이트를 구축하고 제공. 더 많은 정보: https://gohugo.io/commands/hugo_server/.

  • 사이트 구축 및 서빙:

hugo server

  • 지정된 포트 번호에서 사이트 구축 및 제공:

hugo server --port 포트_번호

  • 지원되는 출력 형식 (HTML, XML 등)을 최소화하면서 사이트를 구축하고 제공:

hugo server --minify

  • 도움말 표시:

hugo server --help
