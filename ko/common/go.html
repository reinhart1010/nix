go

Go 소스 코드를 관리. build와 같은 일부 하위 명령에는 자체 사용 설명서가 있습니다. 더 많은 정보: https://golang.org.

  • 패키지 다운로드 및 설치 (import 경로로 지정):

go get 패키지_경로

  • 소스 파일 컴파일 및 실행 (main 패키지를 포함해야 함):

go run 파일.go

  • 소스 파일을 지정한 이름의 실행 파일로 컴파일:

go build -o 실행_파일 파일.go

  • 현재 디렉토리에 있는 패키지 컴파일:

go build

  • 현재 패키지의 모든 테스트 케이스 실행 (_test.go로 끝나야 함):

go test

  • 현재 패키지 컴파일 및 설치:

go install

  • 현재 디렉토리에 새 모듈 초기화:

go mod init 모듈_이름
