wal

배경화면의 주요 색상을 기반으로 색상 테마 생성. 더 많은 정보: https://github.com/dylanaraps/pywal.

  • 색상 테마 미리보기:

wal --preview 이미지.png

  • 색상 테마 생성:

wal -i 이미지.png

  • 밝은 색상 테마 생성:

wal -i 이미지.png -l

  • 데스크탑 배경화면 설정 건너뛰기:

wal -i 이미지.png -n

  • 터미널 색상 설정 건너뛰기:

wal -i 이미지.png -s

  • 이전에 생성된 색상 테마와 배경화면 복원:

wal -R
