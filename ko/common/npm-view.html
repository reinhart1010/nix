npm view

패키지에 대한 레지스트리 정보를 조회. 더 많은 정보: https://docs.npmjs.com/cli/commands/npm-view.

  • 패키지의 최신 버전에 대한 정보 조회:

npm view 패키지

  • 특정 버전의 패키지 정보 조회:

npm view 패키지@버전

  • 패키지의 모든 사용 가능한 버전 조회:

npm view 패키지 versions

  • 패키지 설명 조회:

npm view 패키지 description

  • 패키지의 최신 버전의 의존성 조회:

npm view 패키지 dependencies

  • 패키지의 저장소 URL 조회:

npm view 패키지 repository

  • 패키지의 관리자 조회:

npm view 패키지 maintainers
