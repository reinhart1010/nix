# dir

列出目錄的內容。
更多資訊：<https://docs.microsoft.com/windows-server/administration/windows-commands/dir>.

- 顯示當前目錄的內容：

`dir`

- 顯示指定目錄的內容：

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目錄/完整/路徑</span>

- 顯示當前目錄的內容，包括隱藏的文件：

`dir /A`

- 顯示指定目錄的內容，包括隱藏的文件：

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目錄/完整/路徑</span>` /A`
