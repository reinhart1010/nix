# cd

改變工作目錄。
更多資訊：<https://manned.org/cd>.

- 前往指定目錄：

`cd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目錄/完整/路徑</span>

- 前往目前使用者的家目錄：

`cd`

- 前往目前目錄的上層目錄：

`cd ..`

- 前往之前所在的目錄：

`cd -`
