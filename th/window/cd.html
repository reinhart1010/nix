cd

แสดงชื่อหรือเปลี่ยนไดเร็กทอรีการทำงานปัจจุบัน ข้อมูลเพิ่มเติม: https://learn.microsoft.com/en-us/windows-server/administration/windows-commands/cd

  • แสดงชื่อของไดเร็กทอรีปัจจุบัน:

cd

  • ไปที่รูทของไดรฟ์ปัจจุบัน:

cd \

  • ไปที่ไดเร็กทอรีในไดรฟ์เดียวกัน:

cd ทาง\ไป\สารบบ

  • ไปที่พาเรนต์ของไดเร็กทอรีปัจจุบัน:

cd ..

  • ไปที่ไดเร็กทอรีที่อยู่ต่างไดรฟ์:

cd ทาง\ไป\สารบบ /d
