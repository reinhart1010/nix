# pkgadd

Afegeix un paquet a un sistema CRUX.
Més informació: <https://docs.oracle.com/cd/E19253-01/816-5166/pkgadd-1m/index.html>.

- Instal·la un paquet de software local:

`pkgadd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nom_paquet</span>

- Actualitza un paquet ja instal·lat a partir de un paquet local:

`pkgadd -u `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nom_paquet</span>
