bugreportz

Genera un informe comprimido de errores de Android. Este comando solo se puede usar a través de adb shell. Más información: https://cs.android.com/android/platform/superproject/+/main:frameworks/native/cmds/bugreportz.

  • Genera un informe completo de errores comprimido de un dispositivo Android:

bugreportz

  • Muestra el progreso de una operación bugreportz en ejecución:

bugreportz -p

  • Muestra ayuda:

bugreportz -h

  • Muestra la versión de bugreportz:

bugreportz -v
