az upgrade

Actualiza Azure CLI y sus extensiones. Parte de azure-cli (también conocido como az). Más información: https://learn.microsoft.com/cli/azure/reference-index?view=azure-cli-latest#az-upgrade.

  • Actualiza Azure CLI:

az upgrade

  • Actualiza Azure CLI y sus extensiones:

az upgrade --all

  • Actualiza Azure CLI y sus extensiones sin solicitar confirmación:

az version --all --yes
