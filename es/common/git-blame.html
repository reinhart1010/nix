git blame

Muestra el hash del commit y el último autor de cada línea de un archivo. Más información: https://git-scm.com/docs/git-blame.

  • Muestra el archivo con el nombre del autor y el hash del commit en cada línea:

git blame archivo

  • Muestra el archivo con correo electrónico del autor y hash del commit en cada línea:

git blame -e archivo
