# [

Evalúa una condicion.
Retorna 0 si la condición se cumple, 1 si esta no se cumple.
Más información: <https://www.gnu.org/software/coreutils/test>.

- Prueba si una variable dada es igual a una cadena dada:

`[ "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">$VARIABLE</span>`" == "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/bin/zsh</span>`" ]`

- Prueba si una variable dada está vacía:

`[ -z "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">$BRANCH_DE_GIT</span>`" ]`

- Prueba si un archivo existe:

`[ -f "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo</span>`" ]`

- Prueba si un directorio no existe:

`[ ! -d "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/directorio</span>`" ]`

- Sentencia if-else:

`[ `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">condicion</span>` ] && `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">echo "verdadero"</span>` || `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">echo "falso"</span>
