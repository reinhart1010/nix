# vimtutor

Vim tutor, enseña los comandos básicos de vim.

- Ejecuta vim tutor utilizando el idioma especificado (en, es, de, ...):

`vimtutor `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">idioma</span>

- Sale del tutor:

`<Esc> :q <Enter>`
