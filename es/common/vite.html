Vite

Crea un proyecto Vite. Se utiliza para crear proyectos JavaScript. Plantillas disponibles: vanilla, vanilla-ts, vue, vue-ts, react, react-ts, react-swc, react-swc-ts, preact, preact-ts, lit, lit-ts, svelte, svelte-ts. Más información: https://vitejs.dev/guide.

  • Configuración usando npm 6.x:

npm create vite@latest my-react-app --template react-ts

  • Configuración usando npm 7+, se necesita un guión doble adicional:

npm create vite@latest my-react-app -- --template react-ts

  • Configuración usando yarn:

yarn create vite my-react-app --template react-ts

  • Configuración usando pnpm:

pnpm create vite my-react-app --template react-ts
