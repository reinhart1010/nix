git gc

Optimiza el repositorio local eliminando archivos innecesarios. Más información: https://git-scm.com/docs/git-gc.

  • Optimiza el repositorio:

git gc

  • Optimiza agresivamente (tarda más):

git gc --aggressive

  • Optimiza sin eliminar objetos sueltos (por defecto los elimina):

git gc --no-prune

  • Suprime toda la salida:

git gc --quiet

  • Muestra todas sus funcionalidades:

git gc --help
