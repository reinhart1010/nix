launchd

Gestiona procesos, tanto para el sistema como para los usuarios. No puedes invocar launchd manualmente, usa launchctl para interactuar con él. Más información: https://developer.apple.com/library/archive/documentation/MacOSX/Conceptual/BPSystemStartup/Chapters/Introduction.html.

  • Ejecuta init:

/sbin/launchd

  • Consulta la documentación para interactuar con launchd usando launchctl:

tldr launchctl
