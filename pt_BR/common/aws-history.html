aws history

Exibe o histórico dos comandos para o AWS CLI (o armazenamento do hisitórico dos comandos do AWS CLI deve estar habilitado). Mais informações: https://docs.aws.amazon.com/cli/latest/reference/history/.

  • Lista histórico dos comandos e seus IDs:

aws history list

  • Exibe eventos relacionados a um comando específico dado um ID do comando:

aws history show id_do_comando
