cupsaccept

Aceita trabalhos enviados para um ou mais destinos. NOTA: destino se refere a uma impressora ou uma classe de impressoras. Veja também: cupsreject, cupsenable, cupsdisable, lpstat. Mais informações: https://www.cups.org/doc/man-cupsaccept.html.

  • Aceita trabalhos de impressão para os destinos especificados:

cupsaccept destino1 destino2 …

  • Especifica um servidor diferente:

cupsaccept -h server destino1 destino2 …
