choco

Uma interface de linha de comando para o gerenciador de pacotes Chocolatey. Alguns subcomandos como choco install tem sua própia documentação de uso. Mais informações: https://chocolatey.org.

  • Executar o comando Chocolatey:

choco comando

  • Exibir ajuda generalizada:

choco -?

  • Exibir ajuda em um comando específico:

choco comando -?

  • Exibir a versão do Chocolatey:

choco --version
