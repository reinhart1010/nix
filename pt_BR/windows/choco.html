choco

Uma interface de linha de comando para o gerenciador de pacotes Chocolatey. Alguns subcomandos como install tem sua própia documentação de uso. Mais informações: https://chocolatey.org.

  • Executa o comando Chocolatey:

choco comando

  • Exibe ajuda generalizada:

choco -?

  • Exibe ajuda em um comando específico:

choco comando -?

  • Exibe a versão do Chocolatey:

choco --version
