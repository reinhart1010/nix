pkg

Gestor de pacotes para o Termux. Mais informações: https://wiki.termux.com/wiki/Package_Management.

  • Atualiza todos os pacotes instalados:

pkg upgrade

  • Instala um pacote:

pkg install pacote

  • Desinstala um pacote:

pkg uninstall pacote

  • Reinstala um pacote:

pkg reinstall pacote

  • Procura um pacote:

pkg search pacote
