winget

Windows 软件包管理器命令行工具。 更多信息：https://learn.microsoft.com/windows/package-manager/winget.

  • 安装一个软件包：

winget install package

  • 显示关于一个软件包的信息：

winget show package

  • 查找一个软件包：

winget search package
