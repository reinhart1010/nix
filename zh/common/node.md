# node

服务器后端 JavaScript 平台（Node.js）。
更多信息：<https://nodejs.org>.

- 运行一个 JavaScript 文件：

`node `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">文件名</span>

- 开始一个 REPL 交互式解释器：

`node`

- 执行输入的 JavaScript 代码：

`node -e "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">代码</span>`"`

- 执行输入的 JavaScript 代码并显示结果：

`node -p "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">代码</span>`"`

- 启动检查器并在程序源码解析完成后等待调试器连接：

`node --no-lazy --inspect-brk `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">文件名</span>
