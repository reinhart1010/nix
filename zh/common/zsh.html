zsh

Z SHell. 与 bash 和 sh 兼容的命令行解释器。 更多信息：https://www.zsh.org.

  • 启动交互式解释器：

zsh

  • 执行从参数传入的命令：

zsh -c command

  • 从文件执行命令（脚本）：

zsh file

  • 从文件执行命令并将执行过的命令打印到终端：

zsh -x file
