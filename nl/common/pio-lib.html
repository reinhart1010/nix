pio lib

Beheer PlatformIO bibliotheken. Meer informatie: https://docs.platformio.org/en/latest/core/userguide/lib/.

  • Toon ge誰nstalleerde bibliotheken:

pio lib list

  • Toon ingebouwde bibliotheken gebaseerd op ge誰nstalleerde ontwikkelplatformen en hun geraamtes:

pio lib builtin

  • Zoek naar bestaande bibliotheken:

pio lib search trefwoord

  • Toon details over een bibliotheek:

pio lib show bibliotheek

  • Installeer een bibliotheek:

pio lib install bibliotheek

  • Update ge誰nstalleerde bibliotheken:

pio lib update

  • De誰nstalleer een bibliotheek:

pio lib uninstall bibliotheek

  • Toon PlatformIO bibliotheek register statistieken:

pio lib stats
