crane ls

Toon de tags in een repository. Meer informatie: https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_ls.md.

  • Toon de tags:

crane ls repository

  • Print de volledige image-referentie:

crane ls repository --full-ref

  • Sla digest-tags over:

crane ls -o|–omit-digest-tags

  • Toon de help:

crane ls -h|–help
