choco

De Chocolatey pakket manager. Sommige subcommando's zoals install hebben hun eigen gebruiksdocumentatie. Meer informatie: https://chocolatey.org.

  • Voer een Chocolatey commando uit:

choco commando

  • Toon de algemene help:

choco -?

  • Toon de help voor een specifiek commando:

choco commando -?

  • Bekijk de Chocolatey versie:

choco --version
