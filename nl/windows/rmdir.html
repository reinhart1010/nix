rmdir

Verwijdert een map en zijn inhoud. In PowerShell is deze opdracht een alias van Remove-Item. Deze documentatie is gebaseerd op de Command Prompt (cmd) versie van rmdir. Meer informatie: https://learn.microsoft.com/windows-server/administration/windows-commands/rmdir.

  • Bekijk de documentatie van het equivalente PowerShell-commando:

tldr remove-item

  • Verwijder een lege map:

rmdir pad/naar/map

  • Verwijder een map en zen inhoud recursief:

rmdir pad/naar/map /s

  • Verwijder een map en zen inhoud recursief zonder te vragen:

rmdir pad/naar/map /s /q
