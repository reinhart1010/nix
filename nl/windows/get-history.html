Get-History

Toon de PowerShell commando-geschiedenis. Let op: dit commando kan alleen gebruikt worden via PowerShell. Meer informatie: https://learn.microsoft.com/powershell/module/microsoft.powershell.core/get-history.

  • Toon de commando-geschiedenis met ID:

Get-History

  • Haal het PowerShell geschiedenis-item op via een ID:

Get-History -Id id

  • Toon de laatste N commando's:

Get-History -Count 10
