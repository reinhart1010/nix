dmesg

Schrijft de kernel berichten naar de standaard output. Meer informatie: https://www.unix.com/man-page/sunos/1m/dmesg.

  • Toont kernel berichten:

dmesg

  • Toont hoeveel fysiek geheugen beschikbaar is op het systeem:

dmesg | grep -i memory

  • Toont kernel berichten per pagina:

dmesg | less
