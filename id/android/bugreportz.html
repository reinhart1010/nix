bugreportz

Buat sebuah laporan masalah Android dalam format file arsip (zip). Perintah ini hanya dapat digunakan di dalam adb shell. Informasi lebih lanjut: https://cs.android.com/android/platform/superproject/+/main:frameworks/native/cmds/bugreportz.

  • Buatkan sebuah arsip laporan masalah perangkat Android secara lengkap:

bugreportz

  • Tunjukkan kemajuan terhadap proses bugreportz yang sedang berlangsung:

bugreportz -p

  • Tulis isi suatu laporan masalah Android menuju stdout:

bugreportz -s

  • Tampilkan teks bantuan:

bugreportz -h

  • Tampilkan versi program:

bugreportz -v
