brew search

Cari kumpulan paket cask dan formula. Informasi lebih lanjut: https://docs.brew.sh/Manpage#search--s-options-textregex-.

  • Cari kumpulan cask dan formula menggunakan suatu kata kunci:

brew search kata_kunci

  • Cari kumpulan cask dan formula menggunakan ekspresi reguler (regex):

brew search /ekspresi_reguler/

  • Cari pula menurut deskripsi paket:

brew search --desc kata_kunci

  • Hanya cari kumpulan paket formula:

brew search --formula kata_kunci

  • Hanya cari kumpulan paket cask:

brew search --cask kata_kunci
