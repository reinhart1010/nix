git cp

Salin suatu berkas menuju lokasi baru dengan menyimpan riwayat perubahan atas berkas tersebut. Bagian dari git-extras. Informasi lebih lanjut: https://github.com/tj/git-extras/blob/master/Commands.md#git-cp.

  • Salin suatu berkas dalam suatu repositori Git, menuju tujuan pada direktori yang sama:

git cp nama_berkas nama_berkas_baru

  • Salin berkas menuju tujuan yang lain:

git cp jalan/menuju/berkas jalan/menuju/berkas_baru
