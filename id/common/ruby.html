ruby

Interpreter bahasa pemrograman Ruby. Informasi lebih lanjut: https://www.ruby-lang.org.

  • Memulai REPL (shell interaktif):

irb

  • Menjalankan skrip Ruby:

ruby lokasi/ke/script.rb

  • Menjalankan sebuah perintah Ruby dalam command-line:

ruby -e command

  • Memeriksa kesalahan sintaks dari skrip Ruby:

ruby -c lokasi/ke/script.rb

  • Menampilkan versi Ruby yang anda gunakan:

ruby -v
