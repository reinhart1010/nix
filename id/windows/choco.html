choco

Manajer paket Chocolatey. Beberapa subperintah seperti install mempunyai dokumentasi terpisah. Informasi lebih lanjut: https://chocolatey.org.

  • Jalankan suatu perintah Chocolatey:

choco perintah

  • Tampilkan bantuan:

choco -?

  • Tampilkan bantuan untuk suatu perintah:

choco perintah -?

  • Tampilkan versi aplikasi:

choco --version
