krita

Krita - программа для создания эскизов и рисования, разработанная для цифровых художников. Смотрите также: gimp. Больше информации: https://docs.krita.org/en/reference_manual/linux_command_line.html.

  • Запустить krita:

krita

  • Запустить без заставки:

krita --nosplash

  • Открыть указанные файлы:

krita путь/к/изображению1 путь/к/изображению2 …

  • Запустить с указанным рабочим пространством (Animation):

krita --workspace Animation

  • Запустить в полноэкранном режиме:

krita --fullscreen
