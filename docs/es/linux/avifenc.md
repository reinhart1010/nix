# avifenc

Codificador de formato de archivo de imagen AV1 (AVIF).
Más información: <https://aomediacodec.github.io/av1-avif/>.

- Convierte una imagen PNG específica a AVIF:

`avifenc `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/entrada.png</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/salida.avif</span>

- Codifica con una velocidad específica (6=predeterminado, 0=el más lento y 10=el más rápido):

`avifenc --speed `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">2</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/entrada.png</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/salida.avif</span>
