# distrobox-rm

Quita un contenedor Distrobox. Vea también: `tldr distrobox`.
Más información: <https://distrobox.it/usage/distrobox-rm>.

- Quita un contenedor Distrobox (Consejo: Detenga el contenedor antes de retirarlo):

`distrobox-rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_contenedor</span>

- Quita un contenedor Distrobox a la fuerza:

`distrobox-rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_contenedor</span>` --force`
