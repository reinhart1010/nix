# fincore

Muestra cuánta memoria caché está ocupando un archivo.
Más información: <https://manned.org/fincore>.

- Muestra los detalles de la caché de un archivo:

`fincore `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo</span>

- Muestra todas las columnas de datos posibles:

`fincore --output-all `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo</span>
