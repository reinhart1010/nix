# distrobox-stop

Detiene un contenedor Distrobox. Vea también: `tldr distrobox`.
Más información: <https://distrobox.it/usage/distrobox-stop>.

- Detiene un contenedor Distrobox:

`distrobox-stop `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_contenedor</span>

- Detiene un contenedor Distrobox de un modo no interactivo (no pide confirmación):

`distrobox-stop --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_contenedor</span>` --yes`
