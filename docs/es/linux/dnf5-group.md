# dnf5 group

Este comando es un alias de `dnf group`.
Nota: desde Fedora 37 a 40 (inclusive), `dnf` ejecuta DNF v4 mientras que `dnf5` ejecuta DNF v5.

- Vea la documentación del comando original:

`tldr dnf group`
