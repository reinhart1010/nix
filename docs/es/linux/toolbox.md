# toolbox

Gestiona entornos de línea de comandos en contenedores de Linux.
Algunos subcomandos como `create` tienen su propia documentación de uso.
Más información: <https://manned.org/toolbox.1>.

- Ejecuta un subcomando de `toolbox`:

`toolbox `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcomando</span>

- Muestra ayuda para un subcomando específico (como `create`, `enter`, `rm`, `rmi`, etc.):

`toolbox help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcomando</span>

- Muestra ayuda:

`toolbox --help`

- Muestra la versión:

`toolbox --version`
