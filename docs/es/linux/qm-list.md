# qm list

Lista todas las máquinas virtuales.
Más información: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Lista todas las máquinas virtuales:

`qm list`

- Lista todas las máquinas virtuales con el estado completo de las que se están ejecutando:

`qm list --full 1`
