# apt-add-repository

Gestiona las definiciones del repositorio APT.
Más información: <https://manned.org/apt-add-repository.1>.

- Añade un nuevo repositorio APT:

`apt-add-repository `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositorio</span>

- Elimina un repositorio APT:

`apt-add-repository --remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositorio</span>

- Actualiza la caché de paquetes tras añadir un repositorio:

`apt-add-repository --update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositorio</span>

- Activa las fuentes de paquetes:

`apt-add-repository --enable-source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositorio</span>
