# dos2unix

Cambia saltos de línea con formato DOS a saltos de línea con formato Unix.
Reemplaza CRLF con LF.
Más información: <https://manned.org/dos2unix>.

- Cambia los saltos de línea de un archivo:

`dos2unix `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_archivo</span>

- Crea una copia con saltos de línea en formato Unix:

`dos2unix -n `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_archivo</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_archivo</span>
