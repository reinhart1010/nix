# amass

Herramienta de mapeo de superficie de ataque en profundidad y descubrimiento de activos.
Algunos subcomandos como `amass db` tienen su propia documentación de uso.
Más información: <https://github.com/OWASP/Amass>.

- Ejecuta un subcomando Amass:

`amass `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Muestra ayuda:

`amass -help`

- Muestra ayuda sobre un subcomando de Amass (como `intel`, `enum`, etc.):

`amass -help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Muestra la versión:

`amass -version`
