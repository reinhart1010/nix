# apm

Editor Atom Package Manager.
Vea `atom`.
Más información: <https://github.com/atom/apm>.

- Instala un paquete de <http://atom.io/packages> o un tema de <http://atom.io/themes>:

`apm install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_paquete</span>

- Elimina un paquete/tema:

`apm remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_paquete</span>

- Actualiza un paquete/tema:

`apm upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_paquete</span>
