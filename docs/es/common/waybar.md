# waybar

Barra Wayland altamente personalizable para compositores basados en Sway y Wlroots.
Más información: <https://github.com/Alexays/Waybar>.

- Inicia `waybar` con la configuración y hoja de estilos predeterminada:

`waybar`

- Usa un archivo de configuración diferente:

`waybar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-c|--config</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/archivo_de_configuración.jsonc</span>

- Utiliza un archivo de hoja de estilo diferente:

`waybar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-s|--style</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/a/hoja_de_estilo.css</span>

- Establece el nivel de registro:

`waybar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-l|--log-level</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">trace|debug|info|warning|error|critical|off</span>
