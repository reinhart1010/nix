<hr># deno

Un entorno de ejecución seguro para JavaScript y TypeScript.
Más información: <https://deno.land>.

- Ejecuta un archivo JavaScript o TypeScript:

`deno run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo.ts</span>

- Inicia un REPL (intérprete de comandos interactivo):

`deno`

- Ejecuta un archivo con acceso a la red habilitado:

`deno run --allow-net `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo.ts</span>

- Ejecuta un archivo desde una URL:

`deno run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://deno.land/std/examples/welcome.ts</span>

- Instala un archivo de secuencia de comandos ejecutable desde una URL:

`deno install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://deno.land/std/examples/colors.ts</span>
