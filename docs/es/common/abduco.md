<hr># abduco

Administrador de sesión de terminal.
Más información: <http://www.brain-dump.org/projects/abduco/>.

- Lista sesiones:

`abduco`

- Adjunta a una sesión, creándola si no existe:

`abduco -A `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">bash</span>

- Adjunta a una sesión con `dvtm`, creándola si no existe:

`abduco -A `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre</span>

- Separarse de una sesión:

`Ctrl + \`

- Adjunta a una sesión en modo solo-lectura:

`abduco -Ar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre</span>
