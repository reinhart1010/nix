# basename

Remueve nombres de directorios al inicio de una ruta.
Más información: <https://www.gnu.org/software/coreutils/manual/html_node/basename-invocation.html>.

- Imprime el nombre de un fichero a partir de su ruta:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo</span>

- Imprime el componente terminal de la ruta de un directorio:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/directorio</span>

- Imprime el nombre de un fichero sin un sufijo:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ruta/al/archivo</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sufijo</span>
