# nest

Herramienta de línea de comandos para inicializar, desarrollar y mantener aplicaciones Nest.
Más información: <https://docs.nestjs.com/cli/overview>.

- Muestra la información sobre la versión de nest instalada:

`nest info`

- Crea un nuevo proyecto NestJS en un directorio con el mismo nombre:

`nest new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_proyecto</span>

- Construye un proyecto NestJS específico:

`nest build `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_proyecto</span>

- Ejecuta un proyecto NestJS específico:

`nest start `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_del_proyecto</span>

- Importa una librería al proyecto NestJS actual:

`nest add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nombre_de_la_librería</span>
