# betterlockscreen

简洁的锁屏程序。
更多信息：<https://github.com/pavanjadhaw/betterlockscreen>.

- 锁定屏幕：

`betterlockscreen --lock`

- 更换锁屏背景：

`betterlockscreen -u `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">图片.png</span>

- 锁屏并显示一些自定义文本：

`betterlockscreen -l pixel -t "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">自定义锁屏文本</span>`"`

- 锁屏并自定义显示屏关闭超时秒数：

`betterlockscreen --off `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">5</span>` -l`
