# a2enconf

在基于 Debian 的操作系统上启用 Apache 配置文件。
更多信息：<https://manpages.debian.org/latest/apache2/a2enconf.8.en.html>.

- 启用配置文件：

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">配置文件</span>

- 不显示信息性消息：

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">配置文件</span>
