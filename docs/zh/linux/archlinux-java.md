# archlinux-java

提供 Java 环境设置功能的一个帮助脚本。
更多信息：<https://github.com/michaellass/archlinux-java-run>.

- 列出已安装的 Java 环境：

`archlinux-java status`

- 设置默认的 Java 环境：

`archlinux-java set `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">java_environment</span>

- 取消默认的 Java 环境：

`archlinux-java unset`

- 自动设置默认的 Java 环境：

`archlinux-java fix`
