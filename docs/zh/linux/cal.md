# cal

打印日历信息，高亮显示当前日期。
更多信息：<https://manned.org/cal>.

- 显示当前月份的日历：

`cal`

- 显示上个月、当前月和下个月：

`cal -3`

- 使用星期一作为一周的第一天：

`cal --monday`

- 显示特定年份的日历（4位数字）：

`cal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">年份</span>

- 显示特定月份和年份的日历：

`cal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">月份</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">年份</span>
