# uname

打印当前计算机及其上运行的操作系统的详细信息。
注意：有关操作系统的其他信息，请尝试使用 `sw-vers` 命令。
更多信息：<https://ss64.com/osx/uname.html>.

- 打印内核名称：

`uname`

- 打印系统架构和处理器信息：

`uname -mp`

- 打印内核名称、内核版本号和内核版本详细信息：

`uname -srv`

- 打印系统主机名：

`uname -n`

- 打印所有可用的系统信息：

`uname -a`
