# sw_vers

打印 macOS 操作系统的版本信息。
更多信息：<https://keith.github.io/xcode-man-pages/sw_vers.1.html>.

- 打印所有可用信息（操作系统名称、版本号和构建版本）：

`sw_vers`

- 只打印操作系统的版本号：

`sw_vers -productVersion`

- 只打印构建版本的标识符：

`sw_vers -buildVersion`
