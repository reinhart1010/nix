# pbpaste

将剪贴板的内容发送到标准输出。
相当于在键盘上按下 Cmd + V.
更多信息：<https://ss64.com/osx/pbpaste.html>.

- 将剪贴板的内容写入文件：

`pbpaste > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 将剪贴板的内容用作命令的输入：

`pbpaste | grep foo`
