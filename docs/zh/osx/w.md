# w

显示谁登录以及他们在做什么。
打印用户登录、tty、远程主机、登录时间、空闲时间、当前进程。
更多信息：<https://ss64.com/osx/w.html>.

- 显示登录用户信息：

`w`

- 显示没有标题的登录用户信息：

`w -h`

- 显示有关已登录用户的信息，按其空闲时间排序：

`w -i`
