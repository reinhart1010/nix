# uptime

告知当前系统运行多长时间和其他信息。
更多信息：<https://www.gnu.org/software/coreutils/manual/html_node/uptime-invocation.html>.

- 打印当前时间，运行时间，登录用户数量和其他信息：

`uptime`

- 仅显示系统已启动的时间长度：

`uptime --pretty`

- 打印系统启动的日期和时间：

`uptime --since`

- 显示版本信息：

`uptime --version`
