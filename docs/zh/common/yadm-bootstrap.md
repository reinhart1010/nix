# yadm-bootstrap

执行 Yadm 的 bootstrap 文件。
这个文件应创建在 `$HOME/.config/yadm/bootstrap`。
更多信息：<https://yadm.io/docs/bootstrap>.

- 执行 bootstrap 可执行文件：

`yadm bootstrap`
