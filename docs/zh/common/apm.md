# apm

Atom 编辑器的包管理工具。
参见 `atom`.
更多信息：<https://github.com/atom/apm>.

- 下载包：<http://atom.io/packages> 和主题 <http://atom.io/themes>：

`apm install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">包名</span>

- 移除包 / 主题：

`apm remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">包名</span>

- 升级包 / 主题：

`apm upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">包名</span>
