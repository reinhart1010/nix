更多信息：<https://thunderbird.net>.

`thunderbird`

- 使用特定用户配置文件：

`thunderbird -P `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">配置文件名称</span>

- 使用特定用户配置文件夹：
