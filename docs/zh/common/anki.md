# anki

强大、智能的记忆卡片软件。
更多信息：<https://docs.ankiweb.net>.

- 启动 `anki`：

`anki`

- 用一个特定的配置文件启动 `anki`：

`anki -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">配置文件名称</span>

- 以特定语言启动 `anki`：

`anki -l `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">语言</span>

- 从一个特定的目录而不是默认的（`~/Anki`）启动`anki`：

`anki -b `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/目录</span>
