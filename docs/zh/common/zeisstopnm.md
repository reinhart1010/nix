# zeisstopnm

将 Zeiss 共聚焦文件转换为 Netbpm 格式。
更多信息：<https://manned.org/zeisstopnm.1>.

- 将 Zeiss 共聚焦文件转换为 `.pgm` 或 `.ppm` 格式：

`zeisstopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 将 Zeiss 共聚焦文件转换为 Netbpm 格式，并明确指定目标文件类型：

`zeisstopnm -`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pgm|ppm</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>
