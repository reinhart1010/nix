# yadm-upgrade

将 `yadm` 升级到最新版本。
升级将尝试将你的子模块反初始化并重新初始化。
更多信息：<https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#commands>.

- 将 `yadm` 升级到最新版本：

`yadm upgrade`

- 强制升级，无论有无更改：

`yadm upgrade -f`
