# yadm-introspect

查看由 `yadm` 管理的数据。
内省的目的是支持命令行补全。
更多信息：<https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#commands>.

- 输出命令：

`yadm introspect commands`

- 输出配置：

`yadm introspect configs`

- 为主 `yadm` 命令输出开关：

`yadm introspect switches`

- 输出仓库：

`yadm introspect repo`
