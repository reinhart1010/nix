# unp

解压任意归档文件。
需要安装相关的解压程序，例如 RAR 文件需要安装 `unrar`。
更多信息：<https://manned.org/unp>.

- 解压一个归档文件：

`unp `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/归档文件.zip</span>

- 解压多个归档文件：

`unp `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/归档文件1.tar.gz</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/归档文件2.rar</span>
