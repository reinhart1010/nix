# yadm-list

打印由 `yadm` 管理的文件列表。
更多信息：<https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#commands>.

- 打印当前目录中由 `yadm` 管理的文件列表：

`yadm list`

- 列出所有由 `yadm` 完全管理的文件：

`yadm list -a`
