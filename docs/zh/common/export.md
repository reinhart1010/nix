<hr># export

命令为当前 shell 中的子进程进行环境变量设置。
更多信息：<https://manned.org/export.1posix>.

- 设置环境变量：

`export `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">变量名</span>`=`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">值</span>

- 向环境变量 `PATH` 追加一个路径名：

`export PATH=$PATH:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">追加的 path 路径</span>
