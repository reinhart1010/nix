# yadm-perms

更新权限。
通常不需要运行此命令，因为 `yadm` 默认会自动处理权限。这种自动行为可以通过将配置 `yadm.auto-perms` 设置为 `"false"` 来禁用。
更多信息：<https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#permissions>.

- 修改文件权限：

`yadm perms`
