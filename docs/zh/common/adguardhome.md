# AdGuardHome

一款全网广告拦截与反跟踪软件。
更多信息：<https://github.com/AdguardTeam/AdGuardHome>.

- 运行 AdGuard Home：

`AdGuardHome`

- 使用给定的配置文件运行 AdGuard Home：

`AdGuardHome --config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">给定的/配置文件.yaml</span>

- 设置存储数据的工作目录：

`AdGuardHome --work-dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">工作目录/路径</span>

- 安装或卸载 AdGuard Home 的服务：

`AdGuardHome --service `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">install|uninstall</span>

- 启动 AdGuard Home 的服务：

`AdGuardHome --service start`

- 刷新 AdGuard Home 服务的设置项：

`AdGuardHome --service reload`

- 停止或重启 AdGuard Home 的服务：

`AdGuardHome --service `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">stop|restart</span>
