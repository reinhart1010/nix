# unalias

移除别名。
更多信息：<https://manned.org/unalias>.

- 移除一个别名：

`unalias `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">别名</span>

- 移除所有别名：

`unalias -a`
