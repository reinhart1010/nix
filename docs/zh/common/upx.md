# upx

压缩或解压可执行文件。
更多信息：<https://upx.github.io>.

- 压缩可执行文件：

`upx `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 解压可执行文件：

`upx -d `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 详细帮助：

`upx --help`
