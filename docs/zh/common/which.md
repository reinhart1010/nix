# which

在用户的`PATH`中寻找可执行文件的路径。
更多信息：<https://manned.org/which>.

- 在`PATH`中寻找可执行文件并打印第一个匹配的结果：

`which `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">executable</span>

- 如果有多个匹配结果则打印所有结果：

`which -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">executable</span>
