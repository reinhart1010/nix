# hostname

显示或设置系统的主机名。
更多信息：<https://manned.org/hostname>.

- 显示本机的主机名：

`hostname`

- 显示主机名的网络地址：

`hostname -i`

- 显示主机的所有网络地址：

`hostname -I`

- 显示 FQDN（完全限定域名）：

`hostname --fqdn`

- 设置本机主机名：

`hostname `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">新主机名</span>
