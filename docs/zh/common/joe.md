# joe

Joe 自己的文本编辑器。
更多信息：<https://joe-editor.sourceforge.io>.

- 在 JOE 中打开一个新文件：

`joe`

- 打开一个指定文件：

`joe `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 打开一个指定文件，并将光标定位在指定的行：

`joe +`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">行号</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>

- 以只读模式打开一个指定文件：

`joe -rdonly `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件</span>
