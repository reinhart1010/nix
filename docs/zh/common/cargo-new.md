# cargo new

创建一个新的 Cargo 包。
相当于 `cargo init`，但是需要指定一个目录。
更多信息：<https://doc.rust-lang.org/cargo/commands/cargo-new.html>.

- 使用二进制目标创建一个新的 Rust 项目：
