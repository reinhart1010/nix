# jpegtopnm

将 JPEG/JFIF 文件转换为 PPM 或 PGM 格式。
更多信息：<https://netpbm.sourceforge.net/doc/jpegtopnm.html>.

- 将 JPEG/JFIF 图像转换为 PPM 或 PGM 图像：

`jpegtopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件.jpg</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">路径/到/文件.pnm</span>

- 显示版本信息：

`jpegtopnm -version`
