# yadm-alt

为任何被管理的文件创建符号链接并处理模板。
了解更多关于模板的信息：<https://yadm.io/docs/templates>。
更多信息：<https://yadm.io/docs/alternates>.

- 手动在备用文件之间创建符号链接：

`yadm alt`
