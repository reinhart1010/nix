# who

显示当前登录用户和相关信息（进程，启动时间）。
参见: `whoami`。
更多信息：<https://www.gnu.org/software/coreutils/who>。

- 显示用户名，终端线路，和所有当前登录会话的时间：

`who`

- 显示所有可用信息：

`who -a`

- 显示所有可用信息，包含表格首部名称：

`who -a -H`
