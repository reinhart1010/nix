# bpytop

一个资源监视器，显示有关 CPU、内存、磁盘、网络和进程的信息。
`bashtop` 的 Python 版本。
更多信息：<https://github.com/aristocratos/bpytop>.

- 运行 bpytop：

`bpytop`

- 以最小模式启动，不显示内存和网络框：

`bpytop -m`

- 切换最小模式：

`m`

- 搜索正在运行的程序或进程：

`f`

- 修改设置：

`M`

- 显示版本：

`bpytop -v`
