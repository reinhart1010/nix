# yard

Ruby 的文档工具。
更多信息：<https://yardoc.org/>.

- 创建文档：

`yard`

- 创建文档并保存到一个文件中：

`yard --one-file`

- 列出所有未记录的对象：

`yard stats --list-undoc`
