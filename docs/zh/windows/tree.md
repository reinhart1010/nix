# tree

以目录树的形式显示指定目录的所有文件夹和文件。
更多信息：<https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- 显示当前目录的目录树：

`tree`

- 显示指定目录的目录树：

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录的路径</span>

- 显示目录中文件的目录树：

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录的路径</span>` /f`

- 使用 ASCII 字符而不是扩展字符显示目录树：

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录的路径</span>` /a`
