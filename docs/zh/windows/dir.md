# dir

列出目录的内容。
更多信息：<https://learn.microsoft.com/windows-server/administration/windows-commands/dir>.

- 显示当前目录的内容：

`dir`

- 显示指定目录的内容：

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录的路径</span>

- 显示当前目录的内容，包括隐藏的文件：

`dir /A`

- 显示指定目录的内容，包括隐藏的文件：

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录的路径</span>` /A`
