# choco

Chocolatey 包管理器的命令行界面。
此命令也有关于其子命令的文件，例如：`choco install`.
更多信息：<https://chocolatey.org>.

- 执行 Chocolatey 命令：

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">命令</span>

- 显示基本的帮助页面：

`choco -?`

- 显示指定命令的帮助页面：

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">命令</span>` -?`

- 显示 Chocolatey 的版本：

`choco --version`
