# pushd

将目录储存在栈顶以便稍后访问。
参考`popd`命令来切换到原目录。
更多信息：<https://docs.microsoft.com/windows-server/administration/windows-commands/pushd>.

- 切换目录并将它放至栈：

`pushd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录</span>
