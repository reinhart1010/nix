# mkdir

创建一个目录。
更多信息：<https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- 创建一个目录：

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">目录名</span>

- 递归创建目录及子目录：

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">子目录名</span>
