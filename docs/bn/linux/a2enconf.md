# a2enconf

ডেবিয়ান ভিত্তিক ওএস এ একটি অ্যাপাচি কনফিগারেশন ফাইলকে সক্রিয় করুন।
আরও তথ্য পাবেন: <https://manned.org/a2enconf.8>।

- একটি কনফিগারেশন ফাইল সক্রিয় করুন:

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">কনফিগুরেসন_ফাইল</span>

- কোনো তথ্যবহুল মেসেজ দেখবেন না:

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">কনফিগুরেসন_ফাইল</span>
