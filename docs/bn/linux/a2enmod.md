# a2enmod

ডেবিয়ান ভিত্তিক ওএস এ একটি অ্যাপাচি মজুল সক্রিয় করুন।
আরও তথ্য পাবেন: <https://manned.org/a2enmod.8>।

- একটি মজুল সক্রিয় করুন:

`sudo a2enmod `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">মজিউল</span>

- কোনো তথ্যবহুল মেসেজ দেখবেন না:

`sudo a2enmod --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">মজিউল</span>
