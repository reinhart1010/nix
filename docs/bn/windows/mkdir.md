# mkdir

একটি নির্দেশিকা তৈরি করে।
আরও তথ্য পাবেন: <https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>।

- একটি নির্দেশিকা তৈরি করুন:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">নির্দেশিকা\এর\পথ</span>

- পুনরাবৃত্তি হিসেবে একটি নেস্টেড নির্দেশিকা ট্রি তৈরি করুন:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">উপনির্দেশিকা\এর\পথ</span>
