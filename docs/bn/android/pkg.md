# pkg

Termux এর জন্য একটি প্যাকেজ ব্যবস্থাপনা ইউটিলিটি।
আরও তথ্য পাবেন: <https://wiki.termux.com/wiki/Package_Management>.

- সমস্ত ইনস্টল করা প্যাকেজ আপগ্রেড করুন:

`pkg upgrade`

- একটি নির্দিষ্ট প্যাকেজ ইনস্টল করুন:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">প্যাকেজ</span>

- একটি নির্দিষ্ট প্যাকেজ আনইনস্টল করুন:

`pkg uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">প্যাকেজ</span>

- একটি নির্দিষ্ট প্যাকেজ পুনরায় ইনস্টল করুন:

`pkg reinstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">প্যাকেজ</span>

- একটি নির্দিষ্ট প্যাকেজ অনুসন্ধান করুন:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">প্যাকেজ</span>
