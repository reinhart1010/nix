# yard

ابزار مستند سازی برای زبان برنامه نویسی روبی.
اطلاعات بیشتر: <https://yardoc.org/>.

- ساخت مستندات:

`yard`

- ساخت مستندات و ذخیره آن در یک فایل:

`yard --one-file`

- فهرست تمام اشیاء مستند نشده:

`yard stats --list-undoc`
