<hr># zm

ابزاری برای مدیریت مقاله های مجلات و وبلاگ ها.
اطلاعات بیشتر: <https://github.com/ZERMZeitung/zm2>.

- ساخت یک پیش نویس جدید:

`zm new`

- ویرایش یک پیش نویس:

`zm edit`

- انتشار یک پیش نویس و ثبت در گیت:

`zm publish`
