# choco feature

चॉकलेट के साथ विशेषताओं के साथ इंटरैक्ट करें।
अधिक जानकारी: <https://chocolatey.org/docs/commands-feature>।

- उपलब्ध विशेषताओं की सूची दिखाएँ:

`choco feature list`

- एक विशेषता सक्षम करें:

`choco feature enable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">नाम</span>

- एक विशेषता अक्षम करें:

`choco feature disable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">नाम</span>
