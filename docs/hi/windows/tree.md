<hr># tree

पथ के लिए निर्देशिका संरचना का ग्राफ़िकल ट्री प्रदर्शित करें।
अधिक जानकारी: <https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- वर्तमान निर्देशिका के लिए ट्री प्रदर्शित करें:

`tree`

- किसी विशिष्ट निर्देशिका के लिए वृक्ष प्रदर्शित करें:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">निर्देशिका\का\पथ</span>

- फ़ाइलों सहित निर्देशिका के लिए वृक्ष प्रदर्शित करें:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">निर्देशिका\का\पथ</span>` /f`

- विस्तारित वर्णों के बजाय ASCII वर्णों का उपयोग करके वृक्ष प्रदर्शित करें:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">निर्देशिका\का\पथ</span>` /a`
