# reset

वर्तमान टर्मिनल को `reset` करें। संपूर्ण टर्मिनल स्क्रीन को साफ़ करता है।
अधिक जानकारी: <https://manned.org/reset>।

- वर्तमान टर्मिनल को `reset` करें:

`reset`

- टर्मिनल प्रकार दिखाएं:

`reset -q`
