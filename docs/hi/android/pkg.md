# pkg

टर्मक्स के लिए पैकेज प्रबंधन उपयोगिता।
अधिक जानकारी: <https://wiki.termux.com/wiki/Package_Management>।

- सभी स्थापित पैकेजों को अपग्रेड करें:

`pkg upgrade`

- एक पैकेज स्थापित करें:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">पैकेज</span>

- एक पैकेज अनइंस्टॉल करें:

`pkg uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">पैकेज</span>

- एक पैकेज पुनः स्थापित करें:

`pkg reinstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">पैकेज</span>

- एक पैकेज खोजें:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">पैकेज</span>
