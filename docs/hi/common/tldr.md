# tldr

tldr-pages प्रोजेक्ट से कमांड-लाइन टूल्स के लिए सरल हेल्प पेज प्रदर्शित करता है।
अधिक जानकारी: <https://github.com/tldr-pages/tldr/blob/main/CLIENT-SPECIFICATION.md#command-line-interface>।

- एक कमांड के विशिष्ट उपयोग प्राप्त करें (संकेत: इसका उपयोग कर आप यहाँ आए!):

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">कमांड</span>

- Linux के लिए tar tldr पेज दिखाएं:

`tldr -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">linux</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">tar</span>

- एक Git उपकमांड के लिए सहायता प्राप्त करें:

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">git-checkout</span>

- स्थानीय पृष्ठों को अपडेट करें (यदि ग्राहक कैशिंग का समर्थन करता है):

`tldr -u`
