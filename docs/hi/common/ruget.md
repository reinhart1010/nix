<hr># ruget

Rust में लिखे गए wget का विकल्प।
अधिक जानकारी: <https://github.com/ksk001100/ruget>।

- किसी फ़ाइल में URL की सामग्री डाउनलोड करें:

`ruget `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com/file</span>

- किसी निर्दिष्ट आउटपुट फ़ाइल में URL की सामग्री डाउनलोड करें:

`ruget --output `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">file_name</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com/file</span>
