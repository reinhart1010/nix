# micro

Um editor de texto moderno e intuitivo baseado em terminal.
Você pode usar seu teclado, mas também seu mouse para navegar e/ou selecionar texto.
Mais informações: <https://micro-editor.github.io>.

- Abre um arquivo:

`micro `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/arquivo</span>

- Salva um arquivo:

`<Ctrl> + S`

- Corta a linha inteira:

`<Ctrl> + K`

- Procura por um padrão no arquivo (pressione `Ctrl + N`/`Ctrl + P` para ir para a próxima/anterior correspondência):

`<Ctrl> + F "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">padrão</span>`" <Enter>`

- Executa um comando:

`<Ctrl> + E `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>` <Enter>`

- Realiza uma substituição em todo o arquivo:

`<Ctrl> + E replaceall "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">padrão</span>`" "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">substituição</span>`" <Enter>`

- Sai:

`<Ctrl> + Q`
