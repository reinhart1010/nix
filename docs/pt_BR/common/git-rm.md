<hr># git rm

Remove arquivos do índice do repositório e do arquivo local do sistema.
Mais informações: <https://git-scm.com/docs/git-rm>.

- Remove arquivo do índice do repositório e do arquivo local do sistema:

`git rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">arquivo</span>

- Remove um diretório:

`git rm -r `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">diretório</span>

- Remove arquivo do índice do repositório mas mantém o mesmo localmente:

`git rm --cached `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">arquivo</span>
