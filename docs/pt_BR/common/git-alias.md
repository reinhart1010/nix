<hr># git alias

Cria comandos curtos para comandos Git.
Parte do `git-extras`.
Mais informações: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>.

- Lista todos os comandos curtos:

`git alias`

- Cria um novo comando curto com o `nome`:

`git alias "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome</span>`" "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>`"`

- Procura por um comando curto existente:

`git alias ^`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome</span>
