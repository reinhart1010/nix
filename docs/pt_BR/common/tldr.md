# tldr

Exibe páginas de ajuda simples para ferramentas de linha de comando do projeto tldr-pages.
Mais informações: <https://tldr.sh>.

- Imprime a página do tldr para um comando específico (dica: é assim que você chegou aqui!):

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>

- Imprime a página do tldr para um subcomando específico:

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>`-`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcomando</span>

- Imprime a página do tldr para um comando para uma [p]lataforma específica:

`tldr -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">android|linux|osx|sunos|windows</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>

- Atualiza o cache local das páginas do tldr:

`tldr -u`
