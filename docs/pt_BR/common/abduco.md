# abduco

Gerenciador de sessão no terminal.
Mais informações: <https://www.brain-dump.org/projects/abduco/>.

- Lista sessões:

`abduco`

- Anexa à uma sessão, criando-a se não existir:

`abduco -A `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">bash</span>

- Anexa à uma sessão com `dvtm`, criando-a se não existir:

`abduco -A `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome</span>

- Desanexa de uma sessão:

`<Ctrl> + \`

- Anexa à uma sessão no modo read-only (somente leitura):

`abduco -Ar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome</span>
