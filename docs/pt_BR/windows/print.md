# print

Imprimir um arquivo de texto em uma impressora.
Mais informações: <https://learn.microsoft.com/windows-server/administration/windows-commands/print>.

- Imprime um arquivo de texto na impressora padrão:

`print `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/arquivo</span>

- Imprime arquivo de texto em uma impressora específica:

`print /d:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">impressora</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/arquivo</span>
