# tree

Exibe uma árvore gráfica da estrutura do diretório no caminho.
Mais informações: <https://learn.microsoft.com/windows-server/administration/windows-commands/tree>.

- Exibe a árvore para o diretório atual:

`tree`

- Exibe a árvore para o diretório específico:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/diretório</span>

- Exibe a árvore para o diretório específico incluíndo arquivos:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/diretório</span>` /f`

- Exibe a árvore usando caractéres ASCII:

`tree `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/diretório</span>` /a`
