<hr># dconf read

Lê valores de chave dos bancos de dados dconf.
Veja também: `dconf`.
Mais informações: <https://manned.org/dconf>.

- Imprime um valor de chave específico:

`dconf read `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/caminho/para/chave</span>

- Imprime o valor padrão de uma chave específica:

`dconf read -d `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/caminho/para/chave</span>
