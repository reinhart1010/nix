# ascii

Mostra pseudónimos de caractéres ASCII.
Mais informações: <http://www.catb.org/~esr/ascii/>.

- Mostra pseudónimos ASCII de um carácter:

`ascii `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">a</span>

- Mostra pseudónimos ASCII de forma resumida, modo script-friendly:

`ascii -t `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">a</span>

- Mostra pseudónimo ASCII de múltiplos caracteres:

`ascii -s `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">tldr</span>

- Mostra tabela ASCII em decimal:

`ascii -d`

- Mostra tabela ASCII em hexadecimal:

`ascii -x`

- Mostra tabela ASCII em octal:

`ascii -o`

- Mostra tabela ASCII em binário:

`ascii -b`

- Mostra sumário de opções e tabela ASCII completa:

`ascii`
