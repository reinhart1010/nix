# zathura

Um visualizador de documentos modular e baseado em vim, com uma linha de comando integrada.
Tenha certeza de que um backend está instalado (poppler, PostScript, ou DjVu).
Mais informações: <https://pwmt.org/projects/zathura/>.

- Abre um arquivo:

`zathura `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">caminho/para/arquivo</span>

- Navega esquerda/baixo/cima/direita:

`<H|J|K|L> ou teclas direcionais`

- Rotaciona:

`r`

- Inverte cores:

`Ctrl + R`

- Procura por uma string no documento:

`/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">string</span>

- Cria/remove marcadores de página:

`:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">bmark|bdelete</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nome_do_marcador</span>

- Lista marcadores de página:

`:blist`
