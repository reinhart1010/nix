# a2enmod

Nyalakan suatu modul piranti peladen Apache dalam sistem operasi berbasis Debian.
Informasi lebih lanjut: <https://manned.org/a2enmod.8>.

- Nyalakan suatu modul:

`sudo a2enmod `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">modul</span>

- Jangan menampilkan pesan-pesan informatif selama melakukan operasi:

`sudo a2enmod --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">modul</span>
