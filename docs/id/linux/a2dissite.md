# a2dissite

Matikan fungsi peladenan suatu host maya (virtual host) pada piranti peladen Apache dalam sistem operasi berbasis Debian.
Informasi lebih lanjut: <https://manned.org/a2dissite.8>.

- Matikan suatu host maya:

`sudo a2dissite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">host_maya</span>

- Jangan menampilkan pesan-pesan informatif selama melakukan operasi:

`sudo a2dissite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">host_maya</span>
