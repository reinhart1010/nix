# add-apt-repository

Kelola definisi repositori `apt`.
Informasi lebih lanjut: <https://manned.org/apt-add-repository>.

- Tambah repositori `apt` baru:

`add-apt-repository `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositori</span>

- Hilangkan sebuah repositori `apt`:

`add-apt-repository --remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositori</span>

- Perbarui cache paket setelah menambahkan sebuah repositori:

`add-apt-repository --update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositori</span>

- Izinkan sumber paket untuk diunduh dari repositori:

`add-apt-repository --enable-source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">repositori</span>
