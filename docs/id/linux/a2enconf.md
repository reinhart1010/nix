# a2enconf

Nyalakan suatu konfigurasi piranti peladen Apache yang diatur oleh suatu berkas dalam sistem operasi berbasis Debian.
Informasi lebih lanjut: <https://manned.org/a2enconf.8>.

- Nyalakan konfigurasi yang diatur dalam suatu berkas:

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">berkas_konfigurasi</span>

- Jangan menampilkan pesan-pesan informatif selama melakukan operasi:

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">berkas_konfigurasi</span>
