# choco info

Tampilkan informasi suatu paket piranti lunak secara terperinci dengan Chocolatey.
Informasi lebih lanjut: <https://chocolatey.org/docs/commands-info>.

- Tampilkan informasi mengenai suatu paket:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paket</span>

- Tampilkan informasi mengenai suatu paket dengan versi yang terpasang secara lokal saja:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paket</span>` --local-only`

- Tentukan suatu repositori paket secara kustom sebagai sumber informasi kumpulan paket:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paket</span>` --source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">url_sumber|alias</span>

- Gunakan suatu username dan kata sandi untuk mendukung proses autentikasi pengguna:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paket</span>` --user `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">username</span>` --password `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">kata_sandi</span>
