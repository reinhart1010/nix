<hr># gitstats

Buat laporan statistika atas repositori Git.
Informasi lebih lanjut: <https://gitstats.sourceforge.net>.

- Buat laporan statistik terhadap suatu repositori lokal:

`gitstats `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">jalan/menuju/repo_git/.git</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">jalan/menuju/direktori_luaran</span>

- Lihat hasil laporan dalam suatu peramban web dalam Windows (PowerShell)/macOS/Linux:
