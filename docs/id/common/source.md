# source

Mengeksekusi perintah dalam file pada shell saat ini.
Informasi lebih lanjut: <https://manned.org/source>.

- Mengevaluasi konten pada file yang ditentukan:

`source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">jalan/menuju/file</span>

- Mengevaluasi konten pada file yang ditentukan (`.` adalah alias dari `source`):

`. `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">jalan/menuju/file</span>
