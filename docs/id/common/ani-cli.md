<hr># ani-cli

Program baris perintah (CLI) untuk menelusuri dan menonton film anime.
Informasi lebih lanjut: <https://github.com/pystardust/ani-cli>.

- Cari anime dengan nama:

`ani-cli "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nama_anime</span>`"`

- Unduh ([d]ownload) suatu episode:

`ani-cli -d "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nama_anime</span>`"`

- Gunakan [v]LC untuk memutar film:

`ani-cli -v "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nama_anime</span>`"`

- Tonton satu [e]pisode:

`ani-cli -e `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nomor_episode</span>` "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nama_anime</span>`"`

- Lanjut ([c]ontinue) menonton anime dari riwayat:

`ani-cli -c`

- Mutakhirkan ([U]pdate) program `ani-cli`:

`ani-cli -U`
