# python

Penerjemah bahasa Python.
Informasi lebih lanjut: <https://www.python.org>.

- Jalankan REPL (shell interaktif):

`python`

- Jalankan skrip pada berkas Python:

`python `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">skrip.py</span>

- Jalankan skrip sebagai bagian dari shell interaktif:

`python -i `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">skrip.py</span>

- Jalankan ekspresi Python:

`python -c "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ekspresi</span>`"`

- Jalankan suatu modul perpustakaan sebagai skrip (diakhiri dengan daftar opsi):

`python -m `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">modul</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">argumen</span>

- Pasang suatu paket pustaka menggunakan `pip`:

`python -m pip install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paket</span>

- Jalankan pengawakutu (debugger) terhadap skrip Python secara interaktif:

`python -m pdb `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">jalan/menuju/berkas.py</span>

- Nyalakan program peladen (server) HTTP bawaan terhadap direktori ini menuju port 8000:

`python -m http.server`
