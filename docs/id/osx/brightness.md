# brightness

Menampilkan dan mengubah brightness (tingkat kecerahan) untuk internal dan external displays.
Informasi lebih lanjut: <https://github.com/nriley/brightness>.

- Menampilkan brightness sekarang:

`brightness -l`

- Mengubah brightness menjadi 100%:

`brightness `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1</span>

- Mengubah brightness menjadi 50%:

`brightness `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">0.5</span>
