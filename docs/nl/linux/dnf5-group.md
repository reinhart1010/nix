# dnf5 group

Dit commando is een alias van `dnf group`.
Let op: van Fedora 37 tot 40 (inclusief), runt `dnf` DNF V4 terwijl `dnf5` DNF V5 uitvoert.

- Bekijk de documentatie van het originele commando:

`tldr dnf group`
