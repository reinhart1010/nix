<hr># mkdir

Maak een map aan.
Meer informatie: <https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- Maak een map aan:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad\naar\map</span>

- Maak een geneste mappenstructuur recursief aan:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad\naar\sub_map</span>
