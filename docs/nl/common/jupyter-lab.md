# jupyter lab

Interactieve ontwikkelomgeving voor Jupyter notebooks.
Meer informatie: <https://jupyterlab.readthedocs.io/en/stable/>.

- Start JupyterLab:

`jupyter lab`

- Open een specifiek notebook:

`jupyter lab `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/notebook.ipynb</span>

- Start JupyterLab in een specifieke map:

`jupyter lab --notebook-dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/map</span>

- Start JupyterLab in debug modus:

`jupyter lab --debug`
