# unlink

Verwijder een link naar een bestand van het bestandssysteem.
De inhoud van het bestand gaat verloren als de link de laatste is naar het bestand.
Meer informatie: <https://www.gnu.org/software/coreutils/manual/html_node/unlink-invocation.html>.

- Verwijder het opgegeven bestand als het de laatste link is:

`unlink `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand</span>
