# pbmtosunicon

Converteer een PBM afbeelding naar een Sun icon.
Meer informatie: <https://netpbm.sourceforge.net/doc/pbmtosunicon.html>.

- Converteer een PBM afbeelding naar een Sun icon:

`pbmtosunicon `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/invoer.pbm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/uitvoer.ico</span>
