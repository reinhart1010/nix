# unalias

Verwijder aliassen.
Meer informatie: <https://manned.org/unalias>.

- Verwijder een alias:

`unalias `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">alias_naam</span>

- Verwijder alle aliassen:

`unalias -a`
