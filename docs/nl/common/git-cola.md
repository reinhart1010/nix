# git cola

Een krachtige Git GUI met een gelikte en intuïtieve gebruikersinterface.
Meer informatie: <https://git-cola.readthedocs.io>.

- Start git cola:

`git cola`

- Start git cola in amend mode:

`git cola --amend`

- Vraag voor een Git repository. (Standaard de huidige map):

`git cola --prompt`

- Open de Git repository op de genoemde plek:

`git cola --repo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/git-repository</span>

- Voer het pad filter uit voor de status widget:

`git cola --status-filter `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filter</span>
