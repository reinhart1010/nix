# python

Python taal interpreter.
Meer informatie: <https://www.python.org>.

- Start een REPL (interactieve shell):

`python`

- Voer een specifiek Python bestand uit:

`python `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand.py</span>

- Voer een specfiek Python bestand uit en start een REPL:

`python -i `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand.py</span>

- Voer een Python expressie uit:

`python -c "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">expressie</span>`"`

- Voer het script uit van een gespecificeerd bibliotheek module:

`python -m `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">module</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">argumenten</span>

- Installeer een pakket met `pip`:

`python -m pip install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakket</span>

- Debug interactief een Python script:

`python -m pdb `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand.py</span>

- Start de ingebouwde HTTP server op poort 8000 in de huidige map:

`python -m http.server`
