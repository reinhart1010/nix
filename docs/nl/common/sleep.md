# sleep

Wacht voor een gespecificeerde hoeveelheid tijd.
Meer informatie: <https://pubs.opengroup.org/onlinepubs/9699919799/utilities/sleep.html>.

- Wacht in seconden:

`sleep `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">seconden</span>

- Voer een specifiek commando uit na een wachttijd van 20 seconden:

`sleep 20 && `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">commando</span>
