# basename

Verwijder voorlopende map delen van een pad.
Meer informatie: <https://www.gnu.org/software/coreutils/manual/html_node/basename-invocation.html>.

- Toon alleen de bestandsnaam van een pad:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand</span>

- Toon alleen de meest rechtse map naam van een pad:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/map/</span>

- Toon alleen de bestandsnaam van een pad met een suffix verwijderd:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pad/naar/bestand</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">suffix</span>
