# pkg

Menedżer pakietów FreeBSD.
Więcej informacji: <https://man.freebsd.org/cgi/man.cgi?pkg>.

- Zainstaluj nowy pakiet:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Usuń pakiet:

`pkg delete `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Zaktualizuj wszystkie pakiety:

`pkg upgrade`

- Wyszukaj pakiet:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">słowo_kluczowe</span>

- Wyświetl listę zainstalowanych pakietów:

`pkg info`

- Usuń niepotrzebne zależności:

`pkg autoremove`
