<hr># pkg

Narzędzie do zarządzania pakietami dla Termuxa.
Więcej informacji: <https://wiki.termux.com/wiki/Package_Management>.

- Zaktualizuj wszystkie zainstalowane pakiety:

`pkg upgrade`

- Zainstaluj pakiet:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Odinstaluj pakiet:

`pkg uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Zainstaluj ponownie pakiet:

`pkg reinstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Wyszukaj pakiet:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>
