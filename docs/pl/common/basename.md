# basename

Usuwa wiodące nazwy katalogów ze ścieżki.
Więcej informacji: <https://www.gnu.org/software/coreutils/manual/html_node/basename-invocation.html>.

- Wyświetl wyłącznie nazwę pliku ze ścieżki:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/pliku</span>

- Wyświetl wyłącznie nazwę skrajnie prawego katalogu ze ścieżki:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/katalogu/</span>

- Wyświetl wyłącznie nazwę pliku ze ścieżki, z usuniętym przyrostkiem:

`basename `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/pliku</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">przyrostek</span>
