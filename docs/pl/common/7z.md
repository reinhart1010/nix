# 7z

Archiwizator plików o wysokim stopniu kompresji.
Więcej informacji: <https://manned.org/7z>.

- Zarchiwizuj plik lub katalog:

`7z a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sciezka/do/pliku_lub_katalogu</span>

- Zaszyfruj istniejące archiwum (w tym nagłówki)):

`7z a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zaszyfrowane.7z</span>` -p`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">haslo</span>` -mhe=on `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>

- Wyodrębnij istniejący plik 7z z oryginalną strukturą katalogów:

`7z x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>

- Wyodrębnij archiwum ze ścieżką wyjściową zdefiniowaną przez użytkownika:

`7z x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>` -o`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sciezka/do/wyjscia</span>

- Wypakuj archiwum do stdout:

`7z x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>` -so`

- Archiwizuj przy użyciu określonego typu archiwum:

`7z a -t`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">7z|bzip2|gzip|lzip|tar|zip</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">zarchiwizowane.7z</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sciezka/do/pliku_lub_katalogu</span>

`7z i`
