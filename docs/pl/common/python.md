<hr># python

Interpreter języka Python.
Więcej informacji: <https://www.python.org>.

- Wywołaj interaktywną powłokę Pythona (REPL):

`python`

- Wykonaj skrypt w danym pliku Python:

`python `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">skrypt.py</span>

- Wykonaj skrypt jako część interaktywnej powłoki:

`python -i `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">skrypt.py</span>

- Wykonaj wyrażenie w języku Python:

`python -c "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">wyrazenie</span>`"`

- Uruchom moduł biblioteki jako skrypt (kończy listę opcji):

`python -m `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">moduł</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">argumenty</span>

- Interaktywnie debuguj skrypt w języku Python:

`python -m pdb `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">skrypt.py</span>
