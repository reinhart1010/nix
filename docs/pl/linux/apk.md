# apk

Narzędzie zarządzania pakietami Alpine Linux.
Więcej informacji: <https://wiki.alpinelinux.org/wiki/Alpine_Linux_package_management>.

- Zaktualizuj indeksy repozytoriów ze wszystkich zdalnych repozytoriów:

`apk update`

- Instaluj nowy pakiet:

`apk add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Usuń pakiet:

`apk del `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Napraw pakiet lub zaktualizuj go bez modyfikacji głownych zależności:

`apk fix `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>

- Szukaj pakietu za pomocą słów kluczowych:

`apk search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">słowo kluczowe</span>

- Wyświetl informacje o określonym pakiecie:

`apk info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pakiet</span>
