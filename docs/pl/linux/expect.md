# expect

Wykonuje skrypty, które dokonują interakcji z programami przyjmującymi dane od użytkownika.
Więcej informacji: <https://manned.org/expect>.

- Wykonaj skrypt expect z pliku:

`expect `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/pliku</span>

- Wykonaj podany skrypt expect:

`expect -c "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">polecenia</span>`"`

- Wejdź do interaktywnego REPL (użyj `exit` lub Ctrl + D, aby wyjść):

`expect -i`
