<hr># choco feature

Zarządzanie dostępnymi funkcjani Chocolatey.
Więcej informacji: <https://chocolatey.org/docs/commands-feature>.

- Wyświetlanie listy dostępnych funkcji:

`choco feature list`

- Włączenie podanej funkcji:

`choco feature enable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nazwa_funkcji</span>

- Wyłączenie podanej funkcji:

`choco feature disable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nazwa_funkcji</span>
