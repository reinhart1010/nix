# print

Wyślij plik tekstowy do drukarki.
Więcej informacji: <https://learn.microsoft.com/windows-server/administration/windows-commands/print>.

- Drukuj plik tekstowy używając domyślnej drukarki:

`print `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/pliku</span>

- Drukuj plik tekstowy używakąc wybranej drukarki:

`print /d:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">drukarka</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/pliku</span>
