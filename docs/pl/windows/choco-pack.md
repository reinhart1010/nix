# choco pack

Pakowanie specyfikacji NuGet do pliku nupkg.
Więcej informacji: <https://chocolatey.org/docs/commands-pack>.

- Spakuj specyfikację NuGet do pliku nupkg:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieża/do/specyfikacji</span>

- Spakuj specyfikację NuGet podajęc wersję pliku wyjściowego:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieża/do/specyfikacji</span>` --version `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">wersja</span>

- Spakuj specyfikację NuGet do konkretnego katalogu wyjściowego:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieża/do/specyfikacji</span>` --output-directory `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ścieżka/do/katalogu/wyjściowego</span>
