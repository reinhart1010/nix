# apk

Gestore pacchetti di Alpine Linux.
Maggiori informazioni: <https://wiki.alpinelinux.org/wiki/Alpine_Linux_package_management>.

- Aggiorna gli indici di tutti i repository remoti:

`apk update`

- Installa un nuovo pacchetto:

`apk add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>

- Disinstalla un pacchetto:

`apk del `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>

- Prova a riparare un pacchetto o ad aggiornare un pacchetto senza modificarne le dipendenze:

`apk fix `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paquet</span>

- Ricerca pacchetti data una parola chiave:

`apk search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">parola_chiave</span>

- Ottieni informazioni su un pacchetto:

`apk info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>
