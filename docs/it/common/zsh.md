<hr># zsh

Z SHell.
Inteprete da linea di comando compatibile con `bash` e `sh`.
Maggiori informazioni: <https://www.zsh.org>.

- Fai partire l'interprete interattivo da linea di comando:

`zsh`

- Esegui un comando passandolo come parametro:

`zsh -c `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>

- Esegui comandi da un file (script):

`zsh `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">file</span>

- Esegui comandi da un file e stampali a schermo mentre vengono eseguiti:

`zsh -x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">file</span>
