# more

Aprire un file per la lettura interattiva, con scorrimento e ricerca.
Maggiori informazioni: <https://manned.org/more>.

- Apri un file:

`more `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">percorso/del/file</span>

- Apri un file visualizzato da una riga specifica:

`more +`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">linea_numero</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">percorso/del/file</span>

- Mostra aiuto:

`more --help`

- Prossima pagina:

`<Spazio>`

- Cerca una stringa (tocca `n` per passare alla corrispondenza successiva):

`/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">qualcose</span>

- Smettere:

`q`

- Mostra aiuto per comando interattivo:

`h`
