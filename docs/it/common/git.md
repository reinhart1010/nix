<hr># git

Sistema di controllo versione distribuito.
Alcuni comandi aggiuntivi, come `commit`, hanno la propria documentazione.
Maggiori informazioni: <https://git-scm.com/>.

- Controlla la versione di Git:

`git --version`

- Mostra informazioni di aiuto generali:

`git --help`

- Mostra aiuto per uno specifico comando:

`git help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>

- Esegui un comando Git:

`git `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>
