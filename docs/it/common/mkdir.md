<hr># mkdir

Crea directory.
Maggiori informazioni: <https://www.gnu.org/software/coreutils/mkdir>.

- Crea una directory nella directory corrente o in un dato percorso:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">directory</span>

- Crea directory ricorsivamente (utile per creare directory annidate):

`mkdir -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">percorso/alla/directory</span>
