# choco

Interfaccia a linea di comando per il gestore pacchetti Chocolatey.
Alcuni sottocomandi come `choco install` hanno la loro documentazione.
Maggiori informazioni: <https://chocolatey.org>.

- Esegui comandi Chocolatey:

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">comando</span>

- Guarda il messaggio d'aiuto:

`choco -?`

- Guarda il messaggio d'aiuto di un comando specifico:

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>` -?`

- Controlla la versione di Chocolatey:

`choco --version`
