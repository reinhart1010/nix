# pkg

Strumento di gestione pacchetti per Termux.
Maggiori informazioni: <https://wiki.termux.com/wiki/Package_Management>.

- Aggiorna tutti i pacchetti installati:

`pkg upgrade`

- Installa un pacchetto:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>

- Disinstalla un pacchetto:

`pkg uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>

- Reinstalla un pacchetto:

`pkg reinstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>

- Cerca un pacchetto:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pacchetto</span>
