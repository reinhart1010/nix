<hr># ispell

Interactive spell checking.
More information: <https://www.cs.hmc.edu/~geoff/ispell.html>.

- Start an interactive session:

`ispell`

- Check for typos in the specified file and interactively apply suggestions:

`ispell `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Display the version:

`ispell -v`
