# zic

Compile timezones into binary files.
More information: <https://manned.org/zic>.

- Compile a timezone file from a directory:

`zic -d `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Report warnings during compilation of a specific file:

`zic -v `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>`.infile`
