# pkgadd

Add a package to a CRUX system.
More information: <https://docs.oracle.com/cd/E88353_01/html/E72487/pkgadd-8.html>.

- Install a local software package:

`pkgadd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Update an already installed package from a local package:

`pkgadd -u `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>
