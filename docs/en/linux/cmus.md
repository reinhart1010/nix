# cmus

Command-line Music Player.
Use arrow keys to navigate, `<enter/return>` to select, and numbers 1-8 switch between different views.
See also: `ncmpcpp`, `clementine`, `qmmp`.
More information: <https://cmus.github.io>.

- Open cmus into the specified directory (this will become your new working directory):

`cmus `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Add file/directory to library:

`:add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>

- Pause/unpause current song:

`c`

- Toggle shuffle mode on/off:

`s`

- Quit cmus:

`q`
