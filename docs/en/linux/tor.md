<hr># tor

Enable anonymous communication through the Tor network.
More information: <https://manned.org/man/tor>.

- Connect to the Tor network:

`tor`

- Start a Tor relay:

`tor --relay`

- View Tor configuration:

`tor --config`

- Check Tor status:

`tor --status`

- Run as client only:

`tor --client`

- Run as relay:

`tor --relay`

- Run as bridge:

`tor --bridge`

- Run as a hidden service:

`tor --hidden-service`
