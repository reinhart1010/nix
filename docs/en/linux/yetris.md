# yetris

Clone of the game Tetris in the terminal.
More information: <https://github.com/alexdantas/yetris>.

- Start a Tetris game:

`yetris`

- Navigate the piece horizontally:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">Left|Right arrow key</span>

- Rotate the piece clockwise or counterclockwise:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">x|z</span>

- Hold a piece (only one allowed at a time):

`c`

- Soft drop the piece:

`<Down arrow key>`

- Hard drop the piece:

`<Spacebar>`

- Pause/unpause the game:

`p`

- Quit the game:

`q`
