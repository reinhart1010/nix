# plasma-apply-wallpaperimage

Switch Plasma wallpaper.
More information: <https://invent.kde.org/plasma/plasma-workspace>.

- Apply a wallpaper:

`plasma-apply-wallpaperimage `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/path/to/image</span>

- Apply a wallpaper with a fill mode:

`plasma-apply-wallpaperimage --fill-mode `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">stretch|preserveAspectFit|preserveAspectCrop|tile|tileVertically|tileHorizontally|pad</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/path/to/image</span>

- Display help:

`plasma-apply-wallpaperimage --help`
