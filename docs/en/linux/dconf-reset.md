# dconf reset

Reset key values in dconf databases.
See also: `dconf`.
More information: <https://manned.org/dconf>.

- Reset a specific key value:

`dconf reset `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/path/to/key</span>

- Reset a specific directory:

`dconf reset -f `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/path/to/directory/</span>
