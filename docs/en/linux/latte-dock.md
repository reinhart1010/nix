# latte-dock

Replacement dock for Plasma desktop.
More information: <https://github.com/KDE/latte-dock>.

- Clear QML cache:

`latte-dock --clear-cache`

- Import and load default layout on startup:

`latte-dock --default-layout`

- Load a specific layout on startup:

`latte-dock --layout `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">layout_name</span>

- Import and load a specific layout:

`latte-dock --import-layout `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>
