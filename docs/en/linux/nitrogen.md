# nitrogen

Desktop background browser and setter for X Window.
More information: <https://github.com/l3ib/nitrogen>.

- View and set the wallpapers from a specific directory:

`nitrogen `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Set the wallpaper with automatic size settings:

`nitrogen --set-auto `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Restore the previous wallpaper:

`nitrogen --restore`
