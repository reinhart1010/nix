# qm status

Show virtual machine status.
More information: <https://pve.proxmox.com/pve-docs/qm.1.html>.

- Display the status of a specific virtual machine:

`qm status `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">vm_id</span>

- Display detailed status of a specific virtual machine:

`qm status --verbose `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">true</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">vm_id</span>
