<hr># links

Command-line web browser.
More information: <http://links.twibright.com/>.

- Visit a website:

`links `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com</span>

- Apply restrictions for anonymous account:

`links -anonymous `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com</span>

- Enable Cookies (`1` to enable):

`links -enable-cookies `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">0|1</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://example.com</span>

- Navigate forwards and backwards through the links on a page:

`Up arrow key, Down arrow key`

- Go forwards and backwards one page:

`Left arrow key, Right arrow key`

- Exit:

`q then y`
