# getfacl

Get file access control lists (ACL).
More information: <https://manned.org/getfacl>.

- Display the file access control list:

`getfacl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>

- Display the file access control list with [n]umeric user and group IDs:

`getfacl --numeric `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>

- Display the file access control list with [t]abular output format:

`getfacl --tabular `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>
