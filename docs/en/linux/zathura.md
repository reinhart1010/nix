# zathura

A vim-like modal document viewer, with an integrated command-line.
Make sure a backend is installed (poppler, PostScript, or DjVu).
More information: <https://pwmt.org/projects/zathura/>.

- Open a file:

`zathura `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Navigate left/up/down/right:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">H|J|K|L</span>` OR `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">arrow keys</span>

- Rotate:

`r`

- Invert Colors:

`Ctrl + R`

- Search for text by a given string:

`/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">string</span>

- Create/delete bookmarks:

`:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">bmark|bdelete</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">bookmark_name</span>

- List bookmarks:

`:blist`
