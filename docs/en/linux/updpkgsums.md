# updpkgsums

Update the checksums of the sources in a `PKGBUILD`.
Unless a preexisting hashing algorithm is used, SHA256 will be used.
More information: <https://manned.org/updpkgsums>.

- Update the checksums in a `PKGBUILD`:

`updpkgsums`

- Display Version:

`updpkgsums -v`

- Display help:

`updpkgsums -h`
