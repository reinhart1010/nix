# mkdir

Create a directory.
More information: <https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- Create a directory:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path\to\directory</span>

- Create a nested directory tree recursively:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path\to\sub_directory</span>
