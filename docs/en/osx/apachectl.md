# apachectl

Apache HTTP Server control interface for macOS.
More information: <https://www.unix.com/man-page/osx/8/apachectl/>.

- Start the `org.apache.httpd` launchd job:

`apachectl start`

- Stop the launchd job:

`apachectl stop`

- Stop, then start launchd job:

`apachectl restart`
