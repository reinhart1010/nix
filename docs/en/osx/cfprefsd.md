# cfprefsd

Provides preferences services (`CFPreferences`, `NSUserDefaults`).
It should not be invoked manually.
More information: <https://keith.github.io/xcode-man-pages/cfprefsd.8.html>.

- Start the daemon:

`cfprefsd`
