# xed

Opens files for editing in Xcode.
More information: <https://www.manpagez.com/man/1/xed/>.

- Open file in Xcode:

`xed `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">file1</span>

- Open file(s) in Xcode, create if it doesn't exist:

`xed --create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename1</span>

- Open a file in Xcode and jump to line number 75:

`xed --line 75 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename</span>
