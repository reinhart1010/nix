# git show-ref

Git command for listing references.
More information: <https://git-scm.com/docs/git-show-ref>.

- Show all refs in the repository:

`git show-ref`

- Show only heads references:

`git show-ref --heads`

- Show only tags references:

`git show-ref --tags`
