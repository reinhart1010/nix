# idevicebackup

Create or restore backups for iOS devices.
Note: This tool is outdated. Please see `idevicebackup2`.
More information: <https://manned.org/idevicebackup>.

- Create a backup of the device in the specified directory:

`idevicebackup backup `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Restore a backup from the specified directory:

`idevicebackup restore `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
