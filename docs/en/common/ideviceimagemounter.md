# ideviceimagemounter

Mount disk images on an iOS device.
More information: <https://manned.org/ideviceimagemounter>.

- Mount a disk image on the connected device:

`ideviceimagemounter `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/image_file</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/signature_file</span>

- List currently mounted disk images:

`ideviceimagemounter --list`
