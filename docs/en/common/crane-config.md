<hr># crane config

Get the configuration of an image.
More information: <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_config.md>.

- Get the configuration of an image:

`crane config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">image_name</span>

- Display help:

`crane config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-h|--help</span>
