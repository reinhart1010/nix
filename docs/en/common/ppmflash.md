# ppmflash

Brighten a PPM image file.
More information: <https://netpbm.sourceforge.net/doc/ppmflash.html>.

- Generate a PPM image as output that is `flashfactor` times brighter than the input PPM image:

`ppmflash `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">flashfactor</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.ppm</span>

- Display version:

`ppmflash -version`
