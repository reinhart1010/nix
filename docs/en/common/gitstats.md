# gitstats

Git repository statistics generator.
More information: <https://gitstats.sourceforge.net>.

- Generate statistics for a local repository:

`gitstats `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/git_repo/.git</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_folder</span>

- View generated statistics in a web browser on Windows (PowerShell)/macOS/Linux:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">Invoke-Item|open|xdg-open</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_folder/index.html</span>
