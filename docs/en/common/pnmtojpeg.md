<hr># pnmtojpeg

Converts a PNM image file to the JPEG/JFIF/EXIF image format.
More information: <https://netpbm.sourceforge.net/doc/pnmtojpeg.html>.

- Read a PNM image as input and produce a JPEG/JFIF/EXIF image as output:

`pnmtojpeg `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pnm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.jpg</span>

- Display version:

`pnmtojpeg -version`
