# kaggle

Official CLI for Kaggle implemented in Python 3.
More information: <https://github.com/Kaggle/kaggle-api>.

- View current configuration values:

`kaggle config view`

- Download a specific file from a competition dataset:

`kaggle competitions download `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">competition</span>` -f `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename</span>
