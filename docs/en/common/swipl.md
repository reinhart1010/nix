# swipl

SWI-Prolog - A comprehensive free Prolog environment.
More information: <https://www.swi-prolog.org/>.

- Start an interactive session:

`swipl`

- Execute a command without showing any output:

`swipl --quiet -t "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>`"`

- Execute a script:

`swipl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pl</span>

- Print all shell configuration variables:

`swipl --dump-runtime-variables`

- Print the version:

`swipl --version`
