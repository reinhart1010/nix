# pamtosrf

Convert Netpbm images to SRF format.
See also: `srftopam`.
More information: <https://netpbm.sourceforge.net/doc/pamtosrf.html>.

- Convert a Netpbm image or a stream of Netpbm images to SRF format:

`pamtosrf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pam</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.srf</span>
