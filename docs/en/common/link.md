# link

Create a hard link to an existing file.
For more options, see the `ln` command.
More information: <https://www.gnu.org/software/coreutils/manual/html_node/link-invocation.html>.

- Create a hard link from a new file to an existing file:

`link `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/existing_file</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/new_file</span>
