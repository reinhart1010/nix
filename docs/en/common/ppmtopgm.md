# ppmtopgm

Convert a PPM image to a PGM image.
More information: <https://netpbm.sourceforge.net/doc/ppmtopgm.html>.

- Convert PPM image to PGM image:

`ppmtopgm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pgm</span>

- Display version:

`ppmtopgm -version`
