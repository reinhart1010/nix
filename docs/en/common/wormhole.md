Get things from one computer to another, safely.
More information: <https://magic-wormhole.readthedocs.io>.

`wormhole send `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

`wormhole receive `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">wormhole_code</span>

`wormhole send`
