# idevicebackup2

Create or restore backups for devices running iOS 4 or later.
More information: <https://manned.org/idevicebackup2>.

- Create a backup of the device in the specified directory:

`idevicebackup2 backup `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Restore a backup from the specified directory:

`idevicebackup2 restore `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Enable encryption for backups:

`idevicebackup2 encryption on `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">password</span>

- List the files in the last completed backup:

`idevicebackup2 list`
