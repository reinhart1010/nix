<hr># ppmtoeyuv

Convert a PPM image to an Berkeley YUV file.
More information: <https://netpbm.sourceforge.net/doc/ppmtoeyuv.html>.

- Read a PPM image from the input file, convert it to a Berkeley YUV image and store it in the specified output file:

`ppmtoeyuv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input_file.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_file.eyuv</span>
