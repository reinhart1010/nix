# anytopnm

Converts an arbitrary type of image file to common image formats.
More information: <https://netpbm.sourceforge.net/doc/anytopnm.html>.

- Convert an input image to PBM, PGM, or PPM format irrespective of the input type:

`anytopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.pnm</span>

- Display version:

`anytopnm -version`
