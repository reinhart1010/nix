# az logout

Log out from an Azure subscription.
Part of `azure-cli`.
More information: <https://learn.microsoft.com/cli/azure/reference-index#az_logout>.

- Log out from the active account:

`az logout`

- Log out a specific username:

`az logout --username `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">alias@somedomain.com</span>
