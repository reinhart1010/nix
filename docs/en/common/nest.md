<hr># nest

Command-line tool to initialize, develop, and maintain Nest applications.
More information: <https://docs.nestjs.com/cli/overview>.

- Display information about installed nest version:

`nest info`

- Create a new NestJS project in a directory of the same name:

`nest new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">project_name</span>

- Build a specific NestJS project:

`nest build `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">project_name</span>

- Run a specific NestJS project:

`nest start `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">project_name</span>

- Import a library into the current NestJS project:

`nest add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">library_name</span>
