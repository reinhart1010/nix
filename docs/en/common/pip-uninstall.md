# pip uninstall

Uninstall Python packages.
More information: <https://pip.pypa.io>.

- Uninstall a package:

`pip uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Uninstall packages listed in a specific file:

`pip uninstall --requirement `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/requirements.txt</span>

- Uninstall package without asking for confirmation:

`pip uninstall --yes `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>
