# vectorize-pixelart

Convert PNG pixel art graphics to SVG/EPS vector images.
More information: <https://github.com/und3f/vectorize-pixelart>.

- Convert a PNG to a vector image format:

`vectorize-pixelart `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input.png</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.svg|.eps</span>
