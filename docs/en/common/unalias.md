# unalias

Remove aliases.
More information: <https://manned.org/unalias>.

- Remove an alias:

`unalias `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">alias_name</span>

- Remove all aliases:

`unalias -a`
