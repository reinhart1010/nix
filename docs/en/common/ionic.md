# ionic

A framework to build hybrid mobile apps.
More information: <https://ionicframework.com/docs/cli>.

- Create a new project:

`ionic start`

- Start a local dev server for app dev/testing:

`ionic serve`

- Generate new app component, directive, page, pipe, provider or tabs:

`ionic g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">page</span>

- Run app on an Android/iOS device:

`ionic cordova run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">android|ios</span>` --device`

- Check the health of an Ionic app:

`ionic doctor `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">check</span>

- Display versions of Ionic, Cordova, environment, etc.:

`ionic info`
