<hr># crane manifest

Get the manifest of an image.
More information: <https://github.com/google/go-containerregistry/blob/main/cmd/crane/doc/crane_manifest.md>.

- Get the manifest:

`crane manifest `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">image_name</span>

- Display help:

`crane manifest `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-h|--help</span>
