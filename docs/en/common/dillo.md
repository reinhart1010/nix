# dillo

A lightweight web browser intended for slow computers.
More information: <https://www.dillo.org/>.

- Launch Dillo:

`dillo`

- Launch Dillo with a specific window size and screen location:

`dillo --geometry `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">width</span>`x`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">height</span>`+`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">x_position</span>`+`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">y_position</span>

- Launch Dillo and open a specific URL:

`dillo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">duckduckgo.com</span>

- Launch Dillo and open a file or directory:

`dillo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>

- Launch Dillo in full-screen mode:

`dillo --fullwindow`

- Display version:

`dillo --version`

- Display help:

`dillo --help`
