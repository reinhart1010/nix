# flutter pub

Flutter's package manager.
Note: Packages are available on <https://pub.dev>. See also: `flutter`.
More information: <https://docs.flutter.dev/packages-and-plugins/using-packages>.

- Download/Update all packages specified in `pubspec.yaml`:

`flutter pub get`

- Add a package dependency to an app:

`flutter pub add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package1 package2 ...</span>

- Remove a package dependency from an app:

`flutter pub remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package1 package2 ...</span>

- Upgrade to the highest version of a package that is allowed by `pubspec.yaml`:

`flutter pub upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>
