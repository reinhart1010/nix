# git abort

Abort an ongoing rebase, merge, or cherry-pick.
Part of `git-extras`.
More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-abort>.

- Abort a Git rebase, merge, or cherry-pick:

`git abort`
