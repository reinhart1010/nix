# git merge-into

Merge one branch into another branch.
Part of `git-extras`.
More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-merge-into>.

- Merge a source branch into a specific destination branch:

`git merge-into `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">source_branch</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">destination_branch</span>

- Merge current branch into a specific destination branch:

`git merge-into `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">destination_branch</span>
