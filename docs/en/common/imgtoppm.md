# imgtoppm

Converts various image file formats to the PPM (Portable Pixmap) format.
More information: <https://netpbm.sourceforge.net/doc/imgtoppm.html>.

- Convert an input image to PPM format:

`imgtoppm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.ppm</span>

- Display version:

`imgtoppm -version`
