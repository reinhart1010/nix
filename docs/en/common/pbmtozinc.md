<hr># pbmtozinc

Convert a PBM image to a Zinc bitmap as used by the Zinc Interface Library Version 1.0.
More information: <https://netpbm.sourceforge.net/doc/pbmtozinc.html>.

- Convert a PBM image to a Zinc bitmap:

`pbmtozinc `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/image.pbm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.zinc</span>
