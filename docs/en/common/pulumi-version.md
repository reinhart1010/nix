# pulumi version

Display Pulumi's version number.
More information: <https://www.pulumi.com/docs/iac/cli/commands/pulumi_version/>.

- Display version:

`pulumi version`

- Display help:

`pulumi version `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-h|--help</span>
