# pixi

Developer Workflow and Environment Management for projects.
More information: <https://pixi.sh>.

- Initialize a new project:

`pixi init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/project</span>

- Add project dependencies:

`pixi add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">dependency1 dependency2 ...</span>

- Start a pixi shell in the project environment:

`pixi shell`

- Run a task in the project environment:

`pixi run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">task</span>

- Manage tasks in the project environment:

`pixi task `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>

- Print the help message:

`pixi `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>` --help`

- Clean environment and task cache:

`pixi clean`
