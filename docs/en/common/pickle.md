# pickle

A PHP extension installer based on Composer.
More information: <https://github.com/FriendsOfPHP/pickle>.

- Install a specific PHP extension:

`pickle install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">extension_name</span>

- Convert an existing PECL extension configuration to a Pickle configuration file:

`pickle convert `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Validate a PECL extension:

`pickle validate `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Package a PECL extension for release:

`pickle release `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
