# jadx

Dex to Java decompiler.
Produces Java source code from Android Dex and APK files.
More information: <https://github.com/skylot/jadx>.

- Decompile a Dex file into a directory:

`jadx `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Decompile a Dex file into a specific directory:

`jadx --output-dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>
