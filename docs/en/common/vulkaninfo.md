# vulkaninfo

Print system Vulkan information.
More information: <https://vulkan.lunarg.com/doc/view/latest/linux/vulkaninfo.html>.

- Print full Vulkan information:

`vulkaninfo`

- Print a summary:

`vulkaninfo --summary`

- Make a HTML document of the full Vulkan information:

`vulkaninfo --html`
