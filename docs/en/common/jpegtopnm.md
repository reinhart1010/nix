# jpegtopnm

Convert a JPEG/JFIF file to the PPM or PGM format.
More information: <https://netpbm.sourceforge.net/doc/jpegtopnm.html>.

- Convert JPEG/JFIF image to a PPM or PGM image:

`jpegtopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.jpg</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pnm</span>

- Display version:

`jpegtopnm -version`
