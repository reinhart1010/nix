# kotlin

Kotlin application launcher.
More information: <https://kotlinlang.org>.

- Run a jar file:

`kotlin `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename.jar</span>

- Display Kotlin and JVM version:

`kotlin -version`
