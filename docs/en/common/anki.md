<hr># anki

Powerful, intelligent flashcard program.
More information: <https://docs.ankiweb.net>.

- Launch `anki`:

`anki`

- Launch `anki` with a specific profile:

`anki -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">profile_name</span>

- Launch `anki` in a specific language:

`anki -l `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">language</span>

- Launch `anki` from a specific directory instead of the default (`~/Anki`):

`anki -b `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
