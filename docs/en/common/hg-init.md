# hg init

Create a new repository in the specified directory.
More information: <https://www.mercurial-scm.org/doc/hg.1.html#init>.

- Initialize a new repository in the current directory:

`hg init`

- Initialize a new repository in the specified directory:

`hg init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>
