<hr># npm update

Update packages in the current project.
More information: <https://docs.npmjs.com/cli/commands/npm-update>.

- Update all packages in the current project:

`npm update`

- Update a specific package in the current project:

`npm update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Update a package globally:

`npm update -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Update multiple packages at once:

`npm update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package1 package2 ...</span>
