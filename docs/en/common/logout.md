# logout

Exit a login shell.
More information: <https://www.gnu.org/software/bash/manual/bash.html#index-logout>.

- Exit a login shell:

`logout`

- Exit a login shell and specify a return value:

`logout `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">N</span>
