<hr># ybmtopbm

Convert a Bennet Yee "face" file to PBM.
More information: <https://netpbm.sourceforge.net/doc/ybmtopbm.html>.

- Convert a YBM image file to PBM:

`ybmtopbm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input_file.ybm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_file.pbm</span>
