Convert a Solitaire Image Recorder file to a PNM file.
More information: <https://netpbm.sourceforge.net/doc/sirtopnm.html>.

- Convert a SIR image to a PNM file:

`sirtopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input.sir</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.pnm</span>
