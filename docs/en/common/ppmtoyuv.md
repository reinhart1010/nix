# ppmtoyuv

Convert a PPM image to an Abekas YUV file.
More information: <https://netpbm.sourceforge.net/doc/ppmtoyuv.html>.

- Read a PPM image from the input file, convert it to an Abekas YUV image and store it in the specified output file:

`ppmtoyuv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input_file.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_file.yuv</span>
