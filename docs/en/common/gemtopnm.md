# gemtopnm

Convert a GEM image file into a PNM image.
More information: <https://netpbm.sourceforge.net/doc/gemtopnm.html>.

- Convert a GEM image file to a PNM image:

`gemtopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.img</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.pnm</span>

- Describe the contents of the specified GEM image:

`gemtopnm -d `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.img</span>

- Display version:

`gemtopnm -version`
