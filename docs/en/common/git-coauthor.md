# git coauthor

Add another author to the latest commit. Since this command rewrites the Git history, `--force` will be needed when pushing next time.
Part of `git-extras`.
More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-coauthor>.

- Insert an additional author to the last Git commit:

`git coauthor `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">name</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">name@example.com</span>
