<hr># giftopnm

Convert a GIF file into a PNM image.
More information: <https://netpbm.sourceforge.net/doc/giftopnm.html>.

- Convert a GIF image to a Netpbm image pixel-for-pixel:

`giftopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input.gif</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.pnm</span>

- Display version:

`giftopnm -version`
