# octave

A programming language for scientific computing.
More information: <https://docs.octave.org/latest/Invoking-Octave-from-the-Command-Line.html>.

- Start an interactive session:

`octave`

- Execute a specific script file:

`octave `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/script.m</span>

- Execute a script file with specific arguments:

`octave `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/script.m</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">argument1 argument2 ...</span>

- Start an interactive session with a GUI:

`octave --gui`

- Display help:

`octave --help`

- Display version:

`octave --version`
