# git reset-file

Revert a file to HEAD or a commit.
Part of `git-extras`.
More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-reset-file>.

- Reset a file to HEAD:

`git reset-file `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Reset a file to a specific commit:

`git reset-file `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">commit_hash</span>
