# lolcat

Put a rainbow in everything you `cat` to the console.
More information: <https://github.com/busyloop/lolcat>.

- Print a file to the console in rainbow colors:

`lolcat `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Print the result of a text-producing command in rainbow colors:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fortune</span>` | lolcat`

- Print a file to the console with animated rainbow colors:

`lolcat -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Print a file to the console with 24-bit (truecolor) rainbow colors:

`lolcat -t `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>
