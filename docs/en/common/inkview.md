# inkview

Inkscape graphical SVG previewer.
Also functions as a slideshow viewer.
More information: <http://wiki.inkscape.org/wiki/index.php/Inkview>.

- Preview an SVG:

`inkview `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.svg</span>

- Preview multiple SVGs (use arrow keys to navigate):

`inkview `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file1.svg path/to/file2.svg ...</span>
