# progpilot

A PHP static analysis tool for detecting security vulnerabilities.
More information: <https://github.com/designsecurity/progpilot>.

- Analyze the current directory:

`progpilot`

- Analyze a specific file or directory:

`progpilot `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file_or_directory</span>

- Specify a custom configuration file:

`progpilot --configuration `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/configuration.yml</span>
