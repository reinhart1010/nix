# pamtofits

Convert a Netpbm image to the Flexible Image Transport System (FITS) format.
See also: `fitstopnm`.
More information: <https://netpbm.sourceforge.net/doc/pamtofits.html>.

- Convert a Netpbm image to the FITS format:
