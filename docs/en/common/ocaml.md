# ocaml

The OCaml repl (read-evaluate-print-loop).
Interprets Ocaml commands.
More information: <https://ocaml.org>.

- Read OCaml commands from the user and execute them:

`ocaml`

- Read OCaml commands from a file and execute them:

`ocaml `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.ml</span>

- Run OCaml script with modules:

`ocaml `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">module1</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">module2</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.ml</span>
