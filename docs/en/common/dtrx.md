# dtrx

"Do The Right eXtraction" - extract any archive into new directory by guessing the tool from extension.
More information: <https://github.com/dtrx-py/dtrx>.

- Extract archive, guessing the extraction tool from extension:

`dtrx `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Extract archive, overwrite any existing target output:

`dtrx --overwrite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Extract archive, put everything into current directory:

`dtrx --flat `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>
