# grunt

A JavaScript task runner for automating processes.
More information: <https://github.com/gruntjs/grunt-cli>.

- Run the default task process:

`grunt`

- Run one or more specific space-separated task(s):

`grunt `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">task_name</span>

- Specify an alternative configuration file:

`grunt --gruntfile `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Specify an alternative base path for relative files:

`grunt --base `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Specify an additional directory to scan for tasks in:

`grunt --tasks `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Perform a dry-run without writing any files:

`grunt --no-write`

- List all available options:

`grunt --help`
