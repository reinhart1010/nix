# pamtoqoi

Convert a Netpbm image to a QOI image (Quite OK Image format).
More information: <https://netpbm.sourceforge.net/doc/pamtoqoi.html>.

- Convert a Netpbm image to the QOI format:

`pamtoqoi `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/image.pnm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.qoi</span>
