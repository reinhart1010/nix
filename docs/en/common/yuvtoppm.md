<hr># yuvtoppm

Convert Abekas YUV bytes to PPM.
More information: <https://netpbm.sourceforge.net/doc/yuvtoppm.html>.

- Read Akebas YUV bytes from the specified input file, convert them to a PPM image and store them in the specified output file:

`yuvtoppm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">width</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">height</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input_file.yuv</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_file.ppm</span>
