# ppmdim

Dim a PPM image.
Partly superseded by `pamfunc -multiplier`.
More information: <https://netpbm.sourceforge.net/doc/ppmdim.html>.

- Dim the specified PPM image by dimfactor:

`ppmdim `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">0.6</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.ppm</span>
