# nrm

npm registry manager.
Helps to easily switch between different npm registries.
More information: <https://github.com/Pana/nrm>.

- List all registries:

`nrm ls`

- Change to a particular registry:

`nrm use `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">registry</span>

- Show the response time for all registries:

`nrm test`

- Add a custom registry:

`nrm add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">registry</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">url</span>

- Delete a registry:

`nrm del `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">registry</span>
