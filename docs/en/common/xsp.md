# xsp

Mono ASP.NET Web Server.
More information: <https://www.mono-project.com/docs/web/aspnet/>.

- Listen on all interfaces (`0.0.0.0`) and port `8080`:

`xsp`

- Listen on a specific IP address and port:

`xsp --address `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">127.0.0.1</span>` --port `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">8000</span>
