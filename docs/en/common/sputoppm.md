<hr># sputoppm

Convert an Atari uncompressed Spectrum image to a PPM image.
More information: <https://netpbm.sourceforge.net/doc/sputoppm.html>.

- Convert an SPU file to a PPM image:

`sputoppm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input.spu</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output.ppm</span>
