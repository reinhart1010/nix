# clockwork-cli

A command-line interface for the Clockwork PHP debugging framework.
More information: <https://github.com/ptrofimov/clockwork-cli>.

- Monitor Clockwork logs for the current project:

`clockwork-cli`

- Monitor Clockwork logs for a specific project:

`clockwork-cli `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- Monitor Clockwork logs for multiple projects:

`clockwork-cli `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory1 path/to/directory2 …</span>
