# gcrane help

Help provides help for any command in the application.
More information: <https://github.com/google/go-containerregistry/blob/main/cmd/gcrane/README.md>.

- Display help for a subcommand:

`gcrane help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>

- Display help:

`grance help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">-h|--help</span>
