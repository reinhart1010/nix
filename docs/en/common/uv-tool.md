# uv tool

Install and run commands provided by Python packages.
More information: <https://docs.astral.sh/uv/reference/cli/#uv-tool>.

- Run a command from a package, without installing it:

`uv tool run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>

- Install a Python package system-wide:

`uv tool install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Upgrade an installed Python package:

`uv tool upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- Uninstall a Python package:

`uv tool uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package</span>

- List Python packages installed system-wide:

`uv tool list`
