# docker diff

Inspect changes to files or directories on a container's filesystem.
More information: <https://docs.docker.com/reference/cli/docker/container/diff/>.

- Inspect the changes to a container since it was created:

`docker diff `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">container</span>

- Display help:

`docker diff --help`
