# elixir

Elixir programming language interpreter.
More information: <https://elixir-lang.org>.

- Run an Elixir file:

`elixir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Evaluate Elixir code by passing it as an argument:

`elixir -e "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">code</span>`"`
