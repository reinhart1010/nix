# peco

Interactive filtering tool.
More information: <https://github.com/peco/peco>.

- Start `peco` on all files in the specified directory:

`find `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>` -type f | peco`

- Start `peco` for running processes:

`ps aux | peco`

- Start `peco` with a specified query:

`peco --query "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">query</span>`"`
