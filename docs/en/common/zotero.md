# zotero

Manage your bibliographies.
More information: <https://www.zotero.org/support>.

- Run with the GUI:

`zotero`

- Run in headless mode:

`zotero --headless`

- Run with a specific profile:

`zotero -P `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">profile</span>

- Run the Migration Assistant:

`zotero --migration`
