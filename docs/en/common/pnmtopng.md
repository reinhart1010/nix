# pnmtopng

Convert a PNM image file to PNG image format.
More information: <https://netpbm.sourceforge.net/doc/pnmtopng.html>.

- Read a PNM image as input and produce a PNG image as output:

`pnmtopng `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.pnm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file.png</span>

- Display version:

`pnmtopng -version`
