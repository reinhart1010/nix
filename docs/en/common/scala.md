# scala

Scala application launcher and interactive interpreter.
More information: <https://scala-lang.org>.

- Start a REPL (interactive shell):

`scala`

- Start the interpreter with a dependency in the classpath:

`scala -classpath `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename.jar</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>

- Execute a Scala script:

`scala `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">script.scala</span>

- Execute a `.jar` program:

`scala `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">filename.jar</span>

- Execute a single Scala command in the command-line:

`scala -e `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">command</span>
