# moe

A WYSIWYG text editor for ISO-8859-15 encoded text.
More information: <https://www.gnu.org/software/moe/moe.html>.

- Open moe and create a backup file (file~) when saving edits:

`moe `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Open a file as read-only:

`moe --read-only `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Edit a file without creating backups:

`moe --no-backup `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Edit a file ignoring case in searches:

`moe --ignore-case `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Save and Quit:

`Ctrl + X`
