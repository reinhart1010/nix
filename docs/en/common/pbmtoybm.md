<hr># pbmtoybm

Convert a PBM file to a Bennet Yee "face" file.
More information: <https://netpbm.sourceforge.net/doc/pbmtoybm.html>.

- Convert a PBM image file to YBM:

`pbmtoybm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/input_file.pbm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/output_file.ybm</span>
