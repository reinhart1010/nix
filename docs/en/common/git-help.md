# git help

Display help information about Git.
More information: <https://git-scm.com/docs/git-help>.

- Display help about a specific Git subcommand:

`git help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Display help about a specific Git subcommand in a web browser:

`git help --web `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Display a list of all available Git subcommands:

`git help --all`

- List the available guides:

`git help --guide`

- List all possible configuration variables:

`git help --config`
