# sui

Interact with the Sui network.
More information: <https://docs.sui.io/references/cli/cheatsheet>.

- Execute a Sui subcommand:

`sui `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Build tools for a smart contract:

`sui move `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Publish smart contracts, get object information, execute transactions and more:

`sui client `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">subcommand</span>

- Start a local network:

`sui start`

- Update from source:

`cargo install --locked --git https://github.com/MystenLabs/sui.git --branch testnet sui`
