# npm uninstall

Remove a package.
More information: <https://docs.npmjs.com/cli/v8/commands/npm-uninstall>.

- Remove a package from the current project:

`npm uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package_name</span>

- Remove a package globally:

`npm uninstall -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package_name</span>

- Remove multiple packages at once:

`npm uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">package_name1 package_name2 ...</span>
