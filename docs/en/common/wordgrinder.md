# wordgrinder

Command-line word processor.
More information: <https://cowlark.com/wordgrinder>.

- Start WordGrinder (loads a blank document by default):

`wordgrinder`

- Open a given file:

`wordgrinder `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/file</span>

- Show the menu:

`Alt + M`
