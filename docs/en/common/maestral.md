# maestral

A lightweight Dropbox client for macOS and Linux.
More information: <https://maestral.app/cli>.

- Start the GUI:

`maestral gui`

- Print current status of Maestral:

`maestral status`

- Pause syncing:

`maestral pause`

- Resume syncing:

`maestral resume`
