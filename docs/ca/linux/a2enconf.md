# a2enconf

Activa un fitxer de configuració d'Apache en sistemes operatius basats en debian.
Més informació: <https://manned.org/a2enconf.8>.

- Activa un fitxer de configuració:

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fitxer_configuració</span>

- No mostris missatges informatius:

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fitxer_configuració</span>
