# a2dismod

Desactiva un mòdul Apache en sistemes operatius basats en Debian.
Més informació: <https://manned.org/a2dismod.8>.

- Desactiva un mòdul:

`sudo a2dismod `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">mòdul</span>

- No mostrius missatges informatius:

`sudo a2dismod --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">mòdul</span>
