# a2enmod

Activa un mòdul d'Apache en sistemes operatius basats en Debian.
Més informació: <https://manpages.debian.org/latest/apache2/a2enmod.8.en.html>.

- Activa un mòdul:

`sudo a2enmod `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">mòdul</span>

- No mostris missatges informatius:

`sudo a2enmod --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">mòdul</span>
