# mkdir

Создает каталог или подкаталог в файловой системе.
Больше информации: <https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- Создать новый каталог:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">путь\до\папки</span>

- Чтобы создать дерево каталогов в корневом каталоге введите:

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">путь\до\под_папки</span>
