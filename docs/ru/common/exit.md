# exit

Выйти из оболочки.
Больше информации: <https://pubs.opengroup.org/onlinepubs/9699919799/utilities/V3_chap02.html#exit>.

- Выход из оболочки с кодом выхода последней выполненной команды:

`exit`

- Выйти из оболочки с указанным кодом выхода:

`exit `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">код_выхода</span>
