<hr># toolbox list

기존 `toolbox` 컨테이너와 이미지 나열.
더 많은 정보: <https://manned.org/toolbox-list.1>.

- 모든 `toolbox` 컨테이너와 이미지 나열:

`toolbox list`

- `toolbox` 컨테이너만 나열:

`toolbox list --containers`

- `toolbox` 이미지만 나열:

`toolbox list --images`
