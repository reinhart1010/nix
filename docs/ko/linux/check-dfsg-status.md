# check-dfsg-status

Debian 기반 OS에 설치된 비자유 소프트웨어 패키지를 보고.
이 명령은 이전에 `vrms`로 알려졌습니다.
더 많은 정보: <https://debian.pages.debian.net/check-dfsg-status/>.

- 비자유 및 contrib 패키지와 그 설명 나열:

`check-dfsg-status`

- 패키지 이름만 출력:

`check-dfsg-status --sparse`
