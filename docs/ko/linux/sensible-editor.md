# sensible-editor

기본 편집기를 엽니다.
더 많은 정보: <https://manned.org/sensible-editor>.

- 기본 편집기로 파일 열기:

`sensible-editor `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 파일 끝에 커서를 위치시키고 기본 편집기로 파일 열기:

`sensible-editor + `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 10번째 줄 시작에 커서를 위치시키고 기본 편집기로 파일 열기:

`sensible-editor +10 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 세 개의 파일을 수직 분할된 편집기 창에서 동시에 열기:

`sensible-editor -O3 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일1 경로/대상/파일2 경로/대상/파일3</span>
