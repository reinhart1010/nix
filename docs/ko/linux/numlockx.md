<hr># numlockx

X11 세션에서 넘버락 키 상태를 제어합니다.
더 많은 정보: <https://www.mike-devlin.com/linux/README-numlockx.htm>.

- 현재 넘버락 상태 표시:

`numlockx status`

- 넘버락 켜기:

`numlockx on`

- 넘버락 끄기:

`numlockx off`

- 현재 상태 전환:

`numlockx toggle`
