# ifdown

네트워크 인터페이스 비활성화.
더 많은 정보: <https://manned.org/ifdown>.

- 인터페이스 eth0 비활성화:

`ifdown `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">eth0</span>

- 활성화된 모든 인터페이스 비활성화:

`ifdown -a`
