<hr># xbps-install

XBPS 패키지를 (재)설치하고 업데이트하는 도구.
같이 보기: `xbps`.
더 많은 정보: <https://manned.org/xbps-install.1>.

- 새 패키지 설치:

`xbps-install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 모든 패키지를 동기화하고 업데이트:

`xbps-install --sync --update`
