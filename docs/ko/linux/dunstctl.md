# dunstctl

`dunst` 제어 명령어.
더 많은 정보: <https://manned.org/dunstctl>.

- 알림 일시 중지:

`dunstctl set-paused true`

- 알림 일시 중지 해제:

`dunstctl set-paused false`

- 모든 알림 닫기:

`dunstctl close-all`

- 도움말 표시:

`dunstctl --help`
