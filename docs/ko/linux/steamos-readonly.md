# steamos-readonly

파일 시스템의 읽기 전용 상태 설정.
더 많은 정보: <https://gitlab.com/users/evlaV/projects>.

- 파일 시스템을 수정 가능하게 설정:

`sudo steamos-readonly disable`

- 파일 시스템을 읽기 전용으로 설정:

`sudo steamos-readonly enable`
