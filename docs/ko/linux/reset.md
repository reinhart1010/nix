<hr># reset

현재 터미널을 초기화합니다. 터미널 화면 전체를 지웁니다.
더 많은 정보: <https://manned.org/reset>.

- 현재 터미널 초기화:

`reset`

- 대신 터미널 유형 표시:

`reset -q`
