# inxi

디버깅을 위한 시스템 정보 및 자원의 요약 정보를 표시합니다.
더 많은 정보: <https://manned.org/inxi>.

- CPU, 메모리, 하드 드라이브 및 커널 정보 요약 출력:

`inxi`

- CPU, 메모리, 디스크, 네트워크 및 프로세스의 전체 설명 출력:

`inxi -Fz`

- 배포판 저장소 정보 출력:

`inxi -r`
