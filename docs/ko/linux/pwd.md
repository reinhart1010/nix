# pwd

현재 작업 중인 디렉토리의 이름을 출력합니다.
더 많은 정보: <https://www.gnu.org/software/coreutils/pwd>.

- 현재 디렉토리 출력:

`pwd`

- 현재 디렉토리를 출력하고 모든 심볼릭 링크를 해석 (즉, "물리적" 경로 표시):

`pwd --physical`

- 현재 논리적 디렉토리 출력:

`pwd --logical`
