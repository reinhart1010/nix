# apx

다중 소스를 지원하여 하위 시스템에 패키지를 설치할 수 있는 패키지 관리 도구.
더 많은 정보: <https://github.com/Vanilla-OS/apx>.

- 패키지 관리자 관리에 대한 문서 보기:

`tldr apx pkgmanagers`

- 스택 관리에 대한 문서 보기:

`tldr apx stacks`

- 하위 시스템 관리에 대한 문서 보기:

`tldr apx subsystems`
