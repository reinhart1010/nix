# sestatus

현재 SELinux 상태를 표시합니다.
더 많은 정보: <https://manned.org/sestatus>.

- 현재 상태 표시:

`sestatus`

- 모든 정책 불리언의 현재 상태 표시:

`sestatus -b`

- 현재 파일 및 프로세스 컨텍스트 표시:

`sestatus -v`
