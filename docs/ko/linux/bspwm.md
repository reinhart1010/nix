# bspwm

바이너리 공간 분할 기반의 타일링 윈도우 매니저.
같이 보기: `bspc`, 제어를 위해 사용합니다.
더 많은 정보: <https://github.com/baskerville/bspwm>.

- `bspwm` 시작 (이 명령을 실행할 때 기존의 윈도우 매니저가 열려 있으면 안 됩니다):
