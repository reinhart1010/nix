# snake4

터미널에서 실행하는 뱀 게임.
더 많은 정보: <https://manned.org/snake4.6>.

- 뱀 게임 시작:

`snake4`

- 레벨 선택:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1|2|3|4|5</span>

- 뱀 이동:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">위쪽|아래쪽|왼쪽|오른쪽 화살표 키</span>

- 게임 일시 중지:

`<스페이스바>`

- 게임 종료:

`q`

- 최고 점수 표시:

`snake4 --highscores`
