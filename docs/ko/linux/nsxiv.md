# nsxiv

Neo Simple X 이미지 뷰어.
더 많은 정보: <https://nsxiv.codeberg.page/man>.

- 이미지 열기:

`nsxiv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일1 경로/대상/파일2 ...</span>

- 이미지 모드에서 폴더의 이미지 열기:

`nsxiv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더1 경로/대상/폴더2 ...</span>

- 폴더를 재귀적으로 검색하여 이미지를 보기:

`nsxiv -r `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더1 경로/대상/폴더2 ...</span>

- nsxiv 종료:

`q`

- 썸네일 모드로 전환하거나 선택한 이미지를 이미지 모드에서 열기:

`<Return>`

- 이미지 모드에서 이미지를 앞으로 넘기기:

`n`

- 이미지 모드에서 이미지를 뒤로 넘기기:

`p`
