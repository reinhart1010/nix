<hr># updpkgsums

`PKGBUILD`의 소스 체크섬을 업데이트합니다.
기존 해시 알고리즘이 사용되지 않는 경우, SHA256이 사용됩니다.
더 많은 정보: <https://manned.org/updpkgsums>.

- `PKGBUILD`의 체크섬 업데이트:

`updpkgsums`

- 도움말 표시:

`updpkgsums -h`

- 버전 표시:

`updpkgsums -v`
