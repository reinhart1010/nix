# goobook

`mutt` 또는 명령줄에서 Google 연락처에 접근.
더 많은 정보: <https://manned.org/goobook>.

- OAuth2를 사용하여 `goobook`이 Google 연락처에 접근 허용:

`goobook authenticate`

- 모든 연락처를 XML 형식으로 출력(`stdout`):

`goobook dump_contacts`
