# gnome-extensions

터미널에서 GNOME 확장 관리.
더 많은 정보: <https://wiki.gnome.org/Projects/GnomeShell/Extensions>.

- 설치된 모든 확장 나열:

`gnome-extensions list`

- 특정 확장에 대한 정보 표시:

`gnome-extensions info "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">확장_ID</span>`"`

- 특정 확장 활성화:

`gnome-extensions enable "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">확장_ID</span>`"`

- 특정 확장 비활성화:

`gnome-extension disable "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">확장_ID</span>`"`

- 특정 확장 제거:

`gnome-extension uninstall "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">확장_ID</span>`"`

- 특정 하위 명령(예: `list`)에 대한 도움말 표시:

`gnome-extensions help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위_명령</span>

- 버전 표시:

`gnome-extensions version`
