# nitrogen

X 윈도우용 데스크탑 배경 탐색기 및 설정기.
더 많은 정보: <https://github.com/l3ib/nitrogen>.

- 특정 디렉토리의 배경 화면을 보고 설정:

`nitrogen `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>

- 자동 크기 설정으로 배경 화면 설정:

`nitrogen --set-auto `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 이전 배경 화면 복원:

`nitrogen --restore`
