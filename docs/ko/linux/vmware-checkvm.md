# vmware-checkvm

현재 호스트가 VMware VM인지 확인.
더 많은 정보: <https://manned.org/vmware-checkvm>.

- 현재 VMware 소프트웨어 버전 반환 (시스템이 VM인지 여부는 종료 상태에 따라 결정):

`vmware-checkvm`

- VMware 하드웨어 버전 반환:

`vmware-checkvm -h`
