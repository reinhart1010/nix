# add-apt-repository

`apt` 저장소 정의를 관리합니다.
더 많은 정보: <https://manned.org/apt-add-repository>.

- 새 `apt` 저장소 추가:

`add-apt-repository `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">저장소_명세</span>

- `apt` 저장소 제거:

`add-apt-repository --remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">저장소_명세</span>

- 저장소 추가 후 패키지 캐시 업데이트:

`add-apt-repository --update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">저장소_명세</span>

- 저장소에서 소스 패키지 다운로드 허용:

`add-apt-repository --enable-source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">저장소_명세</span>
