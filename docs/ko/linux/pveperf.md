# pveperf

Proxmox 서버의 벤치마킹 도구. CPU 및 하드 디스크 성능 데이터를 수집합니다.
더 많은 정보: <https://pve.proxmox.com/pve-docs/pveperf.1.html>.

- `/`에 마운트된 하드 디스크의 CPU 및 하드 디스크 성능 데이터를 표시:

`pveperf`
