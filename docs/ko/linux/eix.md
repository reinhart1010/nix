# eix

로컬 Gentoo 패키지를 검색하는 유틸리티.
`eix-update`를 사용하여 로컬 패키지 캐시를 업데이트합니다.
더 많은 정보: <https://wiki.gentoo.org/wiki/Eix>.

- 패키지 검색:

`eix `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">검색어</span>

- 설치된 패키지 검색:

`eix --installed `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">검색어</span>

- 패키지 설명에서 검색:

`eix --description "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">설명</span>`"`

- 패키지 라이선스로 검색:

`eix --license `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이선스</span>

- 검색에서 결과 제외:

`eix --not --license `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이선스</span>
