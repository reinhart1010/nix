# zdump

시간대 정보를 출력.
더 많은 정보: <https://manned.org/zdump>.

- 특정 시간대의 현재 시간 출력:

`zdump `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">시간대</span>

- 도움말 표시:

`zdump --help`
