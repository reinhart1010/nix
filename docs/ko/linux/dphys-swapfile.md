<hr># dphys-swapfile

Debian 기반 Linux 시스템에서 스왑 파일을 관리합니다.
더 많은 정보: <https://manned.org/dphys-swapfile>.

- 스왑 파일 비활성화:

`dphys-swapfile swapoff`

- 스왑 파일 활성화:

`dphys-swapfile swapon`

- 새 스왑 파일 생성:

`dphys-swapfile setup`
