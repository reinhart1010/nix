# a2dissite

Debian 기반 운영 체제에서 Apache 가상 호스트 비활성화.
더 많은 정보: <https://manned.org/a2dissite.8>.

- 가상 호스트 비활성화:

`sudo a2dissite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">가상_호스트</span>

- 정보성 메시지를 표시하지 않음:

`sudo a2dissite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">가상_호스트</span>
