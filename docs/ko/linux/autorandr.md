# autorandr

화면 레이아웃을 자동으로 변경합니다.
더 많은 정보: <https://github.com/phillipberndt/autorandr>.

- 현재 화면 레이아웃 저장:

`autorandr --save `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로필_이름</span>

- 저장된 프로필 표시:

`autorandr`

- 감지된 첫 번째 프로필 불러오기:

`autorandr --change`

- 특정 프로필 불러오기:

`autorandr --load `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로필_이름</span>

- 기본 프로필 설정:

`autorandr --default `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로필_이름</span>
