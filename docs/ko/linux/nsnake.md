# nsnake

터미널에서 즐기는 Snake 게임.
더 많은 정보: <https://github.com/alexdantas/nsnake/>.

- Snake 게임 시작:

`nsnake`

- Snake 조작:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">위|아래|왼쪽|오른쪽 화살표 키</span>

- 게임 일시 정지/재개:

`p`

- 게임 종료:

`q`

- 게임 중 도움말 표시:

`h`
