# tlp

Linux용 고급 전원 관리 도구.
같이 보기: `tlp-stat`.
더 많은 정보: <https://linrunner.de/tlp/>.

- 설정 적용 (현재 전원 공급원에 따라):

`sudo tlp start`

- 배터리 설정 적용 (현재 전원 공급원을 무시하고):

`sudo tlp bat`

- AC 설정 적용 (현재 전원 공급원을 무시하고):

`sudo tlp ac`
