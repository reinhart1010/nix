# dhcpcd

DHCP 클라이언트.
더 많은 정보: <https://roy.marples.name/projects/dhcpcd>.

- 모든 주소 임대 해제:

`sudo dhcpcd --release`

- DHCP 서버에 새 임대 요청:

`sudo dhcpcd --rebind`
