# do-release-upgrade

Ubuntu 릴리스 업그레이더.
더 많은 정보: <https://manned.org/do-release-upgrade.8>.

- 최신 릴리스로 업그레이드:

`sudo do-release-upgrade`

- 최신 개발 릴리스로 업그레이드:

`sudo do-release-upgrade --devel-release`

- 최신 제안된 릴리스로 업그레이드:

`sudo do-release-upgrade --proposed`
