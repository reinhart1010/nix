<hr># gnome-terminal

GNOME 터미널 에뮬레이터.
더 많은 정보: <https://help.gnome.org/users/gnome-terminal/stable/>.

- 새 GNOME 터미널 창 열기:

`gnome-terminal`

- 새 터미널 창에서 특정 명령 실행:

`gnome-terminal -- `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령어</span>

- 마지막으로 열린 창에 새 탭 열기:

`gnome-terminal --tab`

- 새 탭의 제목 설정:

`gnome-terminal --tab --title "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">제목</span>`"`
