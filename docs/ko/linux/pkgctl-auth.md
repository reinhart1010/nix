<hr># pkgctl auth

`pkgctl`을 GitLab과 같은 서비스에 인증합니다.
더 많은 정보: <https://manned.org/pkgctl-auth.1>.

- GitLab 인스턴스에 `pkgctl`을 인증:

`pkgctl auth login`

- 인증 상태 보기:

`pkgctl auth status`
