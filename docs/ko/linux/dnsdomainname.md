# dnsdomainname

시스템의 DNS 도메인 이름을 표시합니다.
참고: 이 도구는 `gethostname`을 사용하여 시스템의 호스트명을 얻고, `getaddrinfo`를 사용하여 정규화된 이름으로 해석합니다.
더 많은 정보: <https://www.gnu.org/software/inetutils/manual/html_node/dnsdomainname-invocation.html>.

- 시스템의 DNS 도메인 이름 표시:

`dnsdomainname`
