<hr># pipewire

PipeWire 데몬 시작.
더 많은 정보: <https://docs.pipewire.org/page_man_pipewire_1.html>.

- PipeWire 데몬 시작:

`pipewire`

- 다른 설정 파일 사용:

`pipewire --config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.conf</span>

- 상세 수준 설정 (error, warn, info, debug 또는 trace):

`pipewire -`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">v|vv|...|vvvvv</span>

- 도움말 표시:

`pipewire --help`
