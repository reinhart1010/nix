<hr># manpath

매뉴얼 페이지의 검색 경로를 확인합니다.
더 많은 정보: <https://manned.org/manpath>.

- 매뉴얼 페이지를 찾는 데 사용되는 검색 경로 표시:

`manpath`

- 전체 전역 manpath 표시:

`manpath --global`
