<hr># ctop

CPU, 메모리 및 블록 IO 사용에 대한 실시간 메트릭으로 컨테이너 성능 및 상태를 즉시 시각화.
더 많은 정보: <https://github.com/bcicen/ctop>.

- [a]활성 컨테이너만 표시:

`ctop -a`

- 컨테이너 정렬 순서를 [r]반대로:

`ctop -r`

- 기본 색상을 [i]반전:

`ctop -i`

- [h]도움말 표시:

`ctop -h`
