<hr># nmcli monitor

NetworkManager 연결 상태 변경 사항 모니터링.
이 하위 명령은 `nmcli m`으로도 호출할 수 있습니다.
더 많은 정보: <https://networkmanager.pages.freedesktop.org/NetworkManager/NetworkManager/nmcli.html>.

- NetworkManager 변경 사항 모니터링 시작:

`nmcli monitor`
