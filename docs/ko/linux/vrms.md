# vrms

Debian 기반 OS에 설치된 비자유 패키지를 보고.
더 많은 정보: <https://salsa.debian.org/debian/check-dfsg-status>.

- 비자유 및 contrib 패키지 (및 설명) 나열:

`vrms`

- 패키지 이름만 출력:

`vrms --sparse`
