# eopkg

Solus용 패키지 관리 도구.
더 많은 정보: <https://help.getsol.us/docs/user/package-management/basics>.

- 특정 패키지 설치:

`sudo eopkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 모든 패키지 업데이트:

`sudo eopkg upgrade`

- 패키지 검색:

`sudo eopkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">검색어</span>
