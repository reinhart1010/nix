<hr># nitch

Nim으로 완전히 작성된 작고 매우 빠른 시스템 정보 표시 도구.
더 많은 정보: <https://github.com/ssleert/nitch>.

- 시스템 정보 표시 (호스트명, 커널, 가동 시간 등):

`nitch`

- [h]도움말 표시:

`nitch --help`

- [v]버전 표시:

`nitch --version`
