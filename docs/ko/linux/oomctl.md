# oomctl

`systemd-oomd`에 저장된 상태를 분석합니다.
더 많은 정보: <https://www.freedesktop.org/software/systemd/man/oomctl.html>.

- `systemd-oomd`에 의해 저장된 cgroups 및 시스템 컨텍스트의 현재 상태를 표시:

`oomctl dump`
