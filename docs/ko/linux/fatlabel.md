# fatlabel

더 많은 정보: <https://manned.org/fatlabel>.

- FAT32 파티션의 레이블 가져오기:

`fatlabel `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/dev/sda1</span>
