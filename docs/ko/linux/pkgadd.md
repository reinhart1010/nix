# pkgadd

CRUX 시스템에 패키지를 추가합니다.
더 많은 정보: <https://docs.oracle.com/cd/E88353_01/html/E72487/pkgadd-8.html>.

- 로컬 소프트웨어 패키지 설치:

`pkgadd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 이미 설치된 패키지를 로컬 패키지로 업데이트:

`pkgadd -u `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>
