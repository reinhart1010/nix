# xman

X Window System용 매뉴얼 페이지 뷰어.
더 많은 정보: <https://manned.org/xman>.

- 3버튼 창에서 `xman` 시작:

`xman`

- 주어진 파일에 저장된 매뉴얼 페이지 출력 열기:

`xman -helpfile `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">파일명</span>

- 매뉴얼 페이지와 디렉터리 모두 표시:

`xman -bothshown`
