# a2enconf

Debian 기반 OS에서 Apache 설정 파일을 활성화합니다.
더 많은 정보: <https://manned.org/a2enconf.8>.

- 설정 파일 활성화:

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">설정_파일</span>

- 정보 메시지를 표시하지 않음:

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">설정_파일</span>
