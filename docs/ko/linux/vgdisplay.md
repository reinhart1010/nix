Logical Volume Manager (LVM) 볼륨 그룹 정보를 표시합니다.
같이 보기: `lvm`.
더 많은 정보: <https://manned.org/vgdisplay>.

- 모든 볼륨 그룹 정보 표시:

`sudo vgdisplay`
