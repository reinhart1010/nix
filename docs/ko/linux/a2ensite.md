# a2ensite

Debian 기반 운영 체제에서 Apache 가상 호스트를 활성화합니다.
더 많은 정보: <https://manned.org/a2ensite.8>.

- 가상 호스트 활성화:

`sudo a2ensite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">가상_호스트</span>

- 정보 메시지 표시 안 함:

`sudo a2ensite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">가상_호스트</span>
