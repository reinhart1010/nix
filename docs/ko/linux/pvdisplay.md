# pvdisplay

Logical Volume Manager (LVM) 물리 볼륨 정보를 표시합니다.
같이 보기: `lvm`.
더 많은 정보: <https://manned.org/pvdisplay>.

- 모든 물리 볼륨 정보 표시:

`sudo pvdisplay`

- 드라이브 `/dev/sdXY`의 물리 볼륨 정보 표시:

`sudo pvdisplay `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">/dev/sdXY</span>
