<hr># caffeine

전체 화면 모드에서 데스크톱의 유휴 상태를 방지합니다.
더 많은 정보: <https://manned.org/caffeine>.

- caffeine 서버 시작:

`caffeine`

- 도움말 표시:

`caffeine --help`

- 버전 표시:

`caffeine --version`
