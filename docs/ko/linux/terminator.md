# terminator

여러 GNOME 터미널을 하나의 창에서 배열.
더 많은 정보: <https://gnome-terminator.org/>.

- `terminator` 창 시작:

`terminator`

- 전체 화면 창으로 시작:

`terminator -f`

- 터미널을 가로로 분할:

`<Ctrl> + <Shift> + O`

- 터미널을 세로로 분할:

`<Ctrl> + <Shift> + E`

- 새 탭 열기:

`<Ctrl> + <Shift> + T`
