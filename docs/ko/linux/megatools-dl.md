# megatools-dl

`mega.nz`에서 파일을 다운로드.
`megatools` 모음의 일부.
더 많은 정보: <https://megatools.megous.com/man/megatools-dl.html>.

- `mega.nz` 링크에서 현재 디렉터리로 파일 다운로드:

`megatools-dl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://mega.nz/...</span>

- `mega.nz` 링크에서 특정 디렉터리로 파일 다운로드:

`megatools-dl --path `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://mega.nz/...</span>

- 다운로드할 파일을 대화형으로 선택:

`megatools-dl --choose-files `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://mega.nz/...</span>

- 다운로드 속도를 KiB/s 단위로 제한:

`megatools-dl --limit-speed `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">속도</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://mega.nz/...</span>
