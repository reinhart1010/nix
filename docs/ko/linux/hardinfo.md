<hr># hardinfo

GUI 창에서 하드웨어 정보를 표시합니다.
더 많은 정보: <https://github.com/lpereira/hardinfo>.

- `hardinfo` 시작:

`hardinfo`

- 보고서를 `stdout`으로 출력:

`hardinfo -r`

- 보고서를 HTML 파일로 저장:

`hardinfo -r -f html > hardinfo.html`
