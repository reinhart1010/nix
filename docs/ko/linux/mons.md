# mons

두 디스플레이를 빠르게 관리.
더 많은 정보: <https://github.com/Ventto/mons>.

- 기본 모니터만 활성화:

`mons -o`

- 보조 모니터만 활성화:

`mons -s`

- 기본 모니터를 보조 모니터에 복제하고, 기본 모니터의 해상도 사용:

`mons -d`

- 기본 모니터를 보조 모니터에 미러링하고, 보조 모니터의 해상도 사용:

`mons -m`
