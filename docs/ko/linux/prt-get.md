# prt-get

CRUX 패키지 관리자.
더 많은 정보: <https://crux.nu/doc/prt-get%20-%20User%20Manual.html>.

- 패키지 설치:

`prt-get install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 의존성 처리를 포함하여 패키지 설치:

`prt-get depinst `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 패키지 수동 업데이트:

`prt-get upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 패키지 제거:

`prt-get remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 로컬 포트 트리에서 시스템 업그레이드:

`prt-get sysup`

- 포트 트리 검색:

`prt-get search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">쿼리</span>

- 패키지 내 파일 검색:

`prt-get fsearch `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">파일</span>
