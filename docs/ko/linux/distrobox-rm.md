# distrobox-rm

Distrobox 컨테이너 제거. 같이 보기: `tldr distrobox`.
더 많은 정보: <https://distrobox.it/usage/distrobox-rm>.

- Distrobox 컨테이너 제거 (팁: 제거하기 전에 컨테이너를 중지하세요):

`distrobox-rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">컨테이너_이름</span>

- Distrobox 컨테이너 강제 제거:

`distrobox-rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">컨테이너_이름</span>` --force`
