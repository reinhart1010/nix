# pkg

FreeBSD 패키지 관리자입니다.
더 많은 정보: <https://man.freebsd.org/cgi/man.cgi?pkg>.

- 새 패키지 설치:

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 패키지 삭제:

`pkg delete `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 모든 패키지 업그레이드:

`pkg upgrade`

- 패키지 검색:

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 설치된 패키지 목록:

`pkg info`

- 필요없는 의존성 제거:

`pkg autoremove`
