# cal

달력 정보를 출력합니다.
더 많은 정보: <https://keith.github.io/xcode-man-pages/cal.1.html>.

- 현재 달의 달력을 표시:

`cal`

- 이전 달, 현재 달, 다음 달 달력을 표시:

`cal -3`

- 특정 월(1-12 또는 이름)의 달력을 표시:

`cal -m `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">월</span>

- 현재 연도의 달력을 표시:

`cal -y`

- 특정 연도(4자리)의 달력을 표시:

`cal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">연도</span>

- 특정 월과 연도의 달력을 표시:

`cal `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">월</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">연도</span>

- 주어진 연도의 부활절 날짜(서방 기독교)를 표시:

`ncal -e `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">연도</span>
