# aiac

OpenAI를 사용하여 IaC 구성, 유틸리티, 쿼리 등을 생성합니다.
더 많은 정보: <https://github.com/gofireflyio/aiac>.

- Azure 스토리지 계정을 위한 Terraform 생성:

`aiac get terraform `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">Azure 스토리지 계정을 위한</span>

- nginx를 위한 Dockerfile 생성:

`aiac get dockerfile `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">보안을 강화한 nginx를 위한</span>

- Terraform을 적용하는 GitHub 액션 생성:

`aiac get github action `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">terraform을 계획하고 적용하는</span>

- Python으로 포트 스캐너 생성:

`aiac get python `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">네트워크의 모든 열린 포트를 스캔하는 코드</span>

- MongoDB 쿼리 생성:

`aiac get mongo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">생성 날짜로 모든 문서를 집계하는 쿼리</span>
