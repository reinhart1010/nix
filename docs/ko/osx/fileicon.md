# fileicon

사용자 지정 파일 및 폴더 아이콘 관리.
더 많은 정보: <https://github.com/mklement0/fileicon>.

- 특정 파일이나 폴더에 사용자 지정 아이콘 설정:

`fileicon set `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일_또는_폴더</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/아이콘_파일.png</span>

- 특정 파일이나 폴더에서 사용자 지정 아이콘 제거:

`fileicon rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일_또는_폴더</span>

- 파일이나 폴더의 사용자 지정 아이콘을 현재 디렉토리에 `.icns` 파일로 저장:

`fileicon get `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일_또는_폴더</span>

- 특정 파일이나 폴더에 사용자 지정 아이콘이 있는지 테스트:

`fileicon test `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일_또는_폴더</span>
