# dark-mode

macOS 다크 모드를 제어합니다.
더 많은 정보: <https://github.com/sindresorhus/dark-mode>.

- 다크 모드 전환 (현재 꺼져 있으면 켜고, 켜져 있으면 끄기):

`dark-mode`

- 다크 모드 켜기:

`dark-mode on`

- 다크 모드 끄기:

`dark-mode off`

- 다크 모드가 켜져 있는지 확인:

`dark-mode status`
