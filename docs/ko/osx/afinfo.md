# afinfo

OS X의 오디오 파일 메타데이터 파서.
OS X 내장 명령어.
더 많은 정보: <https://keith.github.io/xcode-man-pages/afinfo.1.html>.

- 주어진 오디오 파일의 정보 표시:

`afinfo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 오디오 파일의 간략한 설명 출력:

`afinfo --brief `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 오디오 파일의 InfoDictionary 메타데이터 정보 및 내용 출력:

`afinfo --info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- XML 형식으로 출력:

`afinfo --xml `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 오디오 파일의 경고 메시지 출력(있는 경우):

`afinfo --warnings `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 도움말 표시:

`afinfo --help`
