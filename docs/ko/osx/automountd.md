# automountd

`autofs`를 위한 자동 마운트/언마운트 데몬. `launchd`에 의해 필요 시 시작됩니다.
수동으로 실행하지 않아야 합니다.
더 많은 정보: <https://keith.github.io/xcode-man-pages/automountd.8.html>.

- 데몬 시작:

`automountd`

- `syslog`에 자세한 로그 기록:

`automountd -v`
