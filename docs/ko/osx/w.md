# w

로그인한 사용자와 그들이 무엇을 하고 있는지 표시합니다.
사용자 로그인, TTY, 원격 호스트, 로그인 시간, 유휴 시간, 현재 프로세스를 출력합니다.
더 많은 정보: <https://keith.github.io/xcode-man-pages/w.1.html>.

- 로그인한 사용자 정보 표시:

`w`

- 헤더 없이 로그인한 사용자 정보 표시:

`w -h`

- 유휴 시간별로 정렬된 로그인한 사용자 정보 표시:

`w -i`
