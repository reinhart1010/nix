# xed

Xcode에서 파일을 열어 편집합니다.
더 많은 정보: <https://keith.github.io/xcode-man-pages/xed.1.html>.

- Xcode에서 파일 열기:

`xed `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일1 경로/대상/파일2 ...</span>

- Xcode에서 파일을 열고, 없으면 생성:

`xed --create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일1 경로/대상/파일2 ...</span>

- Xcode에서 파일을 열고 75번째 줄로 이동:

`xed --line 75 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
