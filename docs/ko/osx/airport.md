# airport

무선 네트워크 구성 도구.
더 많은 정보: <https://keith.github.io/xcode-man-pages/airport.1.html>.

- 현재 무선 상태 정보 표시:

`airport --getinfo`

- 채널 1에서 무선 트래픽 스니핑:

`airport sniff `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1</span>

- 사용 가능한 무선 네트워크 검색:

`airport --scan`

- 현재 공항 네트워크에서 연결 해제:

`sudo airport --disassociate`
