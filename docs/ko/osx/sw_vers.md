# sw_vers

macOS 운영 체제 버전 정보를 출력합니다.
더 많은 정보: <https://keith.github.io/xcode-man-pages/sw_vers.1.html>.

- 사용 가능한 모든 정보 출력 (운영 체제 이름, 버전 번호, 빌드):

`sw_vers`

- 운영 체제의 버전 번호만 출력:

`sw_vers -productVersion`

- 빌드 식별자만 출력:

`sw_vers -buildVersion`
