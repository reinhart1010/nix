# arch

시스템 아키텍처 이름을 표시하거나 다른 아키텍처에서 명령을 실행합니다.
같이 보기: `uname`.
더 많은 정보: <https://keith.github.io/xcode-man-pages/arch.1.html>.

- 시스템의 아키텍처 표시:

`arch`

- x86_64를 사용하여 명령 실행:

`arch -x86_64 "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>`"`

- arm을 사용하여 명령 실행:

`arch -arm64 "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>`"`
