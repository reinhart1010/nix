# hcloud

Hetzner Cloud용 CLI를 사용하는 방법을 보여줌.
더 많은 정보: <https://github.com/hetznercloud/cli>.

- 사용 가능한 명령 및 플래그 표시:

`hcloud`

- 도움말 표시:

`hcloud -h`

- `hcloud` 컨텍스트에 사용 가능한 명령과 플래그를 표시:

`hcloud context`
