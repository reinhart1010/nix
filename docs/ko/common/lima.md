# lima

이 명령은 기본 VM 인스턴스에 대한 `limactl shell`의 별칭입니다.
다른 인스턴스에서 작업하려면 `$LIMA_INSTANCE` 환경 변수를 설정할 수도 있습니다.

- 원본 명령에 대한 문서 보기:

`tldr limactl`
