# p10k

powerlevel10k의 구성을 관리.
더 많은 정보: <https://github.com/romkatv/powerlevel10k>.

- powerlevel10k를 대화형으로 구성:

`p10k configure`

- powerlevel10k 다시 불러오기:

`p10k reload`

- 도움말 표시:

`p10k help`
