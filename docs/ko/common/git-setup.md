# git setup

디렉터리에 Git 저장소를 생성하고 모든 파일을 커밋.
`git-extras`의 일부.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-setup>.

- 현재 디렉터리에 Git 저장소를 생성하고 모든 파일을 커밋:

`git setup`
