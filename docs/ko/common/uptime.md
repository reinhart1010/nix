# uptime

시스템이 실행된 기간 및 기타 정보.
더 많은 정보: <https://www.gnu.org/software/coreutils/manual/html_node/uptime-invocation.html>.

- 현재 시간, 가동 시간, 로그인한 사용자 수 및 기타 정보를 출력:

`uptime`

- 시스템이 부팅된 시간만 표시:

`uptime --pretty`

- 시스템이 부팅된 날짜와 시간을 출력:

`uptime --since`

- 버전 정보 표시:

`uptime --version`
