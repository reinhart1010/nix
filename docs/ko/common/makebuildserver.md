# makebuildserver

F-Droid 빌드 서버 가상 머신 생성.
더 많은 정보: <https://f-droid.org/en/docs/Build_Server_Setup/>.

- 새 가상 머신을 생성하거나 기존 머신이 있을 경우 업데이트:

`makebuildserver`

- 새로운 가상 머신을 강제로 생성:

`makebuildserver --clean`
