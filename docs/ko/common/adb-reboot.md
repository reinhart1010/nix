# adb reboot

연결된 Android 기기 또는 에뮬레이터를 재부팅.
더 많은 정보: <https://manned.org/adb>.

- 장치를 정상적으로 재부팅:

`adb reboot`

- 장치를 부트로더 모드로 재부팅:

`adb reboot bootloader`

- 장치를 복구 모드로 재부팅:

`adb reboot recovery`

- 장치를 빠른 부팅 모드로 재부팅:

`adb reboot fastboot`
