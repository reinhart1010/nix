# neofetch

운영 체제, 소프트웨어 및 하드웨어에 대한 정보를 표시.
더 많은 정보: <https://github.com/dylanaraps/neofetch>.

- 기본 설정을 반환하고, 프로그램이 처음 실행되는 경우 설정 생성:

`neofetch`

- 출력에서 특정 정보 라인을 표시하거나 숨김, 여기서 'infoname'은 설정 파일의 함수 이름(예: 메모리):

`neofetch --`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">enable|disable</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">정보명</span>

- 운영 체제 아키텍처 숨기기/표시:

`neofetch --os_arch `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">on|off</span>

- 출력에서 CPU 브랜드 활성화/비활성화:

`neofetch --cpu_brand `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">on|off</span>
