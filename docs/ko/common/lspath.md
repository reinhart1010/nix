# lspath

`PATH` 환경 변수의 내용을 나열하고, 선택적으로 페이지를 나눠서 표시합니다.
더 많은 정보: <https://github.com/clementi/lspath>.

- 시스템 `PATH` 변수의 내용을 한 줄에 하나씩 출력:

`lspath`

- 현재 시스템 `PATH` 변수의 내용을 한 줄에 하나씩 출력하고, 페이지로 나눠서 표시:

`lspath --page`
