# frp

고속 역방향 프록시: 특정 서비스를 인터넷이나 기타 외부 네트워크에 노출하도록 네트워크 터널을 빠르게 설정.
더 많은 정보: <https://github.com/fatedier/frp>.

- `frp` 클라이언트 구성요소인 `frpc`에 대한 문서 보기:

`tldr frpc`

- `frp` 서버 구성요소인 `frps`에 대한 문서 보기:

`tldr frps`
