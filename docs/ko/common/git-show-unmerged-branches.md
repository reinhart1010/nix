# git show-unmerged-branches

현재 HEAD에 병합되지 않은 모든 브랜치를 출력.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-show-unmerged-branches>.

- 현재 HEAD에 병합되지 않은 모든 브랜치를 출력:

`git show-unmerged-branches`
