# iex

IEx는 Elixir의 대화형 셸.
더 많은 정보: <https://hexdocs.pm/iex>.

- 대화형 세션을 시작:

`iex`

- 역사를 기억하는 세션을 시작:

`iex --erl "-kernel shell_history enabled"`

- Mix 프로젝트 파일 시작 및 로드:

`iex -S mix`
