# jadx

Dex를 Java로 디컴파일하는 도구.
Android Dex 및 APK 파일에서 Java 소스 코드를 생성합니다.
더 많은 정보: <https://github.com/skylot/jadx>.

- Dex 파일을 디렉터리로 디컴파일:

`jadx `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- Dex 파일을 특정 디렉터리로 디컴파일:

`jadx --output-dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
