<hr># gtop

터미널의 시스템 모니터링 대시보드.
더 많은 정보: <https://github.com/aksakalli/gtop>.

- 시스템 통계 대시보드:

`gtop`

- CPU 사용량을 기준으로 정렬:

`c`

- 메모리 사용량을 기준으로 정렬:

`m`
