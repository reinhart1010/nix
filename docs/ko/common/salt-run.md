# salt-run

minion에서 salt-runners를 실행하기 위한 프론트엔드.
더 많은 정보: <https://docs.saltproject.io/en/latest/ref/cli/salt-run.html>.

- 모든 minion의 상태 표시:

`salt-run manage.status`

- 연결이 끊긴 모든 minion 표시:

`salt-run manage.up`
