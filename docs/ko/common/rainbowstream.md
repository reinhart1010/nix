# rainbowstream

터미널 기반의 Twitter 클라이언트로, 실시간 트윗 스트림, 트렌드, 전송, 검색, 즐겨찾기 및 사용자 관리를 지원합니다.
`h`로 온라인 도움말, 위아래 화살표로 히스토리, 탭으로 자동 완성, 2번 탭으로 제안 기능을 사용할 수 있습니다.
더 많은 정보: <https://github.com/orakaro/rainbowstream>.

- RainbowStream 열기:

`rainbowstream`

- 타임라인 보기 (표시할 트윗 수는 선택 사항이며 기본값은 5개):

`home [`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">최근_트윗_수</span>`]`

- 특정 사용자의 프로필 보기:

`whois @`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">사용자</span>

- 메시지를 그대로 트윗하기:

`t `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">메시지</span>

- 주어진 ID의 트윗을 리트윗하기 (ID는 시간 옆에 표시됨):

`rt `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">트윗_ID</span>

- 주어진 ID의 트윗을 즐겨찾기에 추가하기:

`fav `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">트윗_ID</span>

- 특정 단어 검색하기 (해시태그 포함 또는 미포함):

`s `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">단어</span>
