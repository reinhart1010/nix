# vulkaninfo

시스템의 Vulkan 정보를 출력.
더 많은 정보: <https://vulkan.lunarg.com/doc/view/latest/linux/vulkaninfo.html>.

- 전체 Vulkan 정보 출력:

`vulkaninfo`

- 요약 정보 출력:

`vulkaninfo --summary`

- 전체 Vulkan 정보를 HTML 문서로 생성:

`vulkaninfo --html`
