# nest

Nest 애플리케이션을 초기화하고 개발하며 유지 관리.
더 많은 정보: <https://docs.nestjs.com/cli/overview>.

- 설치된 Nest 버전에 대한 정보 표시:

`nest info`

- 동일한 이름의 디렉터리에 새 NestJS 프로젝트 생성:

`nest new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 특정 NestJS 프로젝트 빌드:

`nest build `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 특정 NestJS 프로젝트 실행:

`nest start `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 현재 NestJS 프로젝트에 라이브러리 추가:

`nest add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리_이름</span>
