<hr># pio lib

PlatformIO 라이브러리 관리.
더 많은 정보: <https://docs.platformio.org/en/latest/core/userguide/lib/>.

- 설치된 라이브러리 나열:

`pio lib list`

- 설치된 개발 플랫폼과 그 프레임워크에 기반한 내장 라이브러리 나열:

`pio lib builtin`

- 기존 라이브러리 검색:

`pio lib search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 라이브러리에 대한 세부 정보 표시:

`pio lib show `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리</span>

- 라이브러리 설치:

`pio lib install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리</span>

- 설치된 라이브러리 업데이트:

`pio lib update`

- 라이브러리 제거:

`pio lib uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리</span>

- PlatformIO 라이브러리 레지스트리 통계 표시:

`pio lib stats`
