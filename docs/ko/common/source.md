# source

현재 셸에서 파일의 명령을 실행.
더 많은 정보: <https://manned.org/source>.

- 주어진 파일의 내용을 평가:

`source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 주어진 파일의 내용을 평가 (`source` 대신 `.` 사용 가능):

`. `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
