# maza

로컬 광고 차단기. Pi-hole과 유사하지만 로컬에서 운영 체제를 사용합니다.
더 많은 정보: <https://github.com/tanrax/maza-ad-blocking>.

- Maza 데이터베이스 업데이트:

`maza update`

- Maza 시작:

`sudo maza start`

- Maza 중지:

`sudo maza stop`

- Maza 상태 표시:

`maza status`
