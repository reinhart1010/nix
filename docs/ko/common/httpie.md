# httpie

HTTPie용 관리 인터페이스.
참조: `http`, 도구 자체.
더 많은 정보: <https://httpie.io/docs/cli/plugin-manager>.

- `http` 업데이트를 확인:

`httpie cli check-updates`

- 설치된 `http` 플러그인 목록 나열:

`httpie cli plugins list`
