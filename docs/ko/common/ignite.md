# ignite

React Native 보일러플레이트, 플러그인, 생성기 등을 위한 CLI.
더 많은 정보: <https://infinite.red/ignite>.

- 새로운 React Native 프로젝트를 생성:

`ignite new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 플러그인에서 파일 생성:

`ignite generate `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플러그인_이름</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 프로젝트에 Ignite 플러그인을 추가:

`ignite add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플러그인_이름</span>

- Remove an Ignite plugin from the project:

`ignite remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">플러그인_이름</span>
