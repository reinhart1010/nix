# whoami

현재 유효 사용자 ID와 연관된 사용자 명을 출력.
더 많은 정보: <https://www.gnu.org/software/coreutils/manual/html_node/whoami-invocation.html>.

- 현재 로그인된 사용자 명 표시:

`whoami`

- 사용자 ID 변경 후 사용자 명 표시:

`sudo whoami`
