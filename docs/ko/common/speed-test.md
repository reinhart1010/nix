# speed-test

<https://speedtest.net>를 사용하여 인터넷 연결 속도와 핑을 테스트합니다.
더 많은 정보: <https://github.com/sindresorhus/speed-test>.

- 인터넷 연결 속도와 핑 테스트:

`speed-test`

- 결과를 JSON으로 출력:

`speed-test --json`

- 결과를 초당 메가바이트(MBps)로 출력:

`speed-test --bytes`

- 더 자세한 정보 출력:

`speed-test --verbose`
