# moe

ISO-8859-15로 인코딩된 텍스트를 위한 WYSIWYG 텍스트 편집기.
더 많은 정보: <https://www.gnu.org/software/moe/moe.html>.

- moe를 열고 편집 내용을 저장할 때 백업 파일(파일~) 생성:

`moe `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 파일을 읽기 전용으로 열기:

`moe --read-only `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 백업 파일을 생성하지 않고 파일 편집:

`moe --no-backup `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 검색 시 대소문자 구분 없이 파일 편집:

`moe --ignore-case `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 저장하고 종료:

`<Ctrl> + X`
