# cheat

대화형 치트 시트를 생성하고 봅니다.
더 많은 정보: <https://github.com/cheat/cheat>.

- 명령어 사용법의 예를 보여줌:

`cheat `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령어</span>

- 명령어에 대한 치트 시트를 편집:

`cheat -e `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령어</span>

- 사용 가능한 치트 시트 목록:

`cheat -l`

- 지정된 명령어 이름에 대한 치트 시트를 검색할 수 있음:

`cheat -s `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령어</span>

- 버전 정보 표시:

`cheat -v`
