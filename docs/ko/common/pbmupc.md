# pbmupc

범용 상품 코드(UPC)의 PBM 이미지를 생성합니다.
더 많은 정보: <https://netpbm.sourceforge.net/doc/pbmupc.html>.

- 지정된 상품 유형, 제조사 코드 및 상품 코드를 위한 UPC 이미지 생성:
