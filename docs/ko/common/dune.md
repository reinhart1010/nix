# dune

OCaml 프로그램용 빌드 시스템.
더 많은 정보: <https://dune.build>.

- 모든 타겟 빌드:

`dune build`

- 작업 공간을 정리:

`dune clean`

- 모든 테스트 실행:

`dune runtest`

- 수동으로 로드할 필요가 없도록, 자동으로 로드된 컴파일된 모듈로 utop REPL을 시작:

`dune utop`
