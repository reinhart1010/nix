# pbmtopgm

PBM 이미지를 개별 픽셀 주변 영역의 평균을 내어 PGM으로 변환.
같이 보기: `pnmconvol`, `pamditherbw`.
더 많은 정보: <https://netpbm.sourceforge.net/doc/pbmtopgm.html>.

- 각 픽셀 주변의 `w`x`h` 크기 영역을 평균 내어 PBM 이미지를 PGM으로 변환:
