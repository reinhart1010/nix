# ppmtopgm

PPM 이미지를 PGM 이미지로 변환.
더 많은 정보: <https://netpbm.sourceforge.net/doc/ppmtopgm.html>.

- PPM 이미지를 PGM 이미지로 변환:

`ppmtopgm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.ppm</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.pgm</span>

- 버전 표시:

`ppmtopgm -version`
