# twm

X Window 시스템을 위한 윈도우 매니저.
더 많은 정보: <https://gitlab.freedesktop.org/xorg/app/twm>.

- 기본 X 서버에 연결:

`twm`

- 특정 X 서버에 연결:

`twm -display `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">디스플레이</span>

- 기본 화면만 관리:

`twm -s`

- 특정 시작 파일 사용:

`twm -f `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 자세한 모드 활성화 및 X에서 예상치 못한 오류 출력:

`twm -v`
