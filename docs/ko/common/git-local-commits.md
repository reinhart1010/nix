<hr># git local-commits

원격 저장소에 푸시되지 않은 로컬 커밋을 표시. 추가 인수는 `git log`에 직접 전달됨.
`git-extras`의 일부.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-local-commits>.

- 푸시되지 않은 커밋 표시:

`git local-commits`
