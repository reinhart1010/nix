<hr># mkdocs

Markdown으로 프로젝트 문서화.
더 많은 정보: <https://www.mkdocs.org>.

- 새 mkdocs 프로젝트 생성:

`mkdocs new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- Mkdocs 개발 서버를 사용하여 현재 디렉토리에서 프로젝트 제공:

`mkdocs serve`

- 현재 디렉토리에서 문서 빌드:

`mkdocs build`

- 현재 디렉토리의 문서를 GitHub 페이지에 배포:

`mkdocs gh-deploy`
