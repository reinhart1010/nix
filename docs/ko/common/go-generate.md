# go generate

소스 파일 내에서 명령을 실행하여 Go 파일 생성.
더 많은 정보: <https://golang.org/cmd/go/#hdr-Generate_Go_files_by_processing_source>.

- 소스 파일 내에서 명령을 실행하여 Go 파일 생성:

`go generate`
