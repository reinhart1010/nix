# thunderbird

이메일 클라이언트 및 RSS 리더.
더 많은 정보: <https://thunderbird.net>.

- thunderbird 열기:

`thunderbird`

- 특정 사용자 프로필 사용:

`thunderbird -P `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로필_이름</span>

- 특정 사용자 프로필 디렉토리 사용:

`thunderbird --profile `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/프로필/디렉토리</span>
