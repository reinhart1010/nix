# texdoc

(La)TeX 명령어 또는 패키지에 대한 적절한 문서를 검색.
더 많은 정보: <https://texdoc.org/index.html>.

- 첫 번째 검색 결과를 기본 PDF 뷰어로 열기:

`texdoc `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">검색어</span>

- 가장 적합한 검색 결과 나열:

`texdoc --list `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">검색어</span>

- texdoc의 전체 문서 열기:

`texdoc `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">texdoc</span>
