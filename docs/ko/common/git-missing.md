# git missing

두 브랜치 간에 공유되지 않은 커밋을 표시.
`git-extras`의 일부.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-missing>.

- 현재 체크아웃된 브랜치와 다른 브랜치 간에 공유되지 않은 커밋 표시:

`git missing `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">브랜치</span>
