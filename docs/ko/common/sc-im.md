# SC-IM

커서 기반의 vim 스타일 스프레드시트 계산기입니다.
이동하려면 hjkl 또는 화살표 키를 사용하세요.
더 많은 정보: <https://github.com/andmarti1424/sc-im>.

- SC-IM 시작:

`scim `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.csv</span>

- 현재 셀에 문자열 입력:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold"><|></span>

- 현재 셀에 숫자 상수 입력:

`=`

- 현재 셀의 문자열 편집:

`E`

- 현재 셀의 숫자 편집:

`e`

- 현재 셀 가운데 정렬:

`|`
