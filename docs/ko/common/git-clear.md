# git clear

현재 브랜치와 `.gitignore`에 포함된 파일들을 포함하여 Git 작업 디렉토리를 새로 클론한 것처럼 초기화합니다.
`git-extras`의 일부입니다.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-clear>.

- 모든 추적된 파일을 초기화하고 `.gitignore`에 포함된 파일을 포함하여 모든 추적되지 않은 파일 삭제:

`git clear`
