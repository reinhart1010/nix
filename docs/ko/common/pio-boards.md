# pio boards

PlatformIO에서 사용할 수 있는 사전 구성된 임베디드 보드 나열.
더 많은 정보: <https://docs.platformio.org/en/latest/core/userguide/cmd_boards.html>.

- 사용 가능한 모든 보드 나열:

`pio boards`

- 설치된 플랫폼의 보드만 나열:

`pio boards --installed`
