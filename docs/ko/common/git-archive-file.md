# git archive-file

현재 Git 브랜치의 모든 파일을 Zip 아카이브로 내보냅니다.
`git-extras`의 일부입니다.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-archive-file>.

- 현재 체크아웃된 커밋을 Zip 아카이브로 패킹:

`git archive-file`
