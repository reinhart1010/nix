<hr># flask

Flask 애플리케이션을 위한 일반 유틸리티 스크립트. `FLASK_APP` 환경 변수에 정의된 애플리케이션을 로드.
더 많은 정보: <https://flask.palletsprojects.com/en/1.1.x/cli/>.

- 개발 서버 실행:

`flask run`

- 앱의 경로 표시:

`flask routes`

- 앱의 컨텍스트에서 Python 대화형 쉘을 실행:

`flask shell`
