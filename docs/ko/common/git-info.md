# git info

Git 저장소 정보 표시.
`git-extras`의 일부.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-info>.

- 원격 위치, 원격 및 로컬 브랜치, 최근 커밋 데이터 및 `.git/config` 설정 표시:

`git info`

- 원격 위치, 원격 및 로컬 브랜치, 최근 커밋 데이터만 표시:

`git info --no-config`
