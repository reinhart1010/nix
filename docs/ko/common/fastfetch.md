# fastfetch

운영체제, 소프트웨어 및 하드웨어에 대한 정보 표시.
더 많은 정보: <https://github.com/LinusDierheimer/fastfetch>.

- 시스템 정보 표시:

`fastfetch`

- 로고와 이스케이프 시퀀스 없이 시스템 정보를 표기:

`fastfetch --pipe`

- 특정 구조를 가져오기:

`fastfetch --structure `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">구조</span>

- 사용자 정의 구성 파일 로드:

`fastfetch --load-config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/구성_파일</span>

- 특정 로고 사용:

`fastfetch --logo `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">로고</span>

- 키와 제목에 특정 색상을 사용:

`fastfetch --color `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">파랑</span>
