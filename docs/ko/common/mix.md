<hr># mix

Elixir 프로젝트를 생성, 컴파일 및 테스트하고, 의존성을 관리하는 작업을 제공하는 빌드 도구.
더 많은 정보: <https://hexdocs.pm/mix>.

- 특정 파일 실행:

`mix run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">나의_스크립트.exs</span>

- 새 프로젝트 생성:

`mix new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 프로젝트 컴파일:

`mix compile`

- 프로젝트 테스트 실행:

`mix test`

- 도움말 표시:

`mix help`
