<hr># meteor

웹 애플리케이션을 구축하기 위한 풀스택 JavaScript 플랫폼.
더 많은 정보: <https://meteor.com>.

- 루트 디렉터리에서 개발 모드로 meteor 프로젝트 실행:

`meteor`

- 지정된 디렉터리에 프로젝트 생성:

`meteor create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>

- 프로젝트에서 현재 사용 중인 패키지 목록 표시:

`meteor list`

- 프로젝트에 패키지 추가:

`meteor add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 프로젝트에서 패키지 제거:

`meteor remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 지정된 디렉터리에 tarball 형식의 프로젝트 배포용 빌드 생성:

`meteor build `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>
