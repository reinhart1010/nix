# yadm-bootstrap

Yadm의 부트스트랩 파일 실행.
이 파일은 `$HOME/.config/yadm/bootstrap`에 생성되어야 합니다.
더 많은 정보: <https://yadm.io/docs/bootstrap>.

- 부트스트랩 실행 파일 실행:

`yadm bootstrap`
