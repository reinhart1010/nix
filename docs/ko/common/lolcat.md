# lolcat

모든 `cat` 결과를 콘솔에 무지개 색상으로 출력합니다.
더 많은 정보: <https://github.com/busyloop/lolcat>.

- 파일을 콘솔에 무지개 색상으로 출력:

`lolcat `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 텍스트를 생성하는 명령의 결과를 무지개 색상으로 출력:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fortune</span>` | lolcat`

- 파일을 콘솔에 애니메이션 무지개 색상으로 출력:

`lolcat -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 파일을 콘솔에 24비트 (진정한 색상) 무지개 색상으로 출력:

`lolcat -t `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
