<hr># psysh

PHP용 런타임 개발자 콘솔, 대화형 디버거 및 REPL.
더 많은 정보: <https://psysh.org>.

- 현재 디렉토리에서 셸 열기:

`psysh`

- 특정 디렉토리에서 셸 열기:

`psysh --cwd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>

- 특정 구성 파일 사용:

`psysh --config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
