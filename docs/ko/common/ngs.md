# ngs

Ops를 위해 특별히 만들어진 스크립트 언어.
더 많은 정보: <https://github.com/ngs-lang/ngs>.

- 코드 스니펫 실행:

`ngs -e "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">echo('ngs is executed')</span>`"`

- 스크립트 실행:

`ngs `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/스크립트.ngs</span>

- 버전 표시:

`ngs --version`
