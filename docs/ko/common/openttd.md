# openttd

Microprose 게임 "Transport Tycoon Deluxe"의 오픈 소스 클론.
더 많은 정보: <https://www.openttd.org>.

- 새 게임 시작:

`openttd -g`

- 시작 시 저장된 게임 불러오기:

`openttd -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 지정된 창 해상도로 시작:

`openttd -r `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1920x1080</span>

- 사용자 정의 구성 파일로 시작:

`openttd -c `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 선택된 비디오, 사운드, 음악 드라이버로 시작:

`openttd -v `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">비디오_드라이버</span>` -s `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">사운드_드라이버</span>` -m `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">음악_드라이버</span>

- 백그라운드에서 포크된 전용 서버 시작:

`openttd -f -D `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">호스트</span>`:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">포트</span>
