<hr># tlmgr path

TeX Live 실행 파일, 매뉴얼 페이지 및 정보 페이지에 대한 심볼릭 링크 추가 또는 제거.
향후 추가된 파일에 대해서는 이 명령을 다시 실행해야 합니다.
더 많은 정보: <https://www.tug.org/texlive/tlmgr.html>.

- TeX Live 파일에 심볼릭 링크 추가:

`sudo tlmgr path add`

- TeX Live 파일에 대한 심볼릭 링크 제거:

`sudo tlmgr path remove`
