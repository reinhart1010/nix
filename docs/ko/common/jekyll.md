# jekyll

간단하고 블로그 친화적인 정적 사이트 생성기.
더 많은 정보: <https://jekyllrb.com/docs/usage/>.

- http://localhost:4000/ 에서 실행되는 개발 서버 생성:

`jekyll serve`

- 증분 재생성 활성화:

`jekyll serve --incremental`

- 자세한 출력 활성화:

`jekyll serve --verbose`

- 현재 디렉토리를 `./_site`로 생성:

`jekyll build`

- 사이트 정리 (사이트 출력 및 `cache` 디렉토리를 제거) 빌드 없이:

`jekyll clean`
