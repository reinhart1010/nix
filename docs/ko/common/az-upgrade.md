# az upgrade

Azure CLI 및 확장을 업그레이드.
`azure-cli`의 일부 (`az`라고도 함).
더 많은 정보: <https://learn.microsoft.com/cli/azure/reference-index?view=azure-cli-latest#az-upgrade>.

- Azure CLI 업그레이드:

`az upgrade`

- Azure CLI 및 확장을 업그레이드:

`az upgrade --all`

- 확인 메시지를 표시하지 않고 Azure CLI 및 확장을 업그레이드:

`az version --all --yes`
