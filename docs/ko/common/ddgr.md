# ddgr

터미널에서 DuckDuckGo (HTML 버전)를 검색.
더 많은 정보: <https://github.com/jarun/ddgr>.

- 대화형 모드 시작:

`ddgr`

- DuckDuckGo에서 키워드를 검색:

`ddgr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 검색 결과 수를 `N`으로 제한:

`ddgr -n `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">N</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 검색결과에 전체 URL을 표시:

`ddgr -x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 키워드로 DuckDuckGo를 검색하고, 브라우저에 첫 번째 검색결과를 보여줌:

`ddgr !w `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 웹사이트별 검색을 수행:

`ddgr -w `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">사이트</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 특정 파일 유형 검색:

`ddgr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>` filetype:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">파일유형</span>
