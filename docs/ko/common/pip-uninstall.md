<hr># pip uninstall

Python 패키지 제거.
더 많은 정보: <https://pip.pypa.io>.

- 패키지 제거:

`pip uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 특정 파일에 나열된 패키지 제거:

`pip uninstall --requirement `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/requirements.txt</span>

- 확인 요청 없이 패키지 제거:

`pip uninstall --yes `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>
