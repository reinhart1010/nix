# runsv

runit 서비스를 시작 및 관리.
더 많은 정보: <https://manned.org/runsv.8>.

- 현재 사용자로 runit 서비스 시작:

`runsv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/서비스</span>

- root로 runit 서비스 시작:

`sudo runsv `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/서비스</span>
