# st-info

연결된 STLink 및 STM32 장치에 대한 정보를 가져옵니다.
더 많은 정보: <https://github.com/texane/stlink>.

- 사용 가능한 프로그램 메모리 용량 표시:

`st-info --flash`

- 사용 가능한 SRAM 메모리 용량 표시:

`st-info --sram`

- 장치의 요약 정보 표시:

`st-info --probe`
