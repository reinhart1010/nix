<hr># jobs

현재 세션의 작업 상태 표시.
더 많은 정보: <https://manned.org/jobs>.

- 모든 작업의 상태 표시:

`jobs`

- 특정 작업의 상태 표시:

`jobs %`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">작업_ID</span>

- 모든 작업의 상태 및 프로세스 ID 표시:

`jobs -l`

- 모든 작업의 프로세스 ID 표시:

`jobs -p`
