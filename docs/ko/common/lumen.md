# lumen

Lumen 마이크로 프레임워크를 위한 명령줄 설치 도구.
더 많은 정보: <https://lumen.laravel.com>.

- 새로운 Lumen 애플리케이션 생성:

`lumen new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">애플리케이션_이름</span>

- 사용 가능한 설치 명령 나열:

`lumen list`
