# joe

Joe의 자체 텍스트 편집기.
더 많은 정보: <https://joe-editor.sourceforge.io>.

- JOE에서 새 파일 열기:

`joe`

- 특정 파일 열기:

`joe `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 특정 파일을 열고 지정된 줄에 커서 위치시키기:

`joe +`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">줄</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 특정 파일을 읽기 전용 모드로 열기:

`joe -rdonly `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
