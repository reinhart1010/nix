<hr># streamlit

Python에서 상호작용하는 데이터 기반 웹 앱을 만드는 프레임워크.
더 많은 정보: <https://docs.streamlit.io/>.

- Streamlit 설치 여부 확인:

`streamlit hello`

- Streamlit 애플리케이션 실행:

`streamlit run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 도움말 표시:

`streamlit --help`

- 버전 표시:

`streamlit --version`
