# npm home

웹 브라우저에서 패키지의 npm 페이지, Yarn 페이지 또는 GitHub 저장소를 엽니다.
더 많은 정보: <https://github.com/sindresorhus/npm-home>.

- 특정 패키지의 `npm` 페이지를 웹 브라우저에서 열기:

`npm-home `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 특정 패키지의 GitHub 저장소를 웹 브라우저에서 열기:

`npm-home -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 특정 패키지의 Yarn 페이지를 웹 브라우저에서 열기:

`npm-home -y `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>
