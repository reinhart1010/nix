# hg branch

브랜치 이름을 생성하거나 표시.
더 많은 정보: <https://www.mercurial-scm.org/doc/hg.1.html#branch>.

- 현재 활성 브랜치의 이름 표시:

`hg branch`

- 다음 커밋을 위한 새 브랜치 생성:

`hg branch `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">브랜치_이름</span>
