# brew autoremove

이전에 종속성으로 설치되어 사용되지 않는 수식을 제거.
더 많은 정보: <https://docs.brew.sh/Manpage#autoremove---dry-run>.

- 사용하지 않는 수식을 모두 제거:

`brew autoremove`

- 제거할 항목을 인쇄하지만, 실제로 아무것도 제거하지 않음:

`brew autoremove --dry-run`
