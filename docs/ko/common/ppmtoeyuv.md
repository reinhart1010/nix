# ppmtoeyuv

PPM 이미지를 Berkeley YUV 파일로 변환.
더 많은 정보: <https://netpbm.sourceforge.net/doc/ppmtoeyuv.html>.

- 입력 파일에서 PPM 이미지를 읽고 Berkeley YUV 이미지로 변환하여 지정된 출력 파일에 저장:
