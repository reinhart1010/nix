# calendar

캘린더 파일에서 예정된 이벤트를 표시.
더 많은 정보: <https://manned.org/calendar>.

- 기본 캘린더에서 오늘과 내일 (또는 주말, 금요일마다) 이벤트를 표시:

`calendar`

- 앞으로 30일 동안의 이벤트를 표시하는 미리([A]head) 보기:

`calendar -A `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">30</span>

- 지난 7일 동안의 이벤트를 표시하는 되돌아([B]ack) 보기:

`calendar -B `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">7</span>

- 사용자 정의 캘린더 파일([f]ile)의 이벤트 표시:

`calendar -f `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
