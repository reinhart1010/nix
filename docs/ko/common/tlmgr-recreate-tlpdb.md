# tlmgr recreate-tlpdb

TeX Live 패키지 데이터베이스를 재생성.
이 명령은 문서가 부족하므로 주의해서 사용해야 함.
더 많은 정보: <https://www.tug.org/texlive/tlmgr.html>.

- `texlive.tlpdb` 데이터베이스 파일을 재생성하고 `stdout`에 덤프:

`sudo tlmgr recreate-tlpdb`
