<hr># w

현재 로그인된 사용자와 그들의 작업을 보여줍니다.
사용자 로그인, TTY, 원격 호스트, 로그인 시간, 유휴 시간, 현재 프로세스를 출력합니다.
더 많은 정보: <https://manned.org/w>.

- 로그인된 사용자 정보 표시:

`w`

- 헤더 없이 로그인된 사용자 정보 표시:

`w -h`
