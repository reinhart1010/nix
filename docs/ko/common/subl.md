<hr># subl

Sublime Text 편집기.
더 많은 정보: <https://www.sublimetext.com>.

- 현재 디렉토리를 Sublime Text에서 열기:

`subl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">.</span>

- 파일 또는 디렉토리를 Sublime Text에서 열기:

`subl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일_또는_폴더</span>

- 파일을 열고 특정 줄 번호로 이동:

`subl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>`:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">줄_번호</span>

- 현재 열린 창에서 파일 또는 디렉토리 열기:

`subl -a `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 새 창에서 파일 또는 디렉토리 열기:

`subl -n `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>
