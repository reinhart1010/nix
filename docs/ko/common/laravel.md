# laravel

Laravel 프레임워크를 위한 명령줄 설치 도구.
더 많은 정보: <https://laravel.com>.

- 새 Laravel 애플리케이션 생성:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>

- 최신 개발 릴리스 사용:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>` --dev`

- 디렉터리가 이미 존재하는 경우 덮어쓰기:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>` --force`

- Laravel Jetstream 스캐폴딩 설치:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>` --jet`

- 특정 스택을 사용하여 Laravel Jetstream 스캐폴딩 설치:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>` --jet --stack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">livewire|inertia</span>

- 팀 지원과 함께 Laravel Jetstream 스캐폴딩 설치:

`laravel new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>` --jet --teams`

- 사용 가능한 설치 도구 명령 나열:

`laravel list`
