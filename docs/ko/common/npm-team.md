# npm team

`npm` 레지스트리에서 조직 내 팀을 관리.
더 많은 정보: <https://docs.npmjs.com/cli/commands/npm-team>.

- 조직 내 팀에 사용자 추가:

`npm team add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직:팀</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">사용자_명</span>

- 팀에서 사용자 제거:

`npm team rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직:팀</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">사용자_명</span>

- 조직 내 새 팀 생성:

`npm team create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직:팀</span>

- 조직에서 팀 삭제:

`npm team destroy `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직:팀</span>

- 조직 내 모든 팀 나열:

`npm team ls `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직</span>

- 특정 팀의 모든 사용자 나열:

`npm team ls `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">조직:팀</span>
