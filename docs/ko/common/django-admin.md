# django-admin

관리 작업을 위한 Django 유틸리티.
더 많은 정보: <https://docs.djangoproject.com/en/4.1/ref/django-admin/>.

- 새로운 Django 프로젝트를 생성:

`django-admin startproject `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 현재 프로젝트 내부에 새로운 애플리케이션 생성:

`django-admin startapp `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">애플리케이션_이름</span>

- 현재 설치된 Django 버전 확인:

`django-admin --version`

- 입력한 명령어에 대해 더 많은 정보 표시:

`django-admin help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령어</span>
