# yadm-list

`yadm`이 관리하는 파일 목록 출력.
더 많은 정보: <https://github.com/TheLocehiliosan/yadm/blob/master/yadm.md#commands>.

- 현재 디렉토리에서 `yadm`이 관리하는 파일 목록 출력:

`yadm list`

- `yadm`이 완전히 관리하는 모든 파일 목록 출력:

`yadm list -a`
