# docker start

중지된 컨테이너 시작.
더 많은 정보: <https://docs.docker.com/reference/cli/docker/container/start/>.

- 도움말 표시:

`docker start`

- Docker 컨테이너 시작:

`docker start `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">컨테이너</span>

- `stdout` 및 `stderr`를 연결하고 신호를 전달하여 컨테이너 시작:

`docker start --attach `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">컨테이너</span>

- 하나 이상의 컨테이너 시작:

`docker start `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">컨테이너1 컨테이너2 ...</span>
