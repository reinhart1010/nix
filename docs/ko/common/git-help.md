# git help

Git에 대한 도움말 정보를 표시합니다.
더 많은 정보: <https://git-scm.com/docs/git-help>.

- 특정 Git 하위 명령에 대한 도움말 표시:

`git help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위_명령</span>

- 웹 브라우저에서 특정 Git 하위 명령에 대한 도움말 표시:

`git help --web `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위_명령</span>

- 사용 가능한 모든 Git 하위 명령의 목록 표시:

`git help --all`

- 사용 가능한 가이드 목록 표시:

`git help --guide`

- 모든 가능한 설정 변수 나열:

`git help --config`
