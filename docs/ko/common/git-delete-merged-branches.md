<hr># git delete-merged-branches

`git branch --merged`에 나열된 master를 제외한 브랜치를 삭제합니다.
`git-extras`의 일부입니다.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-delete-merged-branches>.

- 병합된 브랜치 삭제:

`git delete-merged-branches`
