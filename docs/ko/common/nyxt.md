<hr># nyxt

키보드 기반으로 작동하는 파워 유저를 위한 웹 브라우저.
더 많은 정보: <https://nyxt.atlas.engineer>.

- 모든 프로필 나열:

`nyxt --list-data-profiles`

- `init.lisp` 파일 경로 설정:

`nyxt --init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 자동 구성 파일 경로 변경:

`nyxt --auto-config `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 시스템 정보 출력:

`nyxt --system-information`
