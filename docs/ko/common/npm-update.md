<hr># npm update

현재 프로젝트의 패키지를 업데이트.
더 많은 정보: <https://docs.npmjs.com/cli/commands/npm-update>.

- 현재 프로젝트의 모든 패키지 업데이트:

`npm update`

- 현재 프로젝트의 특정 패키지 업데이트:

`npm update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 전역으로 패키지 업데이트:

`npm update -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 여러 패키지를 한 번에 업데이트:

`npm update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지1 패키지2 ...</span>
