# elvish

표현력이 풍부한 프로그래밍 언어와 다양한 대화형 쉘.
참고: `nu`.
더 많은 정보: <https://elv.sh>.

- 대화형 쉘 세션을 시작:

`elvish`

- 특정 명령([c]ommands)을 실행:

`elvish -c "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">echo 'elvish is executed'</span>`"`

- 특정 스크립트 실행:

`elvish `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/스크립트.elv</span>
