# n

여러 노드 버전을 관리합니다.
더 많은 정보: <https://github.com/tj/n>.

- 지정된 버전의 노드를 설치합니다. 버전이 이미 설치되어 있으면 활성화됩니다:

`n `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">버전</span>

- 설치된 버전을 표시하고 대화형으로 하나를 활성화합니다:

`n`

- 버전을 제거합니다:

`n rm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">버전</span>

- 지정된 버전으로 파일을 실행합니다:

`n use `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">버전</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">파일.js</span>

- 버전의 바이너리 경로를 출력합니다:

`n bin `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">버전</span>
