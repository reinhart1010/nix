# git count

커밋의 총 개수를 출력합니다.
`git-extras`의 일부입니다.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-count>.

- 커밋의 총 개수를 출력:

`git count`

- 기여자별 커밋 개수와 총 커밋 개수를 출력:

`git count --all`
