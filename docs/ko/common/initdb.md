# initdb

디스크에 PostgreSQL 데이터베이스를 생성.
더 많은 정보: <https://www.postgresql.org/docs/9.5/app-initdb.html>.

- `/usr/local/var/postgres`에 데이터베이스를 생성:

`initdb -D /usr/local/var/postgres`
