# vue

Vue.js를 위한 다목적 CLI.
`build`와 같은 일부 하위 명령에는 자체 사용 설명서가 있습니다.
더 많은 정보: <https://cli.vuejs.org>.

- 대화형으로 새 Vue 프로젝트 생성:

`vue create `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">프로젝트_이름</span>

- 웹 UI로 새 프로젝트 생성:

`vue ui`
