# hg root

Hg 저장소의 루트 위치를 표시.
더 많은 정보: <https://www.mercurial-scm.org/doc/hg.1.html#root>.

- 현재 저장소의 루트 위치 표시:

`hg root`

- 지정된 저장소의 루트 위치 표시:

`hg root --cwd `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/폴더</span>
