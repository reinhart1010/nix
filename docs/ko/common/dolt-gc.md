# dolt gc

더 이상 참조되지 않거나, 더 이상 필요하지 않은 데이터를 저장소에서 검색.
더 많은 정보: <https://docs.dolthub.com/cli-reference/cli#dolt-gc>.

- 저장소에서 참조되지 않은 데이터를 정리:

`dolt gc`

- 더 빠르지만 덜 철저한 가비지 수집 프로세스를 시작:

`dolt gc --shallow`
