# haxelib

Haxe 라이브러리 관리자.
더 많은 정보: <https://lib.haxe.org/>.

- Haxe 라이브러리 검색:

`haxelib search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- Haxe 라이브러리 설치:

`haxelib install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리이름</span>

- 특정 버전의 Haxe 라이브러리를 설치:

`haxelib install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리이름</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">버전</span>

- 설치된 모든 Haxe 라이브러리를 업그레이드:

`haxelib upgrade`

- Git 저장소에서 라이브러리의 개발 버전을 설치:

`haxelib git `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리이름</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">git_주소</span>

- Haxe 라이브러리 설치 삭제:

`haxelib remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">라이브러리이름</span>

- 로컬에 설치된 Haxe 라이브러리의 트리를 출력:

`haxelib list`
