더 많은 정보: <https://github.com/busterc/xcv>.

- 파일 자르기:

`xcv x `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">입력_파일</span>

- 파일 복사:

`xcv c `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">입력_파일</span>

- 파일 붙여넣기:

`xcv v `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">출력_파일</span>

- 붙여넣기 가능한 파일 나열:

`xcv l`
