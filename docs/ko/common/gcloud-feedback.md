<hr># gcloud feedback

Google Cloud 팀에게 피드백 제공.
같이 보기: `gcloud`.
더 많은 정보: <https://cloud.google.com/sdk/gcloud/reference/feedback>.

- `gcloud` 팀에게 피드백 제공:

`gcloud feedback`
