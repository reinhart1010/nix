# apm

아톰 편집기 패키지 관리자.
`atom`을 보시오.
더 많은 정보: <https://github.com/atom/apm>.

- <http://atom.io/packages>으로부터 패키지 설치하고 <http://atom.io/themes>에서 테마 설치:

`apm install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지 이름</span>

- 패키지/테마 제거:

`apm remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지 이름</span>

- 패키지/테마 업그레이드:

`apm upgrade `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지 이름</span>
