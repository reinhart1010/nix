# npm uninstall

패키지 제거.
더 많은 정보: <https://docs.npmjs.com/cli/v8/commands/npm-uninstall>.

- 현재 프로젝트에서 패키지 제거:

`npm uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지_이름</span>

- 전역에서 패키지 제거:

`npm uninstall -g `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지_이름</span>

- 여러 패키지를 한 번에 제거:

`npm uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지_이름1 패키지_이름2 ...</span>
