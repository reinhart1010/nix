# pwd

현재 작업 디렉토리의 이름을 출력.
더 많은 정보: <https://www.gnu.org/software/coreutils/pwd>.

- 현재 디렉토리 출력:

`pwd`

- 모든 심볼릭 링크를 해석하여 현재 디렉토리의 "물리적" 경로 출력:

`pwd -P`
