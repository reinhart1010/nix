<hr># hexo

빠르고, 간단하며 강력한 블로그 프레임워크.
더 많은 정보: <https://hexo.io/>.

- 웹사이트 초기화:

`hexo init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">path/to/directory</span>

- 새로운 기사 생성:

`hexo new `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">layout</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">title</span>

- 정적 파일을 생성:

`hexo generate`

- 로컬 서버 실행:

`hexo server`

- 웹사이트 배포:

`hexo deploy`

- 캐시 파일(`db.json`)과 생성된 파일(`public/`)을 정리:

`hexo clean`
