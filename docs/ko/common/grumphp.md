# grumphp

소스 코드 품질 검사를 가능하게 하는 PHP Composer 플러그인.
더 많은 정보: <https://github.com/phpro/grumphp>.

- Git 훅을 등록:

`grumphp git:init`

- 사전 커밋 후크를 수동으로 트리거:

`grumphp git:pre-commit`

- 버전이 지정된 모든 파일을 확인:

`grumphp run`
