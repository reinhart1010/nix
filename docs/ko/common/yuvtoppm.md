# yuvtoppm

Abekas YUV 바이트를 PPM으로 변환.
더 많은 정보: <https://netpbm.sourceforge.net/doc/yuvtoppm.html>.

- 지정된 입력 파일에서 Akebas YUV 바이트를 읽고, 이를 PPM 이미지로 변환하여 지정된 출력 파일에 저장:
