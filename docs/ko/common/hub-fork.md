# hub fork

GitHub 저장소를 포크. `git-extras`의 `git fork`와 같음.
더 많은 정보: <https://hub.github.com/hub-fork.1.html>.

- slug로 GitHub 저장소를 포크:

`hub fork `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">tldr-pages/tldr</span>

- 해당 URL로 GitHub 저장소를 포크:

`hub fork `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">https://github.com/tldr-pages/tldr</span>

- 현재 GitHub 저장소를 포크하고, 원격 이름을 원본으로 설정:

`hub fork --remote-name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">원본</span>
