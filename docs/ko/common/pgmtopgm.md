# pgmtopgm

PGM 이미지 파일 복사.
더 많은 정보: <https://netpbm.sourceforge.net/doc/pgmtopgm.html>.

- `stdin`에서 `stderr`로 PGM 파일 복사:

`pgmtopgm`

- 버전 표시:

`pgmtopgm -version`
