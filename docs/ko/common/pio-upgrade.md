# pio upgrade

PlatformIO를 최신 버전으로 업데이트.
더 많은 정보: <https://docs.platformio.org/en/latest/core/userguide/cmd_upgrade.html>.

- PlatformIO를 최신 버전으로 업데이트:

`pio upgrade`

- PlatformIO를 최신 개발(불안정) 버전으로 업데이트:

`pio upgrade --dev`
