# yadm-encrypt

지정된 암호화 파일에 나열된 파일 암호화.
파일이 암호화된 후 지정된 아카이브 폴더에 저장됩니다.
더 많은 정보: <https://yadm.io/docs/encryption>.

- 지정된 암호화 파일에 나열된 파일 암호화:

`yadm encrypt`
