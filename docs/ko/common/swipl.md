# swipl

SWI-Prolog - 포괄적인 무료 Prolog 환경.
더 많은 정보: <https://www.swi-prolog.org/>.

- 대화형 세션 시작:

`swipl`

- 출력을 표시하지 않고 명령 실행:

`swipl --quiet -t "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>`"`

- 스크립트 실행:

`swipl `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.pl</span>

- 모든 셸 구성 변수 출력:

`swipl --dump-runtime-variables`

- 버전 표시:

`swipl --version`
