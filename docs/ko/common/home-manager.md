<hr># home-manager

Nix를 이용하여 사용자 환경을 관리.
더 많은 정보: <https://github.com/rycee/home-manager>.

- `~/.config/nixpkgs/home.nix`에 정의된 구성을 활성화:

`home-manager build`

- 구성을 활성화하고 해당 구성으로 전환:

`home-manager switch`
