# brew outdated

오래된 casks와 formulae를 나열.
모든 것을 업그레이드하려면, `brew upgrade`를 사용.
더 많은 정보: <https://docs.brew.sh/Manpage#outdated-options-formulacask->.

- 오래된 casks와 formulae를 모두 나열:

`brew outdated`

- 오직 오래된 formulae만 나열:

`brew outdated --formula`

- 오직 오래된 casks를 나열:

`brew outdated --cask`
