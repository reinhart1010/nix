# d8

V8 JavaScript 엔진용 개발자 쉘.
더 많은 정보: <https://v8.dev/docs/d8>.

- REPL (대화형 쉘)을 시작:

`d8`

- JavaScript 파일 실행:

`d8 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일.js</span>

- JavaScript 표현식을 평가:

`d8 -e "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">코드</span>
