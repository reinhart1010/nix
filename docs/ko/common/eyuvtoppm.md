# eyuvtoppm

Berkeley YUV 파일을 PPM으로 변환.
더 많은 정보: <https://netpbm.sourceforge.net/doc/eyuvtoppm.html>.

- 지정된 입력 파일에서 Berkeley YUV 파일을 읽고, 이를 PPM 이미지로 변환한 후 지정된 출력 파일에 저장:
