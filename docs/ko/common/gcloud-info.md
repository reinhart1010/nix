# gcloud info

현재 `gcloud` 환경에 대한 정보를 표시합니다.
더 많은 정보: <https://cloud.google.com/sdk/gcloud/reference/info>.

- `gcloud` 환경 정보 표시:

`gcloud info`

- 네트워크 연결 상태 및 숨겨진 속성 확인:

`gcloud info --run-diagnostics`

- 가장 최근의 로그 파일 내용 출력:

`gcloud info --show-log`
