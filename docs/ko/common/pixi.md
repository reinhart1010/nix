# pixi

프로젝트를 위한 개발자 워크플로우 및 환경 관리 도구.
더 많은 정보: <https://pixi.sh>.

- 새 프로젝트 초기화:

`pixi init `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/프로젝트</span>

- 프로젝트 의존성 추가:

`pixi add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">의존성1 의존성2 ...</span>

- 프로젝트 환경에서 pixi 셸 시작:

`pixi shell`

- 프로젝트 환경에서 작업 실행:

`pixi run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">작업</span>

- 프로젝트 환경에서 작업 관리:

`pixi task `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>

- 도움말 메시지 출력:

`pixi `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>` --help`

- 환경 및 작업 캐시 정리:

`pixi clean`
