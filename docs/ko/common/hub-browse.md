# hub browse

브라우저에서 GitHub 저장소를 열거나 URL을 출력.
더 많은 정보: <https://hub.github.com/hub-browse.1.html>.

- 기본 웹 브라우저에서 현재 저장소의 홈페이지를 열기:

`hub browse`

- 기본 웹 브라우저에서 특정 저장소의 홈페이지를 열기:

`hub browse `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">소유자</span>`/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">레포지토리</span>

- 기본 웹 브라우저에서 특정 저장소의 하위 페이지를 열기, 하위페이지는 "wiki", "commits", "issues" 또는 기타일 수 있음 (기본값: "tree"):

`hub browse `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">소유자</span>`/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">레포지토리</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위페이지</span>
