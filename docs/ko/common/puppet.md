# puppet

서버의 설정을 관리하고 자동화하는 데 도움을 줍니다.
`agent`와 같은 일부 하위 명령에는 자체 사용 설명서가 있습니다.
더 많은 정보: <https://puppet.com/>.

- Puppet 하위 명령 실행:

`puppet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위_명령</span>

- Puppet 버전 확인:

`puppet --version`

- 도움말 표시:

`puppet --help`

- 하위 명령에 대한 도움말 표시:

`puppet help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">하위_명령</span>
