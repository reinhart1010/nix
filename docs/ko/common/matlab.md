<hr># matlab

MathWorks의 수치 계산 환경.
더 많은 정보: <https://uk.mathworks.com/help/matlab/>.

- 시작 시 스플래시 화면 없이 실행:

`matlab -nosplash`

- MATLAB 명령문 실행:

`matlab -r "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">matlab_명령문</span>`"`

- MATLAB 스크립트 실행:

`matlab -r "run(`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/스크립트.m</span>`)"`
