# git clear-soft

현재 브랜치와 `.gitignore`에 포함된 파일을 제외하고 Git 작업 디렉토리를 새로 클론한 것처럼 초기화.
`git-extras`의 일부.
더 많은 정보: <https://github.com/tj/git-extras/blob/master/Commands.md#git-clear-soft>.

- 모든 추적된 파일을 초기화하고 모든 추적되지 않은 파일 삭제:

`git clear-soft`
