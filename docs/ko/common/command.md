<hr># command

Command 명령은 쉘이 프로그램을 실행하고 동일한 이름의 기능, 빌드인 및 별칭을 무시하도록 합니다.
더 많은 정보: <https://manned.org/command>.

- ls 별칭이 존재하더라도 문자 그대로 `ls` 프로그램을 실행:

`command `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ls</span>
