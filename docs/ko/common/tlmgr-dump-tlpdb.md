# tlmgr dump-tlpdb

TeX Live 패키지 데이터베이스 덤프.
더 많은 정보: <https://www.tug.org/texlive/tlmgr.html>.

- 로컬 패키지 데이터베이스 덤프:

`tlmgr dump-tlpdb --local`

- 원격 패키지 데이터베이스 덤프:

`tlmgr dump-tlpdb --remote`

- 로컬 패키지 데이터베이스를 JSON 형식으로 덤프:

`tlmgr dump-tlpdb --local --json`
