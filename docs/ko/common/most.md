# most

하나 또는 여러 파일을 대화형으로 열어 스크롤 및 검색을 할 수 있습니다.
더 많은 정보: <https://manned.org/most>.

- 파일 열기:

`most `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>

- 여러 파일 열기:

`most `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일1 경로/대상/파일2 ...</span>

- "문자열"의 첫 번째 발생 지점에서 파일 열기:

`most `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로/대상/파일</span>` +/`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">문자열</span>

- 열린 파일 사이를 이동:

`:O n`

- 100번째 줄로 이동:

<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">100</span>`j`

- 현재 파일 편집:

`e`

- 현재 창을 반으로 분할:

`<CTRL-x> o`

- 종료:

`Q`
