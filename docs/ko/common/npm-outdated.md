<hr># npm-outdated

오래된 npm 패키지 의존성을 확인.
더 많은 정보: <https://docs.npmjs.com/cli/commands/npm-outdated>.

- 프로젝트에서 오래된 패키지 찾기:

`npm outdated`

- 현재 프로젝트와 상관없이 오래된 패키지 찾기:

`npm outdated --all`
