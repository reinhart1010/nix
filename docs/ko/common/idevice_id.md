<hr># idevice_id

연결된 iOS 장치를 나열하거나 특정 장치의 장치 이름을 출력.
더 많은 정보: <https://manned.org/idevice_id>.

- 연결된 모든 장치의 UDID를 나열:

`idevice_id --list`

- 네트워크를 통해 사용 가능한 모든 장치의 UDID를 나열:

`idevice_id --network`
