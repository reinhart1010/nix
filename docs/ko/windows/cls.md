# cls

화면을 지웁니다.
PowerShell에서는 이 명령이 `Clear-Host`의 별칭입니다. 이 문서는 명령 프롬프트(`cmd`) 버전의 `cls`를 기반으로 합니다.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/cls>.

- 해당 PowerShell 명령의 문서 보기:

`tldr clear-host`

- 화면 지우기:

`cls`
