이 명령은 `Where-Object` 의 에일리어스 (별칭) 입니다.
더 많은 정보: <https://learn.microsoft.com/powershell/module/microsoft.powershell.core/where-object>.

- 원본 명령의 도큐멘테이션 (설명서) 보기:

`tldr Where-Object`
