# color

콘솔의 전경 및 배경 색상을 설정.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/color>.

- 콘솔 색상을 기본값으로 설정:

`color`

- 사용 가능한 색상 값 및 세부 정보 나열:

`color /?`

- 16진수 번호(`1-9,a-f`)를 사용하여 콘솔 전경 및 배경을 특정 색상으로 설정:

`color `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">전경_코드</span><span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">배경_코드</span>
