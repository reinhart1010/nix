# Get-Location

현재 작업 중인 디렉토리의 이름 출력.
이 명령은 PowerShell에서만 실행 가능.
더 많은 정보: <https://learn.microsoft.com/powershell/module/microsoft.powershell.management/get-location>.

- 현재 디렉토리 출력:

`Get-Location`
