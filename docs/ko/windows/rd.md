# rd

이 명령은 Command Prompt에서는 `rmdir`의 별칭이며, PowerShell에서는 `Remove-Item`의 별칭입니다.

- 원본 Command Prompt 명령에 대한 문서 보기:

`tldr rmdir`

- 원본 PowerShell 명령에 대한 문서 보기:

`tldr remove-item`
