# reg unload

`reg load` 명령을 사용하여 로드된 레지스트리에서 데이터를 제거.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/reg-unload>.

- 지정된 키에 대한 레지스트리 데이터 제거:
