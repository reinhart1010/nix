# dir

디렉토리 내용을 나열.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/dir>.

- 현재 디렉토리의 내용 표시:

`dir`

- 지정된 디렉토리의 내용 표시:

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로\대상\폴더</span>

- 숨겨진 파일을 포함하여 현재 디렉토리의 내용 표시:

`dir /a`

- 숨겨진 파일을 포함하여 지정된 디렉토리의 내용 표시:

`dir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">경로\대상\폴더</span>` /a`

- 추가 정보 없이 디렉토리 및 파일의 간단한 목록 표시:

`dir /b`
