# nfsstat

NFS 서버에 대한 호출 수를 표시하거나 재설정합니다.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/nfsstat>.

- NFS 서버에 대한 기록된 호출 수 표시:

`nfsstat`

- NFS 서버에 대한 기록된 호출 수 재설정:

`nfsstat -z`
