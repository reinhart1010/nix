# exit

현재 CMD 인스턴스 또는 현재 배치 파일을 종료.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/exit>.

- 현재 CMD 인스턴스 종료:

`exit`

- 현재 배치 스크립트 종료:

`exit /b`

- 특정 종료 코드를 사용하여 종료:

`exit `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">2</span>
