# choco

Chocolatey 패키지 관리자.
`install`과 같은 일부 하위 명령에는 자체 사용 설명서가 있습니다.
더 많은 정보: <https://chocolatey.org>.

- Chocolatey 명령 실행:

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>

- 도움말 표시:

`choco -?`

- 특정 명령에 대한 도움말 표시:

`choco `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">명령</span>` -?`

- 버전 표시:

`choco --version`
