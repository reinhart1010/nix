# time

시스템 시간을 표시하거나 설정합니다.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/time>.

- 현재 시스템 시간 표시 및 새 시간 입력 요청 (비워두면 변경되지 않음):

`time`

- 현재 시스템 시간 표시 (새 시간 입력 요청 없음):

`time /t`
