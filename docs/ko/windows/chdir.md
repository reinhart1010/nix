# chdir

이 명령은 명령 프롬프트의 `cd` 및 이후 PowerShell의 `Set-Location`의 별칭입니다.
더 많은 정보: <https://learn.microsoft.com/windows-server/administration/windows-commands/chdir>.

- 원본 명령 프롬프트 명령에 대한 문서 보기:

`tldr cd`

- 원본 PowerShell 명령에 대한 문서 보기:

`tldr set-location`
