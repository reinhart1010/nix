# choco feature

Chocolatey를 통해 기능과 상호작용.
더 많은 정보: <https://chocolatey.org/docs/commands-feature>.

- 사용 가능한 기능 목록 표시:

`choco feature list`

- 기능 활성화:

`choco feature enable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>

- 기능 비활성화:

`choco feature disable --name `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">이름</span>
