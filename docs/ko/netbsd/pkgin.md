# pkgin

NetBSD에서 `pkgsrc` 바이너리 패키지를 관리합니다.
더 많은 정보: <https://pkgin.net/#usage>.

- 패키지 설치:

`pkgin install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 패키지 및 해당 의존성 제거:

`pkgin remove `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">패키지</span>

- 모든 패키지 업그레이드:

`pkgin full-upgrade`

- 패키지 검색:

`pkgin search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">키워드</span>

- 설치된 패키지 나열:

`pkgin list`

- 필요없는 의존성 제거:

`pkgin autoremove`
