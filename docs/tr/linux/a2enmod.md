# a2enmod

Debian tabanlı işletim sistemlerinde Apache modülünü etkinleştir.
Daha fazla bilgi için: <https://manned.org/a2enmod.8>.

- Bir modülü etkinleştir:

`sudo a2enmod `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">modül</span>

- Bilgilendirici mesajları gösterme:

`sudo a2enmod --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">modül</span>
