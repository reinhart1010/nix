# a2ensite

Debian tabanlı işletim sistemlerinde Apache sanal hostu etkinleştir.
Daha fazla bilgi için: <https://manpages.debian.org/latest/apache2/a2ensite.8.en.html>.

- Bir sanal hostu etkinleştir:

`sudo a2ensite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sanal_host</span>

- Bilgilendirici mesajları gösterme:

`sudo a2ensite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sanal_host</span>
