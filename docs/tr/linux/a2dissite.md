# a2dissite

Debian tabanlı işletim sistemlerinde bir Apache sanal hostunu devre dışı bırak.
Daha fazla bilgi için: <https://manned.org/a2dissite.8>.

- Uzak hostu devre dışı bırak:

`sudo a2dissite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sanal_host</span>

- Bilgilendirici mesajları gösterme:

`sudo a2dissite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sanal_host</span>
