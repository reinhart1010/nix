# a2disconf

Debian tabanlı işletim sistemlerinde Apache konfigürasyon dosyasını devre dışı bırak.
Daha fazla bilgi için: <https://manpages.debian.org/latest/apache2/a2disconf.8.en.html>.

- Bir konfigürasyon dosyasını devre dışı bırak:

`sudo a2disconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">konfigürasyon_dosyası</span>

- Bilgilendirici mesajı gösterme:

`sudo a2disconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">konfigürasyon_dosyası</span>
