# pw-cli

Pipewire komut satır arayüzü.
Daha fazla bilgi için: <https://docs.pipewire.org/page_man_pw_cli_1.html>.

- Tüm düğümleri (taban ve kaynakları) ID'leri ile birlikte yazdır:

`pw-cli list-objects Node`

- Belirtilen ID'ye sahip objeye dair bilgileri yazdır:

`pw-cli info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">4</span>

- Tüm objelerin bilgilerini yazdır:

`pw-cli info all`
