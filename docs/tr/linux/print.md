# print

`run-mailcap` komutunun print özelliğinin öbür adı.
Normalde `run-mailcap` komutu mime-tarzı/dosya işlemek için kullanılır.
Daha fazla bilgi için: <https://manned.org/print>.

- Bir dosyayı yazdır:

`print `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">dosya_ismi</span>

- `run-mailcap` ile yazdır:

`run-mailcap --action=print `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">dosya_ismi</span>
