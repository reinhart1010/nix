# tldr

Komut satırı araçları için tldr-pages projesinden basit yardım sayfaları görüntüler.
Daha fazla bilgi: <https://tldr.sh>.

- Bir komutun tipik kullanımını göster (ipucu: burayı görüntülemek için kullandığınız komutun aynısı!):

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">komut</span>

- Linux için tar tldr sayfasını göster:

`tldr -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">linux</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">tar</span>

- Bir Git alt komutu için yardım al:

`tldr `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">git-checkout</span>

- (Eğer alıcı önbellek oluşumunu destekliyorsa) Yerel paketleri güncelle:

`tldr -u`
