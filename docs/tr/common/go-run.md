# go run

Binary (ikili sayı değeri) kaydetmeden Go kodunu derle ve çalıştır.
Daha fazla bilgi için: <https://pkg.go.dev/cmd/go#hdr-Compile_and_run_Go_program>.

- Bir Go dosyası çalıştır:

`go run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">örnek/konum/dosya.go</span>

- Ana bir Go paketi çalıştır:

`go run `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">örnek/konum/paket</span>
