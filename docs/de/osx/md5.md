# md5

Berechne kryptografische MD5-Prüfsummen.
Weitere Informationen: <https://ss64.com/osx/md5.html>.

- Berechne die MD5-Prüfsumme einer Datei und gib sie mit Dateinamen auf der Konsole aus:

`md5 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/datei</span>

- Berechne die MD5-Prüfsumme mehrerer Dateien und gib sie mit Dateinamen auf der Konsole aus:

`md5 `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/datei1 pfad/zu/datei2 ...</span>

- Gib nur die MD5-Prüfsumme auf der Konsole aus:

`md5 -q `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/datei</span>

- Berechne die MD5-Prüfsumme für den gegebenen Text:

`md5 -s "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">text</span>`"`
