# a2disconf

Deaktiviert eine Apache-Konfigurationsdatei auf Debian-basierten Betriebssystemen.
Weitere Informationen: <https://manpages.debian.org/latest/apache2/a2disconf.8.en.html>.

- Deaktiviere eine Konfigurationsdatei:

`sudo a2disconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">konfigurationsdatei</span>

- Zeige keine Informationsnachrichten an:

`sudo a2disconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">konfigurationsdatei</span>
