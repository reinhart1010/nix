# a2dissite

Deaktiviert einen Apache virtuellen Host auf Debian-basierten Betriebssystemen.
Weitere Informationen: <https://manned.org/a2dissite.8>.

- Deaktiviere einen virtuellen Host:

`sudo a2dissite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">virtueller_host</span>

- Zeige keine Informationsnachrichten an:

`sudo a2dissite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">virtueller_host</span>
