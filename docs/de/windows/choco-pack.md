# choco pack

Verpacke eine NuGet-Spezifikation in eine nupkg-Datei.
Weitere Informationen: <https://chocolatey.org/docs/commands-pack>.

- Verpacke eine NuGet-Spezifikation in eine nupkg-Datei:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/spezifikation</span>

- Verpacke eine NuGet-Spezifikation in eine nupkg-Datei und bestimme die Version der ausgegebenen Datei:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/spezifikation</span>` --version `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">version</span>

- Verpacke eine NuGet-Spezifikation in eine nupkg-Datei, welche in einem bestimmten Verzeichnis gespeichert wird:

`choco pack `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/spezifikation</span>` --output-directory `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/ausgabe_verzeichnis</span>
