# anki

Leistungsstarkes, intelligentes Lernkartenprogramm.
Weitere Informationen: <https://docs.ankiweb.net>.

- Starte `anki`:

`anki`

- Starte `anki` mit einem bestimmten Profil:

`anki -p `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">profil_name</span>

- Starte `anki` in einer bestimmten Sprache:

`anki -l `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sprache</span>

- Starte `anki` von einem bestimmten Verzeichneis anstelle des Standardverzeichnis (`~/Anki`):

`anki -b `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">pfad/zu/verzeichnis</span>
