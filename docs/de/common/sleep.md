# sleep

Verzögert für einen bestimmten Zeitraum.
Weitere Informationen: <https://www.gnu.org/software/coreutils/sleep>.

- Verzögere in Sekunden:

`sleep `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">sekunden</span>

- Verzögere in Minuten:

`sleep `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">minuten</span>`m`

- Verzögere in Stunden:

`sleep `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">stunden</span>`h`
