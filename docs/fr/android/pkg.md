# pkg

Utilitaire de gestion de paquets pour Termux.
Plus d'informations : <https://wiki.termux.com/wiki/Package_Management>.

- Mets à jour tous les paquets déjà installés :

`pkg upgrade`

- Installe un paquet :

`pkg install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paquet</span>

- Désinstalle un paquet :

`pkg uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paquet</span>

- Réinstalle un paquet :

`pkg reinstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paquet</span>

- Cherche un paquet :

`pkg search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">paquet</span>
