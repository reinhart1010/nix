<hr># antibody

Le gestionnaire de modules shell "le plus rapide".
Plus d'informations : <https://getantibody.github.io>.

- Regroupe tous les modules pour un chargement statique :

`antibody bundle < `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">~/.zsh_modules.txt</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">~/.zsh_modules.sh</span>

- Mets à jour tous les modules :

`antibody update`

- Liste tous les modules installés :

`antibody list`
