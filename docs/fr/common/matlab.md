# matlab

Environnement de calcul numérique créé par MathWorks.
Plus d'informations : <https://uk.mathworks.com/help/matlab/>.

- Lance MATLAB sans afficher l'écran de démarrage :

`matlab -nosplash`

- Exécute une instruction MATLAB :

`matlab -r "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">instruction_matlab</span>`"`

- Exécute un script MATLAB :

`matlab -r "run(`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">chemin/vers/script.m</span>`)"`
