<hr># anytopnm

Convertis n'importe quel type de fichier image vers un format d'image commun.
Plus d'informations : <https://netpbm.sourceforge.net/doc/anytopnm.html>.

- Convertis une image d'entrée au format PBM, PGM ou PPM, peut importe le type d'entrée :

`anytopnm `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">chemin/vers/entrée</span>` > `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">chemin/vers/sortie.pnm</span>

- Afficher la version :

`anytopnm -version`
