# a2ensite

Active un hôte virtuel Apache sur des systèmes d'exploitation (SE) basés sur Debian.
Plus d'informations : <https://manpages.debian.org/latest/apache2/a2ensite.8.en.html>.

- Active un hôte virtuel :

`sudo a2ensite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">hote_virtuel</span>

- N'affiche aucun message (mode silencieux) :

`sudo a2ensite --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">hote_virtuel</span>
