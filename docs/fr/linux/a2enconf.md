# a2enconf

Active un fichier de configuration sur une distribution Debian.
Plus d'informations : <https://manpages.debian.org/latest/apache2/a2enconf.8.en.html>.

- Active un fichier de configuration :

`sudo a2enconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fichier_de_configuration</span>

- N'affiche aucun message (mode silencieux) :

`sudo a2enconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fichier_de_configuration</span>
