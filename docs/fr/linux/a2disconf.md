# a2disconf

Désactive un fichier de configuration sur une distribution Debian.
Plus d'informations : <https://manned.org/a2disconf.8>.

- Désactive un fichier de configuration :

`sudo a2disconf `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fichier_de_configuration</span>

- N'affiche aucun message (mode silencieux) :

`sudo a2disconf --quiet `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">fichier_de_configuration</span>
