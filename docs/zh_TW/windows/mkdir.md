更多資訊：<https://learn.microsoft.com/windows-server/administration/windows-commands/mkdir>.

- 建立一個資料夾：

`mkdir `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">資料夾名稱</span>
