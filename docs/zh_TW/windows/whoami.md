<hr># whoami

顯示關於當前使用者的詳細資訊。
更多資訊：<https://learn.microsoft.com/windows-server/administration/windows-commands/whoami>.

- 顯示當前使用者的名稱：

`whoami`

- 顯示當前使用者所屬的群組：

`whoami /groups`

- 顯示當前使用者的權限：

`whoami /priv`

- 顯示當前使用者的使用者主體名稱（UPN）：

`whoami /upn`

- 顯示當前使用者的登錄 ID：

`whoami /logonid`
