# bugreportz

生成一個壓縮的 Android 錯誤報告。
此命令只能透過 `adb shell` 使用。
更多資訊：<https://cs.android.com/android/platform/superproject/+/master:frameworks/native/cmds/bugreportz>.

- 生成一個完整的 Android 裝置壓縮錯誤報告：

`bugreportz`

- 顯示正在執行的 `bugreportz` 的作業進度：

`bugreportz -p`

- 顯示 `bugreportz` 的版本：

`bugreportz -v`

- 顯示幫助資訊：

`bugreportz -h`
