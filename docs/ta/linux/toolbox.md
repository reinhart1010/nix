# toolbox

லினக்ஸ் இல் கொள்கலன்கள் செய்யப்பட்ட கட்டளை வரி சூழல்களுக்கான கருவி.
`create` போன்ற சில துணைக் கட்டளைகள் அவற்றின் சொந்த பயன்பாட்டு ஆவணங்களைக் கொண்டுள்ளன.
மேலும் விவரத்திற்கு: <https://manned.org/toolbox.1>.

- `toolbox` துணைக் கட்டளையை இயக்கவும்:

`toolbox `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">துணை_கட்டளை</span>

- `toolbox` துணைக் கட்டளைக்கான உதவியைக் காட்டு (`create`, `enter`, `rm`, `rmi` போன்றவை.):

`toolbox help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">துணை_கட்டளை</span>

- பொதுவான உதவியைக் காட்டு:

`toolbox --help`

- `toolbox` பதிப்பைக் காட்டு:

`toolbox --version`
