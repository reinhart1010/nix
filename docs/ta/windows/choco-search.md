# choco search

சாக்லேட்டியுடன் உள்ளூர் அல்லது தொலைநிலைப் பொதியைத் தேடுங்கள்.
மேலும் விவரத்திற்கு: <https://chocolatey.org/docs/commands-search>.

- ஒரு தொகுப்பைத் தேடுங்கள்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>

- உள்நாட்டில் ஒரு தொகுப்பைத் தேடுங்கள்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>` --local-only`

- முடிவுகளில் சரியான பொருத்தங்களை மட்டும் சேர்க்கவும்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>` --exact`

- அனைத்து அறிவுறுத்தல்களையும் தானாக உறுதிப்படுத்தவும்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>` --yes`

- தொகுப்புகளைத் தேட தனிப்பயன் மூலத்தைக் குறிப்பிடவும்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>` --source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">மூல_முகவரி|alias</span>

- அங்கீகாரத்திற்கான பயனர்பெயர் மற்றும் கடவுச்சொல்லை வழங்கவும்:

`choco search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வினவல்</span>` --user `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">பயனர்ப்பெயர்</span>` --password `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">கடவுச்சொல்</span>
