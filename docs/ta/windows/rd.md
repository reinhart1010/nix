# rd

இந்த கட்டளை, கட்டளை வரியில் `rmdir` மற்றும் PowerShell இல் `Remove-Item` என்பதன் மாற்றுப்பெயர்.

- அசல் கட்டளை வரியில் (Command Prompt) கட்டளைக்கான ஆவணங்களைக் காண்க:

`tldr rmdir`

- அசல் PowerShell கட்டளைக்கான ஆவணங்களைக் காண்க:

`tldr remove-item`
