# gem

ரூபி நிரலாக்க மொழிக்கான தொகுப்பு மேலாளருடன் தொடர்பு கொள்ளுங்கள்.
மேலும் விவரத்திற்கு: <https://guides.rubygems.org>.

- தொலை ரத்தினங்களைத் தேடி, கிடைக்கக்கூடிய அனைத்து பதிப்புகளையும் காட்டு:

`gem search `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">வழக்கமான_வெளிப்பாடு</span>` --all`

- ரத்தினத்தின் சமீபத்திய பதிப்பை நிறுவவும்:

`gem install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>

- ஒரு ரத்தினத்தின் குறிப்பிட்ட பதிப்பை நிறுவவும்:

`gem install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>` --version `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1.0.0</span>

- ஒரு ரத்தினத்தின் சமீபத்திய பொருந்தக்கூடிய (SemVer) பதிப்பை நிறுவவும்:

`gem install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>` --version '~> `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1.0</span>`'`

- ஒரு ரத்தினத்தைப் புதுப்பிக்கவும்:

`gem update `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>

- அனைத்து உள்ளூர் ரத்தினங்களையும் பட்டியலிடுங்கள்:

`gem list`

- ஒரு ரத்தினத்தை நிறுவல் நீக்கவும்:

`gem uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>

- ஒரு ரத்தினத்தின் குறிப்பிட்ட பதிப்பை நிறுவல் நீக்கவும்:

`gem uninstall `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ரத்தின_பெயர்</span>` --version `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">1.0.0</span>
