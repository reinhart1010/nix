# openssl

مجموعة أدوات تشفير OpenSSL.
بعض الأوامر الفرعية مثل `req` تحتوي على وثائق استخدام خاصة بها.
لمزيد من التفاصيل: <https://www.openssl.org>.

- عرض الإرشادات:

`openssl help`

- عرض الإرشادات لأمر فرعي محدد:

`openssl help `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">x509</span>

- عرض الإصدار:

`openssl version`
