# gsha384sum

คำสั่งนี้เป็นอีกชื่อหนึ่งของคำสั่ง `-p linux sha384sum`

- เรียกดูรายละเอียดสำหรับคำสั่งตัวเต็ม:

`tldr -p linux sha384sum`
