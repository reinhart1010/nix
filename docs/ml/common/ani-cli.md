# ani-cli

അനിമേ തിരയുവാനും കാണുവാനുമുള്ള കമാൻഡ് ലൈൻ യൂട്ടിലിറ്റി.
കൂടുതൽ വിവരങ്ങൾ: <https://github.com/pystardust/ani-cli>.

- ഒരു അനിമേയുടെ പേര് സെർച്ച് ചെയ്യുവാൻ:

`ani-cli "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">അനിമേയുടെ_പേര്</span>`"`

- ഒരു എപ്പിസോഡ് ഡൗൺലോഡ് ചെയ്യുവാൻ:

`ani-cli -d "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">അനിമേയുടെ_പേര്</span>`"`

- വി.എൽ.സി മീഡിയ പ്ലേയറിൽ കാണുവാൻ:

`ani-cli -v "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">അനിമേയുടെ_പേര്</span>`"`

- നിശ്ചിത എപ്പിസോഡ് കാണുവാൻ:

`ani-cli -e `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">എപ്പിസോഡ്_നമ്പർ</span>` "`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">അനിമേയുടെ_പേര്</span>`"`

- മുൻപ് കണ്ടുകൊണ്ടിരുന്ന അനിമേ തുടർന്ന് കാണുവാൻ:

`ani-cli -c`

- ഈ യൂട്ടിലിറ്റി അപ്ഡേറ്റ് ചെയ്യുവാൻ:

`ani-cli -U`
