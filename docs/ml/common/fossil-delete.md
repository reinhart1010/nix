# fossil delete

ഈ കമാൻഡ് `fossil rm` എന്നത്തിന്റെ അപരനാമമാണ്.
കൂടുതൽ വിവരങ്ങൾ: <https://fossil-scm.org/home/help/delete>.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr fossil rm`
