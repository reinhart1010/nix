<hr># cls

ഈ കമാൻഡ് `clear-host` എന്നത്തിന്റെ അപരനാമമാണ്.
കൂടുതൽ വിവരങ്ങൾ: <https://learn.microsoft.com/windows-server/administration/windows-commands/cls>.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr clear-host`
