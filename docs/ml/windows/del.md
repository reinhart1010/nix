<hr># del

ഈ കമാൻഡ് `remove-item` എന്നത്തിന്റെ അപരനാമമാണ്.
കൂടുതൽ വിവരങ്ങൾ: <https://learn.microsoft.com/windows-server/administration/windows-commands/del>.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr remove-item`
