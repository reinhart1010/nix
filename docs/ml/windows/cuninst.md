# cuninst

ഈ കമാൻഡ് `choco uninstall` എന്നത്തിന്റെ അപരനാമമാണ്.
കൂടുതൽ വിവരങ്ങൾ: <https://docs.chocolatey.org/en-us/choco/commands/uninstall>.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr choco uninstall`
