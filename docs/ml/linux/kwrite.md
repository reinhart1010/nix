# kwrite

KDE ഡെസ്ക്ടോപ് പ്രോജക്ടിന്റെ ടെക്സ്റ്റ് എഡിറ്റർ.
കാണുക `kate`.
കൂടുതൽ വിവരങ്ങൾ: <https://apps.kde.org/kwrite/>.

- ഒരു ടെക്സ്റ്റ് ഫയൽ ഓപ്പൺ ചെയ്യുവാൻ:

`kwrite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയലിലേക്കുള്ള/പാത</span>

- ഒന്നിൽ കൂടുതൽ ഫയലുകൾ ഓപ്പൺ ചെയ്യുവാൻ:

`kwrite `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയൽ1 ഫയൽ2 ...</span>

- ഒരു പ്രത്യേക എൻകോഡിങ്ങിൽ ഫയൽ ഓപ്പൺ ചെയ്യുവാൻ:

`kwrite --encoding=`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">UTF-8</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയലിലേക്കുള്ള/പാത</span>

- ഒരു ടെക്സ്റ്റ് ഫയലിന്റെ ആവശ്യമുള്ള ലൈനിലേക്കും കോളത്തിലേക്കും നാവിഗേറ്റ് ചെയ്യുവാൻ:

`kwrite --line `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ലൈൻ_നമ്പർ</span>` --column `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">കോളം_നമ്പർ</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയലിലേക്കുള്ള/പാത</span>
