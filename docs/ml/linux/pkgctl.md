കൂടുതൽ വിവരങ്ങൾ: <https://man.archlinux.org/man/pkgctl.1>.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr pkgctl auth`
