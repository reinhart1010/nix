# gedit

ഗ്നോം ഡെസ്ക്ടോപ് പ്രോജക്ടിന്റെ ടെക്സ്റ്റ് എഡിറ്റർ.
കൂടുതൽ വിവരങ്ങൾ: <https://help.gnome.org/users/gedit/stable/>.

- ഒരു ടെക്സ്റ്റ് ഫയൽ ഓപ്പൺ ചെയ്യുവാൻ:

`gedit `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയലിലേക്കുള്ള/പാത</span>

- ഒന്നിൽ കൂടുതൽ ടെക്സ്റ്റ് ഫയലുകൾ ഓപ്പൺ ചെയ്യുവാൻ:

`gedit `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയൽ1 ഫയൽ2 ...</span>

- ഒരു പ്രത്യേക എൻകോഡിങ്ങിൽ ഫയൽ ഓപ്പൺ ചെയ്യുവാൻ:

`gedit --encoding `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">UTF-8</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ഫയലിലേക്കുള്ള/പാത</span>

- സപ്പോർട്ട് ചെയ്യപ്പെട്ട എല്ലാവിധ എൻകോഡിങ്ങും കാണുവാൻ:

`gedit --list-encodings`
