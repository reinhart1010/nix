# lsmod

ലിനക്സ് കെർണൽ മൊഡ്യൂളുകളുടെ അവസ്ഥ കാണാൻ.
ലിനക്സ് കെർണൽ മൊഡ്യൂൾ ലോഡ് ചെയ്യാൻ `modprobe` കാണുക.
കൂടുതൽ വിവരങ്ങൾ: <https://manned.org/lsmod>.

- ഇപ്പോൾ ലോഡ് ചെയ്ത മൊഡ്യൂളുകൾ കാണിക്കുക:

`lsmod`
