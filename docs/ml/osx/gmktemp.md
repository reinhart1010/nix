<hr># gmktemp

ഈ കമാൻഡ് `-p linux mktemp` എന്നത്തിന്റെ അപരനാമമാണ്.

- യഥാർത്ഥ കമാൻഡിനായി ഡോക്യുമെന്റേഷൻ കാണുക:

`tldr -p linux mktemp`
