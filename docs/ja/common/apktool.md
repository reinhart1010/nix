# apktool

APK ファイルの解析ツールです。
詳しくはこちら: <https://ibotpeaches.github.io/Apktool/>

- APK ファイルのデコード:

`apktool d `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">APK ファイルのパス</span>

- 指定したディレクトリから APK ファイルを生成:

`apktool b `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ディレクトリへのパス</span>

- フレームワークをインストールし、保存する:

`apktool if `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">フレームワークファイルへのパス</span>
