# asdf

さまざまなパッケージのバージョンを管理するためのコマンドラインインターフェイスです。
詳しくはこちら: <https://asdf-vm.com>

- 利用可能なプラグインの一覧を表示:

`asdf plugin list all`

- プラグインのインストール:

`asdf plugin add `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">プラグイン名</span>

- パッケージで利用可能なすべてのバージョンの一覧を表示:

`asdf list all `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ名</span>

- パッケージの特定バージョンのインストール:

`asdf install `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ名</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">バージョン</span>

- パッケージのグローバルバージョンの設定:

`asdf global `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ名</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">バージョン</span>

- パッケージのローカルバージョンの設定:

`asdf local `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ名</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">バージョン</span>
