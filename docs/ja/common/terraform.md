# terraform

インフラストラクチャをコードとして作成し、クラウドプロバイダにデプロイします。
もっと詳しく: <https://www.terraform.io/>。

- 新規または既存のTerraform設定を初期化する:

`terraform init`

- 設定ファイルが構文的に正しいか確認する:

`terraform validate`

- Terraformの言語スタイルの規則に従って設定をフォーマットする:

`terraform fmt`

- 実行プランを生成して表示する:

`terraform plan`

- インフラストラクチャを構築または変更する:

`terraform apply`

- Terraformが管理するインフラストラクチャを破棄する:

`terraform destroy`
