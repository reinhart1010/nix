# choco info

Chocolateyのパッケージに関する詳細情報を表示します。
詳しくはこちら: <https://chocolatey.org/docs/commands-info>

- 特定のパッケージに関する情報を表示します:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ</span>

- ローカルパッケージ情報のみを表示します:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ</span>` --local-only`

- パッケージ情報を受信するカスタムソースを指定します:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ</span>` --source `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ソースURL|エイリアス</span>

- 認証用のユーザー名とパスワードを入力します:

`choco info `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パッケージ</span>` --user `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">ユーザー名</span>` --password `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">パスワード</span>
