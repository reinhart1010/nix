# sass

Konwertuje pliki SCSS lub Sass na CSS.
Więcej informacji: <https://sass-lang.com/documentation/cli/dart-sass>.

- Konwertuj plik SCSS lub Sass do CSS i wypisz wynik:

`sass `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwejsciowy.scss|plikwejsciowy.sass</span>

- Konwertuj plik SCSS lub Sass do CSS i zapisz wynik w pliku:

`sass `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwejsciowy.scss|plikwejsciowy.sass</span>` `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwyjsciowy.css</span>

- Obejrzyj zmiany w pliku SCSS lub Sass i wyślij lub zaktualizuj plik CSS o tej samej nazwie:

`sass --watch `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwejsciowy.scss|plikwejsciowy.sass</span>

- Obejrzyj zmiany w pliku SCSS lub Sass i wyślij lub zaktualizuj plik CSS o podanej nazwie:

`sass --watch `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwejsciowy.scss|plikwejsciowy.sass</span>`:`<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">plikwyjsciowy.css</span>
