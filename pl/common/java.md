<hr># java

Java Application Launcher.
Więcej informacji: <https://java.com>.

- Uruchom plik .class java, który zawiera główną metodę, używając tylko nazwy klasy:

`java `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nazwaklasy</span>

- Uruchom program .jar:

`java -jar `<span class="tldr-var badge badge-pill bg-dark-lm bg-white-dm text-white-lm text-dark-dm font-weight-bold">nazwapliku.jar</span>

- Wyświetl wersje JDK, JRE i HotSpot:

`java -version`

- Wyświetl informacje o użyciu polecenia java:

`java -help`
