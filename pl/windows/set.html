set

Wyświetl lub ustaw zmienne środowiskowe dla bieżącej instancji CMD. Więcej informacji: https://learn.microsoft.com/windows-server/administration/windows-commands/set.

  • Lista wszystkich bieżących zmiennych środowiskowych:

set

  • Ustaw zmienną środowiskową na określoną wartość:

set nazwa=wartość

  • Wymień zmienne środowiskowe zaczynające się od podanego ciągu znaków:

set nazwa

  • Pyta użytkownika o wartość dla określonej zmiennej:

set /p nazwa=wprowadzona_wartość
