drush

Shell da linea di comando ed interfaccia di scripting per Drupal. Maggiori informazioni: https://www.drush.org.

  • Scarica il modulo "foo":

drush dl foo

  • Scarica la versione 7.x-2.1-beta1 del modulo "foo":

drush dl foo-7.x-2.1-beta1

  • Abilita il modulo "foo":

drush en foo

  • Disabilita il modulo "foo":

drush dis foo

  • Pulisci tutte le cache:

drush cc all

  • Pulisci le cache CSS e JavaScript:

drush cc css-js
