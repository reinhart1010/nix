csvclean

Trova e corregge errori di sintassi comuni in file CSV. Incluso in csvkit. Maggiori informazioni: https://csvkit.readthedocs.io/en/latest/scripts/csvclean.html.

  • Correggi un file CSV:

csvclean file.csv

  • Elenca le posizioni di errori di sintassi in un file CSV:

csvclean -n file.csv
