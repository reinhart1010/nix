7zr

Archiviatore di file con alto fattore di compressione. Versione standalone di 7z che supporta solo file .7z. Maggiori informazioni: https://manned.org/7zr.

  • Archivia un file o una directory:

7zr a archivio.7z percorso/del/file_o_directory

  • Estrai un archivio mantenendo la gerarchia delle directory:

7zr x archivio.7z

  • Elenca i contenuti in un archivio:

7zr l archivio
