git status

Mostra le modifiche ai file in un repository Git. Elenca i file modificati, aggiunti e cancellati rispetto al commit corrente. Maggiori informazioni: https://git-scm.com/docs/git-status.

  • Mostra i file modificati che non sono stati ancora committati:

git status

  • Mostra l'output in formato ridotto:

git status -s

  • Nascondi i file non tracciati dall'output:

git status --untracked-files=no

  • Mostra informazioni sul ramo ed in formato ridotto:

git status -sb
