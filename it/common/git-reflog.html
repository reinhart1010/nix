git reflog

Mostra la cronologia delle modifiche a riferimenti locali come HEAD, rami o tag. Maggiori informazioni: https://git-scm.com/docs/git-reflog.

  • Mostra il reflog di HEAD:

git reflog

  • Mostra il reflog di uno specifico ramo:

git reflog nome_ramo
