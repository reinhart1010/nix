couchdb

Interfaccia da linea di comando per il server di database Apache CouchDB. Maggiori informazioni: https://couchdb.apache.org.

  • Avvia couchdb:

couchdb

  • Avvia una shell couchdb interattiva:

couchdb -i

  • Avvia couchdb come processo in background:

couchdb -b
